+-------------------------------------------------------------------+
| - **Repeated Percentage Change**                                  |
+===================================================================+
| When applying many percentage changes one after the other, we     |
| multiply.                                                         |
|                                                                   |
| We do NOT add or subtract the percentages.                        |
|                                                                   |
| 1.  Convert each percentage to a multiplier.                      |
|                                                                   |
| 2.  Multiply the multipliers together. This is the overall        |
|     percentage change.                                            |
|                                                                   |
| 3.  Multiply by the original amount.                              |
+-------------------------------------------------------------------+

+--------------------------------------------------------------------------------+
| - **Example** Calculate repeated percentage change.                            |
+===================================+============================================+
| Increase \$1000 by 30%, then increase the result by another 20%.               |
+-----------------------------------+--------------------------------------------+
| 1\.                               | The multiplier for 30% increase is         |
| $\$ 1000 \times 1.3 \times 1.2$   | $\times$ ......                            |
|                                   |                                            |
| 2\. $= \$ 1000 \times 1.56$       | The multiplier for 20% increase is         |
|                                   | $\times$ ......                            |
| 3\. $= \$ 1560$                   |                                            |
|                                   | We *multiply \| add* the multipliers.      |
|                                   |                                            |
|                                   | The combined multiplier is $\times 1.56,$  |
|                                   | so the overall increase is ......%         |
+-----------------------------------+--------------------------------------------+

# The Unitary Method for Percentages

+-------------------------------------------------------------------+
| - **Review**                                                      |
+===================================================================+
| - Divide to find the value of 1 of a quantity.                    |
|                                                                   |
| +------------------------------+------------------------------+   |
| | a.  5 apples cost \$20,\     | b.  12 oranges cost \$60,\   |   |
| |     how much is 1 apple?     |     how much is 1 orange?    |   |
| +==============================+==============================+   |
| | c.  5 apples cost \$2,\      | d.  \$60 can buy 15          |   |
| |     how much is 1 apple?     |     oranges,\                |   |
| |                              |     how much is 1 orange?    |   |
| +------------------------------+------------------------------+   |
|                                                                   |
| - Use the unitary method to find a new value.                     |
|                                                                   |
| +------------------------------+------------------------------+   |
| | a.  6 bananas are \$30.      | b.  \$40 can buy 80 lollies. |   |
| |                              |                              |   |
| | How much does 8 bananas      | How much does 55 lollies     |   |
| | cost?                        | cost?                        |   |
| +==============================+==============================+   |
+-------------------------------------------------------------------+

+-------------------------------------------------------------------+
| - **Unitary Method**                                              |
+===================================================================+
| The unitary method for percentages involves finding a new         |
| percentage, given a particular percentage.                        |
|                                                                   |
| 1.  Identify the percentage the given value represents.           |
|                                                                   |
| 2.  Divide to find 1%.                                            |
|                                                                   |
| 3.  Multiply to find the percentage you need (100% for the        |
|     original).                                                    |
|                                                                   |
| - Leave numbers as exact fractions; do not round until the very   |
|   last step.                                                      |
+-------------------------------------------------------------------+

+--------------------------------------------------------------------------------+
| - **Example** Solve percentage problems using the unitary method.              |
+===================================+============================================+
| 8% of an amount of money is \$70. What is the full amount?                     |
+-----------------------------------+--------------------------------------------+
| $$\ \ 8\%\  = \ \$ 70$$           | \$70 represents ............% of the       |
|                                   | original.                                  |
| $$\div 8\ \ \ \  \div 8$$         |                                            |
|                                   | We divide 8% by ...... to find 1%.         |
| $$1\% = \frac{\$ 70}{8}$$         |                                            |
|                                   | We multiply 1% by ......... to find 100%,  |
| $$\times 100\ \ \ \  \times 100$$ | which is the ......... amount.             |
|                                   |                                            |
| $$100\% = \$ 875$$                |                                            |
|                                   |                                            |
| The full amount is \$875          |                                            |
+-----------------------------------+--------------------------------------------+

+-------------------------------------------------------------------+
| - **Identifying what Percentage a Value Represents**              |
+===================================================================+
| When a problem tells you a value **after** a percentage change,   |
| that value represents a percentage of the original. You must      |
| identify this percentage before solving the problem.              |
|                                                                   |
| The original amount is always 100%                                |
|                                                                   |
| After an **increase** of $x$%, the new value is                   |
| $\left( \mathbf{100 + x} \right)\mathbf{\%}$ of the original.     |
|                                                                   |
| After a **decrease** of $x$%, the new value is                    |
| $\left( \mathbf{100 - x} \right)\mathbf{\%}$ of the original.     |
|                                                                   |
| Words like 'increased by, decreased by' represent the change,     |
| $x\%.$                                                            |
+-------------------------------------------------------------------+

+-------------------------------------------------------------------+
| - **Key Ideas**                                                   |
+===================================================================+
| 1.  The original amount is always .........%                      |
|                                                                   |
| 2.  After an increase of $x$%, the final value represents ($100$  |
|     ... $x$) % of the original.                                   |
|                                                                   |

|                                                                   |

|     in value, .........%                                          |

+-------------------------------------------------------------------+
| - **Unitary Method**                                              |
+===================================================================+
| 1.  Identify the percentage the given value represents.           |
|                                                                   |
| 2.  Divide to find 1%.                                            |

|                                                                   |
| - Leave numbers as exact fractions; do not round until the very   |
|   last step.                                                      |
+-------------------------------------------------------------------+
