# Syllabus Content

**ME1-11-03** solves problems in three dimensions using trigonometry and
simplifies expressions, proves results and solves problems involving
compound angles using trigonometric identities

**Further trigonometric equations**

- Solve trigonometric equations involving factorisation and/or
  substitution of trigonometric identities over restricted domains

- Examine the representations of $a{\ cos\ }x + b\sin x$  as
  $R\cos{(x \pm \alpha)}$ or $R\sin{(x \pm \alpha)}\$with and without
  graphing applications, where $a \neq 0\$and $b \neq 0$

- Apply the representations of $a{\ cos\ }x + b\sin x$  as
  $R\cos{(x \pm \alpha)}$ or $R\sin{(x \pm \alpha)}$ to graph functions,
  solve equations of the form $a\cos x + b\sin x = c$ over restricted
  domains, and model and solve related problems

- Interpret solutions of trigonometric equations graphically with and
  without graphing applications

- Model and solve practical problems using trigonometric equations and
  analyse their solutions in a variety of contexts

# Solving Trigonometric Equations by using Identities

+-------------------------------------------------------------------+
| - **General Strategy for Solving Trigonometric Equations**        |
+===================================================================+
| When a trigonometric equation has **different trigonometric**     |
| **functions** or **different angles**, use the trigonometric      |
| identities to:                                                    |
|                                                                   |
| 1\. Make all the angles the same.                                 |
|                                                                   |
| 2\. Make all the functions the same.                              |
+-------------------------------------------------------------------+

| - **Identify** homogeneous trigonometric equations                                                                                    |
+---------------------------------------------------------------------------------------------------------------------------------------+
| **Homogeneous** trigonometric equations contain terms where the powers of sin $x$ and cos $x$ add to the same total.                  |
+----------------------------+----------------------------------+---------------------------------+-------------------------------------+
| ✖                          | ✔                                | ✔                               | ✖                                   |
|                            |                                  |                                 |                                     |
| $$\sin^{2}x + \sin x = 0$$ | $$\sin^{2}x + \sin x\cos x = 0$$ | $$3\sin^{4}x - 4\cos^{4}x = 2$$ | $$\sin^{4}x + \sin^{2}x\cos x = 2$$ |
+----------------------------+----------------------------------+---------------------------------+-------------------------------------+

+-------------------------------------------------------------------+
| - **Strategies for Solving Homogeneous Equations**                |
+===================================================================+
| To solve, divide every term by a power of $\cos x$ to get a       |
| polynomial in $\tan x$.                                           |
|                                                                   |
| - When you divide by $\cos x$, you are assuming that              |
|   $\cos x \neq 0$.\                                               |
|   Therefore, you must **always** check whether $x =$              |
|   $\frac{\pi}{2}$ or $\frac{3\pi}{2}$ is a solution by            |
|   substituting.                                                   |
|                                                                   |
| - Remember that $\sin{2x}$ and $\cos{2x}$ can be expanded to give |
|   terms of degree 2.                                              |
+-------------------------------------------------------------------+

+-----------------------------------------------------------------------------------------------------------------------------------------------+
| - **Example** Solve homogeneous trigonometric equations                                                                                       |
+===============================================================================================================================================+
| Solve $\sin^{2}x - \sin x\cos x = 0$ for $x \in \lbrack 0,2\pi)$                                                                              |
|                                                                                                                                               |
| $\div \cos^{2}x\ \ \ \ \ \ \ \ \  \div \cos^{2}x$                                                                                             |
|                                                                                                                                               |

|                                                                                                                                               |

|                                                                                                                                               |
| $$\ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \therefore x = 0,\ \pi,\frac{\pi}{4},\ \frac{5\pi}{4}$$ |
|                                                                                                                                               |
| Check whether $x =$ $\frac{\pi}{2},\frac{3\pi}{2}$ is a solution by substitution                                                              |

| - **Squaring Both Sides**                                         |
+===================================================================+
| Squaring both sides lets you apply a double-angle identity to     |
| simplify the equation.                                            |
|                                                                   |
| There is *usually* a better method you could use.                 |
|                                                                   |
| - Squaring introduces extra solutions, so always check your       |
|   answers by substituting them into the original equation.        |
+-------------------------------------------------------------------+

+--------------------------------------------------------------------------------------------------------------------------------------------------------------------------------+

Foundation

1.  Consider the equation $sin\ 2x\  - \ cos\ x\  = \ 0.$

    a.  By using a double-angle formula and then factoring, show that
        $cos\ x\  = \ 0$ or $\sin\ x\  = \frac{1}{2}.$

    b.  Hence solve the equation for $0\  \leq \ x\  \leq \ 2\pi.$

$${sin\ 2x\  - \ cos\ x\  = \ 0
}{2\ sin\ x\ cos\ x\  - \ cos\ x\  = \ 0
}{cos\ x(2\ sin\ x\  - \ 1)\  = \ 0
}{\therefore\ cos\ x\  = \ 0\ or\ sin\ x\  = \frac{1}{2}
}{cos\ x\  = \ 0:\ x\  = \frac{\pi}{2},\frac{3\pi}{2}
}{sin\ x\  = \frac{1}{2}:\ x\  = \frac{\pi}{6},\frac{5\pi}{6}
}{\therefore\ x\  = \frac{\pi}{6},\frac{\pi}{2},\frac{5\pi}{6}or\frac{3\pi}{2}}$$

2.  Consider the equation $cos\ 2x\  - \ cos\ x\  = \ 0.$

    a.  By using a double-angle formula and then factoring, show that
        $cos\ x\  = \ 1$ or $- \frac{1}{2}.$

    b.  Hence solve the equation for $0\  \leq \ x\  \leq \ 2\pi.$

$${cos\ 2x\  - \ cos\ x\  = \ 0
}{2\ cos²x\  - \ 1\  - \ cos\ x\  = \ 0
}{2\ cos²x\  - \ cos\ x\  - \ 1\  = \ 0

}{cos\ x\  = \  - \frac{1}{2}:\ x\  = \frac{2\pi}{3},\frac{4\pi}{3}
}{\therefore\ x\  = \ 0,\frac{2\pi}{3},\frac{4\pi}{3}or\ 2\pi}$$
