# Overview

## Syllabus Content

**ME1-11-01** solves problems involving inequalities, functions and
their inverses, graphical relationships between functions, and
parametric equations

**Inverse functions**

- Describe a function as one-to-one if every element in the range of the
  function corresponds to exactly one element of the domain

- Establish that the reflection of a point in the line $y = x$ reverses
  the coordinates of the point

- Define an inverse function $f^{- 1}$ informally as a function that
  reverses or undoes the effect of the function $f$

- Recognise that inverse functions exist for one-to-one functions

- Determine the equation for the inverse function $f^{- 1}(x)$ of a
  given one-to-one function $y = f(x)$ by interchanging the variables
  $x$ and $y$ in $y = f(x)$

- Compare the graphs of a function and its inverse function using
  graphing applications, and recognise that the two graphs are
  reflections of each other in the line $y = x$

- Establish that the domain of $f^{- 1}(x)$ is the range of $f(x)$, and
  the range of $f^{- 1}(x)$ is the domain of $f(x)$, and use this
  relationship to solve problems

- Graph the inverse function of a given one-to-one function

- Explain that the reflection in the line $y = x$ exchanges horizontal
  and vertical lines and recognise that the horizontal line test can
  therefore be applied to the graph of $y = f(x)$ to determine whether
  its reflection in the line $y = x$ is a function

- Apply restrictions to the domain of a function, if it is not
  one-to-one, to obtain an inverse function

- Define formally $f^{- 1}(x)$ to be the inverse function of $f(x)$ if
  the relationships $f\left( f^{- 1}(x) \right) = x$ and
  $f^{- 1}\left( f(x) \right) = x$ hold, and use this definition to
  solve problems

- Solve problems based on the relationship between a function and its
  inverse function using algebraic and graphical techniques, including
  determining the points of intersection of a function and its inverse,
  where they exist

# Inverse of a Relation

- An **inverse operation** will undo an operation, for example, the
  inverse of $\times 7$ is $\div 7$.

- The **inverse** **of a function** will undo the function. It is called
  the **inverse relation.**

## Equation of the Inverse Relation

- The **inverse** of a function **undoes** what the original function
  does.

- The graph of a function and its inverse are **reflections across**
  $\mathbf{y = x}$.

- Find a function's inverse by **swapping the** $\mathbf{x}$ **and**
  $\mathbf{y}$ **variables**.

# Determining if the Inverse Relation is a Function 

- The inverse of a function is not always a function itself.

- Remember: if a relation passes the vertical line test, it is a
  function.

- Since the inverse swaps $x$ and $y$ coordinates,\
  if a function passes the **horizontal line test**, then its **inverse
  will also be a function**.

- **Even functions** **do not have an inverse function**, as its
  symmetry means it will fail the horizontal line test.

  --------------------------------------------------
  **Type of      **Vertical     **Horizontal Line
  Relation**     Line Test**    Test**
  -------------- -------------- --------------------
  One-to-one     Passes         Passes (inverse is
                 (function)     function)

  Many-to-one    Passes         Fails
                 (function)     

  One-to-many    Fails          Passes (inverse is
                                function)

  Many-to-many   Fails          Fails
  --------------------------------------------------

# Inverse Function Notation

- If the inverse relation is itself a function, we call it the **inverse
  function**.

- The inverse function of $f(x)$ is written as $f^{- 1}(x)$.

![A diagram of a function AI-generated content may be
incorrect.](media/Further Work with Functions 2_Inverse functions/media/image53.png){width="1.7122298775153106in"
height="1.0386428258967628in"}

- Since the inverse function "undoes" the original function, the
  composition of a function and its inverse cancels out.

$f\left( f^{- 1}(x) \right) = x$ and $f^{- 1}\left( f(x) \right) = x$

- Reminder that $f^{- 1}(x)$ and $\left( f(x) \right)^{- 1}$ are
  different.

+------------------------------------+
| **Finding the Inverse Function**   |
+====================================+
| If you are given $f(x)$ and asked  |
| to find $f^{- 1}(x)$:              |
|                                    |
| 1.  Rewrite $f(x)\$using $y =$     |
|                                    |
| 2.  Exchange the $x$ and $y$.      |
|                                    |
| 3.  Rearrange for $y$, then write  |
|     as $f^{- 1}(x)$.               |
+------------------------------------+

- If the original function has a restriction, find the domain of the
  inverse by:

  - Using the range of the original function as the domain of the
    inverse, or

  - Sketching the reflection of the original function, or

  - Solving the relevant inequality

# Restricting the Domain so the Inverse is a Function

- When a function's inverse is not a function, we need to restrict the
  domain so that the new restricted function has an inverse.

+----------------------------------------------+
| **Restricting the Domain**                   |
+==============================================+
| - We call a function **invertible** if its   |
|   inverse is also a function.                |
|                                              |
| - A function is invertible when it is        |
|   **one-to-one**.                            |
|                                              |
| - Make a function invertible by limiting its |
|   domain based on **turning points.**        |
|                                              |
| <!-- -->                                     |
|                                              |
| - Restrict domains so that the curve is      |
|   **strictly increasing** or **strictly      |
|   decreasing**, i.e., the gradient does not  |
|   change sign,\                              |
|   and include the turning point.             |
+----------------------------------------------+

## Point of Intersection of $\mathbf{f}\left( \mathbf{x} \right)$ and $\mathbf{f}^{\mathbf{- 1}}\left( \mathbf{x} \right)$

+------------------------------------------------------+
| **Point of Intersection of**                         |
| $\mathbf{f}\left( \mathbf{x} \right)$ **and**        |
| $\mathbf{f}^{\mathbf{- 1}}\left( \mathbf{x} \right)$ |
+======================================================+
| If a function and its inverse function intersects,\  |
| it will be a point on the line $y = x$.              |
|                                                      |
| To find the point of intersection,\                  |
| let $f(x) = y$ or $f^{- 1}(x) = y$ and solve for     |
| $x.$                                                 |
+------------------------------------------------------+

- For a function to intersect with its inverse, the point of
  intersection must satisfy both f$(x) = y$ and $f^{- 1}(x)\  = \ y$,
  which means we need:

$$(x,\ y)\  = \ (y,\ x)$$

- This is only possible when $x = y$. So, functions intersect with their
  inverses at points that lie on the line $y = x$.
