# Overview

## Syllabus Content

**ME1-11-01** solves problems involving inequalities, functions and
their inverses, graphical relationships between functions, and
parametric equations

**Graphical relationships**

- Examine the relationship between the graph of $y = f(x)$ and the graph
  of $y = \frac{1}{f(x)}$ using graphing applications, and graph
  $y = \frac{1}{f(x)}$ given $y = f(x)$ in algebraic or graphical form,
  identifying any vertical and horizontal asymptotes of $y = f(x)$ and
  $y = \frac{1}{f(x)}$

- Graph $y = \sec x$, $y = {cosec}x$ and $y = \cot x$ in both radians
  and degrees, identifying key properties including asymptotes, period,
  domain, range and symmetry, and compare each graph with the graph of
  its reciprocal

- Examine the relationship between the graph of $y = f(x)$ and the
  graphs of $y = \left| f(x) \right|$ and $y = f\left( |x| \right)$
  using graphing applications, and graph $y = \left| f(x) \right|$ and
  $y = f\left( |x| \right)$ given $y = f(x)$ in algebraic or graphical
  form

- Examine the relationship between the graphs of $y = f(x)$ and
  $y = g(x)$ and the graphs of $y = f(x) + g(x)$ and $y = f(x) - g(x)$
  using graphing applications, and graph $y = f(x) + g(x)$ and
  $y = f(x) - g(x)$ given $y = f(x)$ and $y = g(x)$ in algebraic or
  graphical form

- Determine the domains and ranges of the sum and difference of
  functions where possible, and, if appropriate, verify them using a
  graphing application

- Apply knowledge of graphical relationships to solve problems involving
  graphs of functions, justifying conclusions in the context of the
  problem where appropriate

# Sketching The Reciprocal of a Function

- The reciprocal of a function $f(x)$ is $\frac{1}{f(x)}$, or
  $\left( f(x) \right)^{- 1}$.

- To sketch the graph, the key features to identify are **vertical
  asymptotes**, **intercepts**, **behaviour as**
  $\mathbf{x \rightarrow \pm \infty}$ and behaviour as
  $\mathbf{x \rightarrow}$ **asymptotes**^±^.

- Use these rules to sketch $y =$ $\frac{1}{f(x)}$ given only the graph
  of $y = f(x)$:

1.  **New coordinates and sign.**

> If the graph of $y = f(x)$ has a point $(a,b)$, where $b \neq 0$, then
> $y =$ $\frac{1}{f(x)}$ hhas a point at $\left( a,\frac{1}{b} \right)$.
>
> If $f(x)$ is negative, then $\frac{1}{f(x)}$ is negative.
>
> If $f(x)$ is positive, then $\frac{1}{f(x)}$ is positive.
>
> If $f(x) = 1$, then $\frac{1}{f(x)}$ $= 1$ as well.

1.  **Vertical asymptotes** **and behaviour.**

> If $f(x) = 0$, then $\frac{1}{f(x)}$ is a vertical asymptote.
>
> If $f(x)$ is very small, then $\frac{1}{f(x)}$ is very large.
>
> If $f(x) \rightarrow 0^{-}$, then $\frac{1}{f(x)}$
> $\rightarrow - \infty$
>
> If $f(x) \rightarrow 0^{+}$, then $\frac{1}{f(x)}$
> $\rightarrow + \infty$

2.  **Horizontal asymptotes** **and behaviour.**

> If $f(x)$ is very large, then $\frac{1}{f(x)}$ is very small.\
> If $f(x) \rightarrow + \infty$, then $\frac{1}{f(x)}$
> $\rightarrow 0^{+}$ (horizontal asymptote)
>
> If $f(x) \rightarrow - \infty$, then $\frac{1}{f(x)}$
> $\rightarrow 0^{-}$ (horizontal asymptote)
>
> If $f(x)$ has an existing horizontal asymptote $f(x) \rightarrow c$,
> then $\frac{1}{f(x)}$ $\rightarrow$ $\frac{1}{c}$

3.  **Turning points:**

> If $f(x)$ has a turning point at $(a,b)$, then $\frac{1}{f(x)}$ has a
> turning point at $\left( a,\frac{1}{b} \right)$.

4.  **Gradient:**

> If $f(x)$ is increasing, then $\frac{1}{f(x)}$ is decreasing.
>
> If $f(x)$ is decreasing, then $\frac{1}{f(x)}$ is increasing.
>
> **Example** Sketch a function $\frac{1}{f(x)}$, given the graph of
> $f(x)$.

# Products of Graphs

- To find the **product** of two graphs $p(x) = f(x)g(x)$:

1.  Multiply $y$-coordinates.

2.  Key points to look out for:

    a.  Zeroes of $f(x)$ or $g(x)$: $p(x) = 0$

    b.  Where $f(x) = 1\$or $- 1$: $p(x) = g(x)$ or $- g(x)$

    c.  Where $g(x) = 1\$or $- 1$: $p(x) = f(x)$ or $- f(x)$

3.  Determine the end behaviour as $x \rightarrow \pm \infty$ and
    connect the points using a smooth curve.

    a.  If $f(x) \rightarrow 0$ or $g(x) \rightarrow 0$, then
        $p(x) \rightarrow \ 0$.

    b.  If either $f(x)$ or $g(x) \rightarrow \pm \infty$, then their
        product $p(x)$ will be:

        i.  $+ \infty$ if $f(x)$ and $g(x)$ have the same sign.

        ii. $- \infty\$if $f(x)$ and $g(x)$ have different sign.

    c.  If $f(x) \rightarrow 0$ while $g(x) \rightarrow \infty$, the
        product may be unknown.

10. Using algebra, determine the symmetry of the sum, difference, and
    product of functions.

+-----------------------------------------+---------------------------------------+----------------------------------------+---------------------------------------+
|                                         | $\mathbf{f}\left( \mathbf{x} \right)$ | $\mathbf{f}\left( \mathbf{x} \right)$  | $\mathbf{f}\left( \mathbf{x} \right)$ |
|                                         | **even**                              | **odd**                                | **even**                              |
|                                         | $\mathbf{g}\left( \mathbf{x} \right)$ | $\mathbf{g}\left( \mathbf{x} \right)$  | $\mathbf{g}\left( \mathbf{x} \right)$ |
|                                         | **even**                              | **odd**                                | **odd**                               |
+=========================================+=======================================+========================================+=======================================+
| $$\mathbf{s}\left( \mathbf{x} \right)$$ | $$s( - x) =$$                         | $$s( - x) = f( - x) + g( - x)$$        | $$s( - x) = f( - x) + g( - x)$$       |
|                                         |                                       |                                        |                                       |
| $$= f(x) + g(x)$$                       | $$s( - x) = f( - x) + g( - x)$$       | $$= - f(x) - g(x)\ \ \ $$              | $$= f(x) - g(x)\ \ \ \ \ \ \ $$       |
|                                         |                                       |                                        |                                       |
|                                         | $$= f(x) + g(x)\ \ \ \ \ \ \ $$       | $\therefore$ odd                       | $\therefore$ neither                  |
|                                         |                                       |                                        |                                       |
|                                         | $\therefore$ even                     |                                        |                                       |
+-----------------------------------------+---------------------------------------+----------------------------------------+---------------------------------------+
| $$\mathbf{d}\left( \mathbf{x} \right)$$ | $$d( - x) = f( - x) - g( - x)$$       | $$d( - x) = f( - x) - g( - x)$$        | $$d( - x) = f( - x) - g( - x)$$       |
|                                         |                                       |                                        |                                       |
| $$= f(x) - g(x)$$                       | $$= f(x) - g(x)\ \ \ \ \ \ \ $$       | $$= - f(x) + g(x)\ \ \ \ $$            | $$= f(x) + g(x)\ \ \ \ \ \ $$         |
|                                         |                                       |                                        |                                       |
|                                         | $\therefore$ even                     | $$= - \left( f(x) - g(x) \right)$$     | $\therefore$ neither                  |
|                                         |                                       |                                        |                                       |
|                                         |                                       | $\therefore$ odd                       |                                       |
+-----------------------------------------+---------------------------------------+----------------------------------------+---------------------------------------+
| $$\mathbf{p}\left( \mathbf{x} \right)$$ | $$p( - x) = f( - x)g( - x)$$          | $$p( - x) = f( - x)g( - x)\ \ \ \ \ $$ | $$p( - x) = f( - x)g( - x)\ \ $$      |
|                                         |                                       |                                        |                                       |
| $$= f(x)g(x)$$                          | $$= f(x)g(x)$$                        | $$= - f(x) \times - g(x)$$             | $$= f(x) \times - g(x)$$              |
|                                         |                                       |                                        |                                       |
|                                         | $\therefore$ even                     | $$= f(x)g(x)\ \ \ \ \ \ \ \ \ \ \ \ $$ | $$= - f(x)g(x)\ \ \ \ \ $$            |
|                                         |                                       |                                        |                                       |
|                                         |                                       | $\therefore$ even                      | $\therefore$ odd                      |
