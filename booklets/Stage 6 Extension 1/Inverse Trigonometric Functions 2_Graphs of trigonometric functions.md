**Graphs of inverse trigonometric functions**

- Graph $y = \sin x$ for $- \frac{\pi}{2} \leq x \leq \frac{\pi}{2}$,

2.  a.  Explain why
        $\sin^{- 1}{\left( \sin\frac{5\pi}{3} \right) \neq \frac{5\pi}{3}}.$

The range of sin⁻¹ is
$\left\lbrack - \frac{\pi}{2},\frac{\pi}{2} \right\rbrack$, but
$\frac{5\pi}{3} \notin \ \left\lbrack - \frac{\pi}{2},\frac{\pi}{2} \right\rbrack$.

Since $\frac{5\pi}{3}$ is outside the range of sin⁻¹, it cannot equal
$sin^{- 1}\left( \sin\left( \frac{5\pi}{3} \right) \right).$

b.  What condition must $x$ meet for
    $\sin^{- 1}\left( \sin x \right) = x$?

Since
$\sin^{- 1}x \in \left\lbrack - \frac{\pi}{2},\frac{\pi}{2} \right\rbrack,\$then
$x \in \left\lbrack - \frac{\pi}{2},\frac{\pi}{2} \right\rbrack$

c.  Describe how to evaluate
    $\sin^{- 1}\left( \sin\frac{5\pi}{3} \right)$.

Find the angle equivalent to $\frac{5\pi}{3}$ in the domain
$\left\lbrack - \frac{\pi}{2},\frac{\pi}{2} \right\rbrack$

Then use the identity $\sin^{- 1}\left( \sin x \right) = x$

$$Since\ \sin\left( \frac{5\pi}{3} \right) = \sin\left( - \frac{\pi}{3} \right)$$

$$\sin^{- 1}\left( \sin\frac{5\pi}{3} \right) = \sin^{- 1}\left( \sin\left( - \frac{\pi}{3} \right) \right) = - \frac{\pi}{3}$$

6.  ![](media/Inverse Trigonometric Functions 2_Graphs of trigonometric functions/media/image16.png){width="2.4069444444444446in"
    height="2.354861111111111in"}A person stands $x$ metres away from a

> The viewing angle $\theta$ is the difference in angle of elevation
> from the observer to the top and the base of the billboard.
>
> Show that $\theta = \tan^{- 1}\left( \frac{6x}{x^{2} + 16} \right)$

$$Let\ \alpha\  = \ angle\ of\ elevation\ to\ top\ of\ billboard$$

$$Let\ \beta\  = \ angle\ of\ elevation\ to\ base\ of\ billboard$$

$$tan\ \alpha\  = \frac{8}{x}$$

$$tan\ \beta\  = \frac{2}{x}$$

$$viewing\ angle\ \theta\  = \ \alpha\  - \ \beta$$

$$tan\ \theta\  = tan(\alpha\  - \ \beta) = \frac{tan\ \alpha\  - \ tan\ \beta}{1\  + \ tan\ \alpha\ tan\ \beta}$$

$$tan\ \theta\  = \frac{\frac{8}{x} - \frac{2}{x}}{1\  + \ \left( \frac{8}{x} \right)\left( \frac{2}{x} \right)}$$

$$= \frac{\frac{6}{x}}{1\  + \frac{16}{x^{2}}}$$

$$= \frac{\frac{6}{x}}{\frac{x^{2} + \ 16}{x^{2}}}$$

$$= \ \left( \frac{6}{x} \right) \times \ \left( \frac{x^{2}}{x^{2} + \ 16} \right)$$

$$= \frac{6x}{x^{2} + \ 16}$$

$$Therefore:\ \theta\  = \ \tan^{- 1}\left( \frac{6x}{x^{2} + \ 16} \right)$$
