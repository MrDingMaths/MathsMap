# Syllabus Content

**ME1-12-02** operates with 2D and 3D vectors and uses 2D vectors to
solve problems involving motion in two dimensions

**Projectile motion**

- Recognise that the gravitational force on a mass may be regarded as a
  constant acting in a downwards direction when the motion of the object
  is restricted to a small region near the Earth's surface

- Model and analyse a projectile's path where the projectile is a point
  and air resistance is negligible, subject to only acceleration due to
  gravity, assuming that the projectile is moving close to the Earth's
  surface

- Represent the motion of a projectile using vectors

- Recognise that the horizontal and vertical components of the motion of
  a projectile can be represented by horizontal and vertical vectors

- Derive and use the equations of motion of a projectile in vector form
  by splitting 2D motion into horizontal and vertical components to
  solve problems on projectiles

- Find the Cartesian equation of the path of a projectile using
  parametric equations for the horizontal and vertical components of the
  displacement vector

- Determine features of the flight of a projectile, including time of
  flight, maximum height, range, instantaneous velocity and impact
  velocity

- Solve problems relating to the path of a projectile in which the
  initial velocity and/or angle of projection may be unknown, in a
  variety of contexts

# Projectile Motion and Velocity as a Vector

+-----------------------------------------------------------------------+
| - **Definition of Projectile Motion**                                 |
+===================================+===================================+
| A **projectile** is an object dropped, thrown, struck, or fired into  |
| the air near Earth\'s surface.                                        |
|                                                                       |
| We model a projectile as a **single point**.                          |
|                                                                       |
| The **only force** acting on a projectile is **gravity**. We ignore   |
| air resistance.                                                       |
|                                                                       |
| There is **only vertical acceleration** downwards, and **no           |
| horizontal acceleration.**                                            |

# Deriving the Six Equations of Motion

+-------------------------------------------------------------------+
| - **The Six Equations of Motion**                                 |
+===================================================================+
| The three equations of motion in one dimension describe           |
| **displacement**, **velocity**, and **acceleration** with respect |
| to time $t$, related by differentiation and integration.          |
|                                                                   |
| $x(t):\$displacement $\dot{x}(t)$: velocity $\ddot{x}:$           |
| acceleration                                                      |
|                                                                   |
| In projectile motion, there are six equations of motion:          |
|                                                                   |
| three for horizontal motion and three for vertical motion.        |
|                                                                   |
| Every projectile motion problem starts from two facts:            |
|                                                                   |
| - $\ddot{x}(t) = 0$ No horizontal force acts, so there is no      |
|   horizontal acceleration.                                        |
|                                                                   |
| - $\ddot{y}(t) = - g$ The only vertical force is the constant     |
|   force of gravity.                                               |
|                                                                   |
| The remaining four equations follow by integrating these twice.   |
+-------------------------------------------------------------------+

+----------------------------------------------------------------------------------------------------------------------------------------------------------------+
| - **Initial Conditions for Projection from the ground**                                                                                                        |
+======================================================================================================+============================+============================+
| During integration, we need **initial conditions** for **velocity** and **displacement** to find the particular solutions of the velocity and displacement     |
| equations.                                                                                                                                                     |
|                                                                                                                                                                |
| When a projectile is fired from the ground, we set the projection point as the **origin:**                                                                     |
+------------------------------------------------------------------------------------------------------+----------------------------+----------------------------+
| ![](media/Introduction to Vectors 4_Projectile Motion/media/image2.png){width="1.3652777777777778in" | $${x(0) = 0                | $${y(0) = 0                |
| height="0.8361111111111111in"}                                                                       | }{\dot{x}(0) = V\cos\theta | }{\dot{y}(0) = V\sin\theta |
|                                                                                                      | }{\ddot{x}(0) = 0}$$       | }{\ddot{y}(0) = - g}$$     |

+----------------------------------------------------------------------------------------------------------+
| By deriving the six equations using the general initial velocity of $V$ m/s at $\theta$ above            |

+---------------------+---------------------------------------+--------------------------------------------+
|                     | **Horizontal**                        | **Vertical**                               |
+---------------------+---------------------------------------+--------------------------------------------+

Foundation

Take $g = 10\ {ms}^{- 2}$

1.  A particle is projected from the origin with speed 20 m/s at 60° to
    the horizontal. By integrating, find the velocity and displacement,
    and write the acceleration, velocity, and displacement in vector
    form.

Shown here is the derivation using $\mathbf{i}$ and $\mathbf{j}$
notation

$$\mathbf{a} = 0\mathbf{i} - 10\mathbf{j}$$

$$\mathbf{v} = C_{1}\mathbf{i} + ( - 10t + C_{2})\mathbf{j}$$

$$\mathbf{v}(0) = 20cos60^{\circ}\mathbf{i} + 20sin60^{\circ}\mathbf{j}$$

$$\mathbf{v}(0) = 10\mathbf{i} + 10\sqrt{3}\mathbf{j}$$

$$C_{1} = 10,\quad\quad C_{2} = 10\sqrt{3}$$

$$\boxed{\mathbf{v} = 10\mathbf{i} + (10\sqrt{3} - 10t)\mathbf{j}}$$

$$\mathbf{r} = (10t + C_{3})\mathbf{i} + \left( 10\sqrt{3}t - 5t^{2} + C_{4} \right)\mathbf{j}$$

$$\mathbf{r}(0) = 0\mathbf{i} + 0\mathbf{j}$$

$$C_{3} = 0,\quad\quad C_{4} = 0$$

$$\boxed{\mathbf{r} = 10t\mathbf{i} + \left( 10\sqrt{3}t - 5t^{2} \right)\mathbf{j}}$$

$$\boxed{\mathbf{a} = - 10\mathbf{j}}$$

# Launching from a Platform

+-----------------------------------------------------------------------------------------------------------------------------------------------------------+
| - **Projection from a Platform**                                                                                                                          |
+====================================+====================================+====================================+============================================+
| When a projectile launches from height $h$ above the ground, there are two approaches:                                                                    |
|                                                                                                                                                           |
| Set the initial vertical displacement to $h$; the equation of vertical displacement becomes a more complex quadratic.                                     |
|                                                                                                                                                           |
| Keep the origin at the launch point and set ground level to $- h$; the initial conditions stay the same, but the landing condition becomes $y(t) = - h$   |
|                                                                                                                                                           |
| **Option 1: Option 2:**                                                                                                                                   |
|                                                                                                                                                           |
| ![](media/Introduction to Vectors 4_Projectile Motion/media/image2.png){width="1.3520833333333333in"                                                      |
| height="1.3777777777777778in"}![](media/Introduction to Vectors 4_Projectile Motion/media/image2.png){width="1.3520833333333333in"                        |
| height="1.3777777777777778in"}                                                                                                                            |
|                                                                                                                                                           |
| Origin $(0,h)$ Origin $(0,0)$                                                                                                                             |
|                                                                                                                                                           |
| Ground $y = 0$ Ground $y = - h$                                                                                                                           |
|                                                                                                                                                           |
| Mr Ding recommends option 2.                                                                                                                              |
|                                                                                                                                                           |
| So, when a projectile is fired from a **platform, treat the platform as the origin**.                                                                     |
|                                                                                                                                                           |
| All our initial conditions stay the same.                                                                                                                 |
+-------------------------------------------------------------------------+---------------------------------------------------------------------------------+
| $${x(0) = 0                                                             | $${y(0) = 0                                                                     |
| }{\dot{x}(0) = V\cos\theta                                              | }{\dot{y}(0) = V\sin\theta                                                      |
| }{\ddot{x}(0) = 0}$$                                                    | }{\ddot{y}(0) = - g}$$                                                          |
+-------------------------------------------------------------------------+---------------------------------------------------------------------------------+
| So, our equations of motion stay the same.                                                                                                                |
+------------------------------------+-------------------------------------------------------------------------+--------------------------------------------+
|                                    | **Horizontal**                                                          | **Vertical**                               |
+------------------------------------+-------------------------------------------------------------------------+--------------------------------------------+
| **Acceleration**                   | $$\ddot{x} = 0$$                                                        | $$\ddot{y} = - g$$                         |
+------------------------------------+-------------------------------------------------------------------------+--------------------------------------------+
| **Velocity**                       | $$\dot{x} = V\cos\theta$$                                               | $$\dot{y} = - gt + V\sin\theta$$           |
+------------------------------------+-------------------------------------------------------------------------+--------------------------------------------+
| **Displacement**                   | $$x = Vt\cos\theta$$                                                    | $$y = - \frac{1}{2}gt^{2} + Vt\sin\theta$$ |
+------------------------------------+-------------------------------------------------------------------------+--------------------------------------------+
| - When using the equations, treat the platform as $y = 0$                                                                                                 |
|                                                                                                                                                           |
| - But when answering a question, add the height of the platform in your final answer.\                                                                    |
|   E.g. if you find max height is $y = 3$, and your platform is 20 m high, answer as 23 metres                                                             |
+-----------------------------------------------------------------------------------------------------------------------------------------------------------+

For all questions in this section take $g = 10$ m/s² unless otherwise
stated.

16. Points $A$ and $B$ lie on a vertical line above the origin $O$ on
    the ground, with $OA = kh$ and $OB = h$ where $k > 1$.\
    A particle is projected horizontally from $A$ with speed $U$. Some
    time later, a second particle is projected horizontally from $B$
    with speed $V$. Both land at the same point on the ground.\
    Show that $\frac{V}{U} = \sqrt{k}$.

$$A:\ x_{A} = Ut,\ y_{A} = kh - \frac{1}{2}gt^{2}$$

$$\text{land }y_{A} = 0$$

$$t = \sqrt{\frac{2kh}{g}}$$

$$x_{A} = U\sqrt{\frac{2kh}{g}}$$

$$B\ (let\ time\ be\ T):y_{B} = h - \frac{1}{2}gT^{2} = 0$$

$$T = \sqrt{\frac{2h}{g}}$$

$$x_{B} = V\sqrt{\frac{2h}{g}}$$

$$\text{Same landing point: }U\sqrt{\frac{2kh}{g}} = V\sqrt{\frac{2h}{g}}$$

$$\frac{V}{U} = \sqrt{\frac{\left( \frac{2kg}{h} \right)}{\left( \frac{2h}{g} \right)}} = \sqrt{k}$$

17. A projectile is launched from a point at height $h$ above the ground
    (origin at the launch point) with speed $V$ at angle $\theta$. Show
    it lands after
    $t = \frac{V\sin\theta + \sqrt{V^{2}\sin^{2}\theta + 2gh}}{g}$.

$$a)\ y = - \frac{1}{2}gt^{2} + V\sin\theta\, t;\ $$

$$\text{land }y = - h:\ \frac{1}{2}gt^{2} - V\sin\theta\, t - h = 0$$

$$gt^{2} - 2V\sin\theta\, t - 2gh = 0$$

$$t = \frac{2V\sin\theta + \sqrt{4V^{2}sin^{2}\theta + 8gh}}{2g}\ $$

$$= \frac{V\sin\theta + \sqrt{V^{2}sin^{2}\theta + 2gh}}{g}\ $$

18. A ball was projected from ground level and, when at its highest
    point, just cleared a 3 m wall. Given the initial velocity was 20
    m/s at angle α to the horizontal, prove that
    $\alpha\  = \ \sin^{- 1}\left( \frac{\sqrt{15}}{10} \right).$

At the highest point, vertical velocity = 0.

Vertical component: $\dot{y} = \ 20\ \sin\ \alpha\  - \ 10t\  = \ 0$

$$t\  = \frac{20\ \sin\ \alpha}{10}$$

Height at highest point = 3:

$$y = - 5t^{2} + 20\sin\alpha t$$

$$- 5\left( \frac{20\ \sin\ \alpha}{10} \right)^{2} + 20\sin\alpha\left( \frac{20\ \sin\ \alpha}{10} \right) = 3$$

$$- 20\sin^{2}\alpha + 40\sin^{2}\alpha = 3$$

$$\sin^{2}\alpha = \frac{3}{20}$$

$$\sin\alpha = \frac{\sqrt{3}}{\sqrt{20}} = \frac{\sqrt{3}}{2\sqrt{5}} = \frac{\sqrt{15}}{10}$$

$$\therefore\alpha = \sin^{- 1}\left( \frac{\sqrt{15}}{10} \right)$$

19. Two particles $P_{1}$ and $P_{2}$ are projected from the origin with
    initial velocity vectors $20\mathbf{i}\  + \ 30\mathbf{j}$ and
    $60\mathbf{i}\  + \ 50\mathbf{j}$ respectively. $P_{2}$ is projected
    2 seconds after $P_{1}$.\
    Determine whether they collide and, if so, when.

$$P_{1}:\ x_{1} = \ 20t,\ y_{1} = \  - 5t^{2} + \ 30t$$

$$P_{2}(t\  \geq \ 2):\ x_{2} = \ 60(t\  - \ 2),\ y_{2} = \  - 5(t\  - \ 2)^{2} + \ 50(t\  - \ 2)$$

Let $x_{1} = \ x_{2}:$

$$\ 20t\  = \ 60t\  - \ 120\ $$

$$t\  = \ 3$$

Check $y$ values:

$$y_{1}(3) = \  - 45\  + \ 90\  = \ 45$$

$$y_{2}(3) = \  - 5\  + \ 50\  = \ 45$$

They collide at $t\  = \ 3\$s (1 s after $P_{2}$ is projected).

20. A particle is projected from level ground with initial speed $V$ m/s
    at an angle of $\theta$ to the horizontal.

    a.  Prove that:

        i.  the greatest height is $\frac{V^{2}sin^{2}\theta}{2g}$
            metres,

        ii. the horizontal range is $\frac{V^{2}sin\ 2\theta}{g}$
            metres.

    b.  If the horizontal range is five times the greatest height, prove
        that $\theta\  = \tan^{- 1}\left( \frac{4}{5} \right)$

a i)

$$At\ greatest\ height,\ \dot{y} = \ 0:$$

$$V\sin\theta - \ gt\  = \ 0$$

$$t\  = \frac{V\ \sin\ \theta}{g}$$

$$H\  = \ V\ \sin\ \theta\  \cdot \frac{V\ \sin\ \theta}{g} - \frac{1}{2}g\left( \frac{V\ \sin\ \theta}{g} \right)^{2}$$

$$\  = \frac{V^{2}sin^{2}\theta}{g} - \frac{V^{2}sin^{2}\theta}{2g}$$

$$= \frac{V^{2}sin^{2}\theta}{2g}$$

a ii)

Lands when $y\  = \ 0:$

$$V\sin\theta \cdot \ t\  - \frac{1}{2}gt^{2} = \ 0$$

$$t\left( V\ \sin\ \theta\  - \frac{1}{2}gt \right) = \ 0$$

$$t\  = \frac{2V\ \sin\ \theta}{g}$$

$$R\  = \ V\cos\theta \cdot \frac{2V\ \sin\ \theta}{g} = \frac{V^{2}\left( 2\ \sin\ \theta\ \cos\ \theta \right)}{g}$$

$$= \frac{V^{2}\sin\ 2\theta}{g}$$

b\)

$$R\  = \ 5H:$$

$$\frac{V^{2}\sin\ 2\theta}{g} = \ 5\  \cdot \frac{V^{2}\sin^{2}\theta}{2g}$$

$$2\ \sin\ \theta\ \cos\ \theta\  = \frac{5}{2}sin²\theta$$

Divide both sides by sin θ (sin θ ≠ 0):

$$2\ cos\ \theta\  = \frac{5}{2}\sin\ \theta$$

$$\tan\ \theta\  = \frac{4}{5}$$

$$\therefore\ \theta\  = \ tan^{- 1}\left( \frac{4}{5} \right)$$

1.  A particle is projected from the origin at $30$ m/s and passes
    through $(30,20)$.\
    It is given that $\overset{̰}{r} = \left( \begin{array}{r}
    30\cos\theta t \\
     - 5t^{2} + 30\sin\theta t
    \end{array} \right)$\
    Show that $\tan^{2}\theta - 6\tan\theta + 5 = 0$ and find both
    possible angles.

$$x = 30cos\theta\, t = 30$$

$$t = \frac{1}{\cos\theta}$$

$$y = 20:\ 30sin\theta \cdot \frac{1}{\cos\theta} - 5\left( \frac{1}{\cos\theta} \right)^{2} = 20$$

$$30tan\theta - 5\sec^{2}\theta = 20$$

$$30tan\theta - 5\left( 1 + \tan^{2}\theta \right) = 20$$

$$5\tan^{2}\theta - 30tan\theta + 25 = 0$$

$$\tan^{2}\theta - 6tan\theta + 5 = 0$$

$$\left( \tan\theta - 1 \right)\left( \tan\theta - 5 \right) = 0$$

$$\theta = 45{^\circ}\text{ or }\tan^{- 1}5 \approx 78.7{^\circ}$$

3.  A projectile was fired from level ground and just cleared a wall 10
    m high and 20 m from the point of projection. If the angle of
    projection was 36°, find the initial speed to the nearest integer.\
    Note: the height of the wall is not necessarily the maximum height.

$$Let\ \alpha\  = \ 36{^\circ},\ horizontal\ distance\ x\  = \ 20,\ vertical\ height\ y\  = \ 10.$$

$$x\  = \ V\ \cos\ \alpha\  \cdot \ t\ $$

$$y\  = \ V\ \sin\ \alpha\  \cdot \ t\  - \frac{1}{2}(10)t^{2}$$

$$From\ x\  = \ 20:\ t\  = \frac{20}{V\ \cos\ 36{^\circ}}$$

$$Sub\ into\ y\  = \ 10:$$

$$10\  = \ V\ \sin\ 36{^\circ}\  \cdot \ \left( \frac{20}{V\ \cos\ 36{^\circ}} \right) - \ 5\left( \frac{20}{V\ \cos\ 36{^\circ}} \right)^{2}$$

$$10\  = \ 20\ \tan\ 36{^\circ}\  - \frac{5(400)}{V^{2}cos^{2}36{^\circ}}$$

$$\frac{5(400)}{V^{2}cos^{2}36{^\circ}} = \ 20\ \tan\ 36{^\circ}\  - \ 10$$

$$V^{2} = \frac{2000}{cos^{2}36{^\circ}\left( 20\ \tan\ 36{^\circ}\  - \ 10 \right)}$$

$$V²\  \approx \ 674.5$$

$$V\  \approx \ 26\ m/s$$

    Taking the origin at the base, it is projected at $20$ m/s and lands

    Find the two possible angles to 1 d.p.

$$x = 20cos\theta\, t = 40$$

$$t = \frac{2}{\cos\theta}$$

$$y = - 5t^{2} + 20sin\theta\, t + 15$$

$$\text{land }y = 0:\  - 5\left( \frac{2}{\cos\theta} \right)^{2} + 20sin\theta \cdot \frac{2}{\cos\theta} + 15 = 0$$

$$- 20\sec^{2}\theta + 40tan\theta + 15 = 0$$

$$- 20\left( 1 + \tan^{2}\theta \right) + 40tan\theta + 15 = 0$$

$$20\tan^{2}\theta - 40tan\theta + 5 = 0$$

$$4\tan^{2}\theta - 8tan\theta + 1 = 0$$

$$= 1 \pm \frac{\sqrt{3}}{2}$$

$$\theta \approx 61.8{^\circ}\text{ or }7.6{^\circ}$$

5.  A particle is projected with initial speed 50 m/s. On its flight it
    passes through a point P at horizontal distance 100 m and vertical
    distance 25 m from the point of projection.

    a.  Show that the angle of projection α is
        $\tan^{- 1}\left( \frac{1}{2} \right)$ or
        $\tan^{- 1}\left( \frac{9}{2} \right).$

    b.  For each angle, find:

        i.  the time to reach $P,$

        ii. the velocity at $P$ (to 1 d.p. where necessary).

a\) $x\  = \ 50\cos\alpha t\  = \ 100$

$$\ t\  = \frac{2}{\cos\alpha}$$

$$Sub\ into\ y\  = \ 50\ \sin\ \alpha\ t\  - \ 5t²\  = \ 25:$$

$$50\ \sin\ \alpha\  \cdot \ \left( \frac{2}{\cos\alpha} \right)\  - \ 5\left( \frac{4}{cos^{2}\alpha} \right)\  = \ 25$$

$$100\ \tan\ \alpha\  - \ 20\ \sec ²\ \alpha\  = \ 25$$

$$100\ \tan\ \alpha\  - \ 20(1\  + \ \tan ²\ \alpha)\  = \ 25$$

$$20\ \tan ²\ \alpha\  - \ 100\ \tan\ \alpha\  + \ 45\  = \ 0$$

$$4\ \tan ²\ \alpha\  - \ 20\ \tan\ \alpha\  + \ 9\  = \ 0$$

$$(2\ \tan\ \alpha\  - \ 1)(2\ \tan\ \alpha\  - \ 9)\  = \ 0$$

$$\tan\ \alpha\  = \frac{1}{2}or\ \tan\ \alpha\  = \frac{9}{2}$$

$$\therefore\ \alpha\  = \ \tan^{- 1}\left( \frac{1}{2} \right)\ or\ \alpha\  = \ \tan^{- 1}\left( \frac{9}{2} \right)$$

b\)

$$\alpha\  = \tan^{- 1}\left( \frac{1}{2} \right):\ \sin\ \alpha\  = \frac{1}{\sqrt{5}},\ \cos\ \alpha\  = \frac{2}{\sqrt{5}}$$

$$i)\ t\  = \frac{2}{\cos\alpha}\  = \frac{2}{\frac{2}{\sqrt{5}}}\  = \ \sqrt{5}\  \approx \ 2.2\ s$$

$$ii)\ \dot{x} = \ 50\ \cos\ \alpha\  = \frac{100}{\sqrt{5}} = \ 20\sqrt{5}$$

$$\dot{y} = \ 50\ \sin\ \alpha\  - \ 10t\  = \frac{50}{\sqrt{5}} - \ 10\sqrt{5} = \ 10\sqrt{5} - \ 10\sqrt{5} = \ 0$$

Speed = $20\sqrt{5}\  \approx \ 44.7\ m/s\$, directed horizontally.

($P$ is the maximum height)

$$\alpha\  = \ \tan^{- 1}\left( \frac{9}{2} \right):\ \sin\ \alpha\  = \frac{9}{\sqrt{85}},\ \cos\ \alpha\  = \frac{2}{\sqrt{85}}$$

$$i)\ t\  = \frac{2}{\cos\alpha}\  = \frac{2}{\frac{2}{\sqrt{85}}} = \ \sqrt{85} \approx \ 9.2\ s$$

$$ii)\ \dot{x} = \ 50\ \cos\ \alpha\  = \frac{100}{\sqrt{85}}$$

$$\dot{y} = \ 50\ \sin\ \alpha\  - \ 10t\  = \frac{450}{\sqrt{85}} - \ 10\sqrt{85} = \frac{450\  - \ 850}{\sqrt{85}} = \  - \frac{400}{\sqrt{85}}$$

| - **Investigation** General Cartesian equation of the path of a       |
|   projectile                                                          |
+=======================================================================+
| We have expressed the horizontal and vertical displacements in terms  |
| of time $t$.                                                          |
|                                                                       |
| This is a **parametrically** defined curve, with parameter $t$.       |
|                                                                       |
| $$x = Vt\cos\theta\ \ and\ \ y = - \frac{1}{2}gt^{2} + Vt\sin\theta$$ |
|                                                                       |
| It would be useful to have a **Cartesian** equation that relates $x$  |
| and $y$ directly.                                                     |
|                                                                       |
| Eliminate the parameter $t$ to show that                              |
|                                                                       |
| $$y = - \frac{gx^{2}}{2V^{2}}(1 + \tan^{2}\theta) + x\tan\theta$$     |
+-----------------------------------------------------------------------+
| - **Cartesian Equation of Projectile Motion**                         |
+-----------------------------------------------------------------------+
| $${y = - \frac{gx^{2}}{2V^{2}}\sec^{2}\theta + x\tan\theta            |
| }{y = - \frac{gx^{2}}{2V^{2}}(1 + \tan^{2}\theta) + x\tan\theta}$$    |
|                                                                       |
| Do not memorise these equations. They will either be given, or you    |
| will need to derive them.                                             |
|                                                                       |
| The Cartesian equation can be very useful as it relates $x$ and $y$   |
| directly:                                                             |
|                                                                       |
| - The instantaneous direction of the projectile is given by           |
|   $\tan\phi = \frac{dy}{dx}$.                                         |
|                                                                       |
| - The range is given by setting $y = 0$.                              |
|                                                                       |
| - The maximum height can be found by setting $\frac{dy}{dx}$ $= 0$,   |
|   or by using $x = -$ $\frac{b}{2a}$                                  |
+-----------------------------------------------------------------------+

+-----------------------------------------------------------------------------------------------------------------------------------------------------------------+
