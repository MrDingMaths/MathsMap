# Syllabus Content

**ME1-12-05** applies calculus to solve problems involving polynomials,
further rates of change, areas and volumes and differential equations

**Multiplicity of zeroes of polynomial functions**

- Use the product rule to prove that if $P(x)$ has a zero, $\alpha$, of
  multiplicity $m > 1$, then $\alpha$ is a zero of $P'(x)$ of
  multiplicity $m - 1$, and use the result to determine the multiplicity
  of a discovered zero of $P(x)$ and solve related polynomial problems

- Prove that if $\alpha$ is a zero of multiplicity $m$, meeting the
  $x$-axis at $A(\alpha,0)$, then if $m = 1$, the curve crosses the
  $x$-axis at $A$ at an acute or obtuse angle; if $m > 1$ is even, the
  curve is tangent to the $x$-axis at $A$ and does not cross it; if
  $m > 1$ is odd, the curve has a horizontal inflection at $A$

- Graph a polynomial function in factored form, identifying its turning
  points and points of inflection if possible, and explore its behaviour
  as $x \rightarrow \infty$ and $x \rightarrow - \infty$, verifying the
  shape of the graph using graphing applications

Foundation

1.  Consider the polynomial $P(x)\  = \ x³\  - \ 4x²\  - \ 3x\  + \ 18$.

    a.  Show that $P(3)$ and $P'(3)$ are both zero.

P(3) = 27 - 36 - 9 + 18 = 0

P\'(x) = 3x² - 8x -- 3

P\'(3) = 27 - 24 -- 3 = 0

b.  What can be deduced from the results in part **a**?

x = 3 is a root of multiplicity at least 2, which means (x - 3)² is a
factor of P(x).

c.  Hence use the sum of zeroes to find all the zeroes of $P(x)$.

Let roots be $3,\ 3,\ \alpha$

$$3 + 3 + \alpha = - \frac{b}{a}$$

$$6 + \alpha = 4$$

$$\alpha = - 2$$

d.  Hence factor P(x).

$$P(x) = (x\  - \ 3)^{2}(x\  + \ 2)$$

2.  Consider the polynomial
    $P(x)\  = \ x⁴\  + \ 8x³\  + \ 18x²\  + \ 16x\  + \ 5.$

    a.  Show that $P( - 1)$, $P'( - 1)$ and $P''( - 1)$ are all zero.

P(-1) = 1 - 8 + 18 - 16 + 5 = 0

P\'(x) = 4x³ + 24x² + 36x + 16

P\'(-1) = -4 + 24 - 36 + 16 = 0

P\'\'(x) = 12x² + 48x + 36

P\'\'(-1) = 12 - 48 + 36 = 0

b.  What can be deduced from the results in part **a**?

$x = - 1$ is a root of multiplicity at least 3, so (x + 1)³ is a factor
of P(x).

c.  Hence use the product of zeroes to find all the zeroes of $P(x).$

Let roots be $- 1, - 1, - 1,\alpha$

$$- 1 \times - 1 \times - 1 \times \alpha = \frac{e}{a}$$

$$- \alpha = 5;\therefore\alpha = - 5$$

d.  Hence factor $P(x).$

P(x) = (x + 1)³(x + 5)

3.  The polynomial $P(x)\  = \ x³\  - \ 27x\  + \ 54$ has a double zero.

    a.  Find the zeroes of $P'(x)$.

P\'(x) = 3x² - 27 = 0

x = ±3

b.  Determine which of the zeroes of $P'(x)\$is the double zero of
    $P(x)$.

P(3) = 0

P(-3) = 108 ≠ 0

So $x = 3$ is a double zero.

c.  Find the remaining simple zero of $P(x)$.

Let roots be $3,\ 3,\ \alpha$

$$3 + 3 + \alpha = - \frac{b}{a}$$

$$6 + \alpha = 0$$

$$\therefore\alpha = - 6$$

4.  The polynomial $P(x)\  = \ 2x³\  + \ 5x²\  - \ 4x\  - \ 12$ has a
    double zero.

    a.  Find the double zero.

$$x = 2$$

b.  Find the remaining simple zero, and hence factor $P(x)$.

$$Remaining\ simple\ zero:\ \frac{3}{2}$$

$$P(x) = (x + 2)^{2}(2x - 3)$$

5.  Let $P(x) = 4x³ - 8x² - 11x - 3$.

    a.  Show that $x = -$ $\frac{1}{2}$ is a double root of $P(x)$.

$$P'(x) = 12x^{2} - 16x - 11$$

$$P'\left( - \frac{1}{2} \right) = 0$$

b.  Hence, find the third root.

$$x = - 3$$

6.  Show that if $ax^{3} + bx^{2} + cx + d = 0$ has a triple root, then
    it must occur at $x = -$ $\frac{b}{3a}$

P\'(x) = 3ax² + 2bx + c

P\'\'(x) = 6ax + 2b

Let $P^{''}(x) = 0$

$$6ax + 2b = 0$$

$$x = - \frac{b}{3a}$$

7.  The polynomial
    $P(x)\  = \ 8x⁴\  - \ 28x³\  + \ 30x²\  - \ 13x\  + \ 2$ has a
    triple zero.

    a.  Find the triple zero.\
        Hint: try fractions.

$$x = \frac{1}{2}$$

b.  Find the remaining simple zero, and hence factor $P(x)$.

Remaining simple zero is $x = 2$

$$P(x) = (2x - 1)^{3}(x - 2)$$

Development

8.  Find the equation of the quartic polynomial equation with a triple
    root at $x = 3$,\
    single root at $x\  = \  - 2\$and $y$-intercept $- 27$.

$$P(x) = k(x - 3)^{3}(x + 2)$$

Sub in $(0, - 27)$ to find $k$

$$- 27 = k(x - 3)^{3}(x + 2)$$

$$- 27 = k( - 3)^{3}(2)$$

$$\therefore k = \frac{1}{2}$$

$$P(x) = \frac{1}{2}(x - 3)^{3}(x + 2)$$

9.  The polynomial $P(x) = px⁴ + qx³ - 2x² + 4x + 6\$has a double root
    at $x = - 1$.\
    Find the value of $p$ and $q$.

$$P( - 1) = p - q - 2 - 4 + 6 = 0$$

$$P'(x) = 4px^{3} + 3qx^{2} - 4x + 4$$

$$P'( - 1) = - 4p + 3q + 4 = 0$$

Solving simultaneously,

$$p = q = 8$$

P\'(x) = 4x³ - 30x² + 68x -- 42

P\'(3) = 108 - 270 + 204 -- 42 = 0

Since both P(3) = 0 and P\'(3) = 0, x = 3 is a double root.

b.  Hence solve the equation.

Either by using sum & product of roots or by dividing by
