# Syllabus Content

**MAV-12-02** models, analyses and solves practical problems involving
functions and their transformations

**Transformations of trigonometric functions**

- Use graphing applications to explore reflections, translations and
  dilations of, and confirm that the principles of transformations hold
  for, the trigonometric functions $f(x) = \sin x$, $f(x) = \cos x$ and
  $f(x) = \tan x$ 

- Apply the principles of transformations to graph transformations of
  the trigonometric functions $f(x) = \sin x$, $f(x) = \cos x$ and
  $f(x) = \tan x$, describe which transformations have been applied to
  $y = f(x)$ and determine the domain, range, horizontal translation,
  vertical translation, dilation factor, period and amplitude when
  applicable

- Solve equations and graphical problems involving transformations of
  the trigonometric functions $f(x) = \sin x$*,* $f(x) = \cos x$ or
  $f(x) = \tan x$, using graphing applications or otherwise, within a
  specified domain

+===================================================================+
| In maths, we describe waves using the terms:                      |
|                                                                   |

|                                                                   |
| - Amplitude                                                       |
|                                                                   |

| **Vertical translations and mean position**                                                                                                                                      |
|                                                                                                                                                                                  |
| The **mean position** is the average $y$-value of the wave.                                                                                                                      |
|                                                                                                                                                                                  |
| - $y = \sin x$ and $y = \cos x$ have mean position 0.                                                                                                                            |
|                                                                                                                                                                                  |
| - $y = \sin{(x)} + c\$and $y = \cos{(x)} + c$ have mean position $c$.                                                                                                            |
|                                                                                                                                                                                  |
| - $y = \tan x$ does not have a mean position.                                                                                                                                    |
|                                                                                                                                                                                  |
| - $y = \tan(x) + c$ shifts the graph vertically.                                                                                                                                 |
|                                                                                                                                                                                  |
| **Vertical dilations and amplitude**                                                                                                                                             |
|                                                                                                                                                                                  |
| - The **amplitude** of a wave is the distance from the mean position to the maximum value.                                                                                       |
|                                                                                                                                                                                  |
| - $y = \sin x$ and $y = \cos x$ both have amplitude 1.                                                                                                                           |
|                                                                                                                                                                                  |
| - $y = k\sin x$ and $y = k\cos x$ both have amplitude $k$.                                                                                                                       |
|                                                                                                                                                                                  |
| - $y = \tan x$ does not have an amplitude.                                                                                                                                       |
|                                                                                                                                                                                  |
| - $y = k\tan x$ changes the steepness of the graph.                                                                                                                              |
|                                                                                                                                                                                  |
| **Horizontal dilations and period**                                                                                                                                              |
|                                                                                                                                                                                  |
| The **period** of a wave function is the length of the smallest repeating unit.                                                                                                  |
|                                                                                                                                                                                  |
| - $y = \sin x$ and $y = \cos x$ both have period $2\pi$.                                                                                                                         |
|                                                                                                                                                                                  |
| - $y = \sin(ax)$ and $y = \cos(ax)$ both have period $\frac{2\pi}{a}$.                                                                                                           |
|                                                                                                                                                                                  |
| - $y = \tan x$ has period $\pi$.                                                                                                                                                 |
|                                                                                                                                                                                  |
| - $y = \tan(ax)$ has period $\frac{\pi}{a}$.                                                                                                                                     |
|                                                                                                                                                                                  |
| **Horizontal translations and phase**                                                                                                                                            |
|                                                                                                                                                                                  |
| The **phase** of a wave is the horizontal shift from the original position.                                                                                                      |
|                                                                                                                                                                                  |
| - $y = \sin x$, $y = \cos x$, $y = \tan x$ have phase 0.                                                                                                                         |
|                                                                                                                                                                                  |
| - $y = \sin{(x - b)},$ $y = \cos{(x - b)},$ $y = \tan{(x - b)}$ have phase $- b$ and shift right by $b$ units.                                                                   |

11. **HSC Advanced Sample Question Band 5**

> $f(x) = 2\sin(2x)$ is defined in the domain
> $\left\{ x:\frac{\pi}{8} \leq x \leq \frac{\pi}{3} \right\}$.
>
> What is the range of this function?

Maximum occurs when $\sin(2x) = 1$

$$2x = \frac{\pi}{2}$$

$$x = \frac{\pi}{4}$$

Therefore, maximum value of $2$ is in the domain.

Check endpoints: $y\left( \frac{\pi}{8} \right) = \sqrt{2}$

$$y\left( \frac{\pi}{3} \right) = \sqrt{3}$$

So, the minimum value is $\sqrt{2}$

$$\therefore\left\lbrack \sqrt{2},2 \right\rbrack$$
