# Syllabus Content

**MAV-12-03** uses arithmetic and geometric sequences and series to
model and solve problems

**Sequences and series**

- Define a sequence as an ordered list of objects

- Use the notation $a_{n}$, where $n$ is a positive integer, to
  represent the $n$th term of a sequence

- Distinguish between a finite sequence $a_{1},\ a_{2},\ldots,\ a_{m}$
  that terminates at its $m$th term, for some whole number $m \geq 1$,
  and an infinite sequence $a_{1},\ a_{2},\ldots$ that never terminates

- Define the $n$th partial sum $S_{n}$ of a sequence
  $a_{1},\ a_{2},\ldots$ to be the sum of the first $n\$terms of the
  sequence: $S_{n} = a_{1} + a_{2} + \ldots + a_{n}$, for all whole
  numbers $n \geq 1$

- Define a series formally as the sum of the terms of an infinite
  sequence and use the notation $a_{1} + a_{2} + a_{3} + \ldots$ for the
  series corresponding to the sequence $a_{1},a_{2},a_{3}\ldots$

- Use summation notation to represent the sum of terms $a_{i}$ to
  $a_{j}$ of a sequence where $j > i$,
  $\sum_{k = i}^{j}a_{k} = a_{i} + a_{i + 1\ } + a_{i + 2} + \,\ldots\, + \, a_{j - 1} + a_{j}$

# Sequences

+-------------------------------------------------------------------+
| - **Sequences**                                                   |
+===================================================================+
| A sequence is a collection of terms arranged in a specific order, |
| where each term is obtained according to a rule.                  |
|                                                                   |
| Some examples of sequences:                                       |
|                                                                   |
| 2, 4, 6, 8, 10 1, 4, 9, 16, 25 1, 2, 4, 8, 16 1, 1, 2, 3, 5, 8    |
+-------------------------------------------------------------------+

+-------------------------------------------------------------------+
| - **Terms of a Sequence**                                         |
+===================================================================+
| We express terms of a sequence as                                 |
| $a_{1},\ a_{2},\ a_{3},\ldots a_{n}$ where:                       |
|                                                                   |
| $a_{1}$ is the 1^st^ term                                         |
|                                                                   |
| $a_{2}$ is the 2^nd^ term                                         |
|                                                                   |
| $a_{n}$ is the $n$^th^ term                                       |
|                                                                   |
| - $n$ must be a positive integer $n \geq 1$                       |
+-------------------------------------------------------------------+

+-------------------------------------------------------------------+
| - **Recursive Rule**                                              |
+===================================================================+
| A **recursive** **rule**, or **term-to-term rule**, describes how |
| to find the next term in a sequence by using the current term. It |
| shows the operation(s) you perform on one term to get the         |
| following term.                                                   |
|                                                                   |
| E.g. $a_{n} = a_{n - 1} + 3$ means: "to get the next term, add 3  |
| to the previous term"                                             |
+-------------------------------------------------------------------+

+-------------------------------------------------------------------+
| - $\mathbf{n}$**^th^ Term Formula**                               |
+===================================================================+
| The $\mathbf{n}$**^th^ term formula,** or **position-to-term      |
| rule**, directly calculates any term using its position number in |
| the sequence.                                                     |
|                                                                   |
| This is more powerful than a term-to-term rule because you can    |
| instantly find any term without working through the entire        |
| sequence.                                                         |
|                                                                   |
| E.g., $a_{n} = 3n + 5$ means: "to get the $n$th term, multiply by |
| 3 and add 5"                                                      |
+-------------------------------------------------------------------+

# Series and Partial Sums

+-------------------------------------------------------------------+
| - **Series**                                                      |
+===================================================================+
| For a sequence $a_{1},\ a_{2},\ a_{3}\ldots$, the **series** is   |
| the **infinite sum** of all the terms of a sequence.              |
|                                                                   |
| $$a_{1} + a_{2} + a_{3} + \ldots$$                                |
+-------------------------------------------------------------------+

+-------------------------------------------------------------------+
| - **Partial Sum of a Series**                                     |
+===================================================================+
| The **partial sum** of a series is:                               |
|                                                                   |
| $$S_{n} = a_{1} + a_{2} + a_{3} + \ldots + a_{n}$$                |
+-------------------------------------------------------------------+

+-------------------------------------------------------------------------------+
| - **Example** Finding the partial sum of a series.                            |
+===================================+===========================================+
| A series is                       | A series is                               |
| $5 + 15 + 25 + 35 + 45\ldots$     | $1 + 4 + 9 + 16 + 25 + 36\ldots$          |
|                                   |                                           |
| Find $S_{4}$                      | Find $S_{7}$                              |
|                                   |                                           |
| $$S_{4} = 5 + 15 + 25 + 35$$      | $$S_{7} = 1 + 4 + 9 + 16 + 25 + 36 + 49$$ |
|                                   |                                           |
| $= 80$                            | $= 140$                                   |
+-----------------------------------+-------------------------------------------+

+------------------------------------------------------------------------------------------------+
| - **Sigma Notation**                                                                           |
+================================================================================================+
| We use sigma notation to write the partial sum of a sequence using its $n$^th^ term formula:   |
|                                                                                                |
| $$\sum_{n\  = \ i}^{j}a_{n}\  = \ a_{i} + a_{i + 1} + a_{i + 2} + \ldots + a_{j - 1} + a_{j}$$ |
|                                                                                                |
| This means starting from $n = i\$and ending at $n = j$, add up all the terms of the sequence   |
| $a_{n}$.                                                                                       |
+------------------------------------------------------------------------------------------------+
