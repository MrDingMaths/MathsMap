# Syllabus Content

**MAV-12-03** uses arithmetic and geometric sequences and series to
model and solve problems

**Geometric sequences and series**

- Define a sequence $a_{n}$ to be a geometric sequence, or geometric
  progression (GP), if every ratio $\frac{a_{n}}{a_{n - 1}}$ of
  successive terms is the same where $n \geq 2$, that is
  $a_{n} = ra_{n - 1}$ for some non-zero real number $r$ called the
  common ratio

- Develop the formula $a_{n} = ar^{n - 1}$ for the $n$th term of a GP,
  where $a$ is the first term and $n \geq 1$, and use it to solve
  problems

- Recognise that $a_{n}$ is an exponential function of $n$ in a GP

- Prove by expansion
  $(x - 1)\left( x^{n - 1} + x^{n - 2} + \ldots + x^{2} + x + 1 \right) = x^{n} - 1\$for
  whole numbers $n \geq 1$

- Develop the formula $S_{n} = \frac{a(1 - r^{n})}{1 - r}$ for the sum
  of the first $n$ terms of a GP where $r \neq 1$, and use this formula
  to solve problems

- Examine the behaviour of $a_{n}$ and $S_{n}$ as $n \rightarrow \infty$
  for a GP when $|r| < 1$

- Develop the formula $S_{\infty} = \frac{a}{1 - r}$ for the limiting
  sum of a geometric series with $|r| < 1$, and use this formula to
  solve problems

- Apply the formulas for geometric sequences and series to model and
  solve growth and decay problems where a quantity is an exponential
  function of time

# Geometric Sequences

+-------------------------------------------------------------------+
| - **Geometric Sequences**                                         |
+===================================================================+
| A **geometric sequence**, also called a **geometric progression** |
| or **exponential sequence,** is a sequence where there is a       |
| **common ratio** between each pair of consecutive terms.          |
|                                                                   |
| For example:                                                      |
|                                                                   |
| $$4,\ 12,\ 36,\ 108,\ 324\ldots$$                                 |
|                                                                   |
| The common ratio ($r$) is the same value that you multiply each   |
| term to get the next term.                                        |
|                                                                   |
| It can be:                                                        |
|                                                                   |
| - Greater than one (sequence increases, also called exponential   |
|   growth)                                                         |
|                                                                   |
| - Between zero and one (sequence decreases, also called           |
|   exponential decay)                                              |
|                                                                   |
| - Negative (sequence alternates between positive and negative     |
|   terms)                                                          |
|                                                                   |
| We check whether a sequence is geometric by finding the ratio     |
| between consecutive terms and seeing if they are constant.        |
|                                                                   |
| $$\frac{a_{n}}{a_{n - 1}} = r$$                                   |
+-------------------------------------------------------------------+

+-------------------------------------------------------------------------------------------------------------------------------------------+

# Partial Sum of a Geometric Series

+-------------------------------------------------------------------+
| - **Investigation** Partial sum of a Geometric Series.            |
+===================================================================+
| Consider the partial sum of a geometric sequence below:           |
|                                                                   |
| $$S_{n} = a + ar + ar^{2} + ar^{3} + \ldots ar^{n - 1}$$          |
|                                                                   |
| A quick way of finding the partial sum is to multiply both sides  |
| by $r$.                                                           |
|                                                                   |
| $rS_{n} =$ ..........................................             |
|                                                                   |
| Subtract the two equations above:                                 |
|                                                                   |
| $rS_{n} - S_{n} =$ ..........................................     |
|                                                                   |
| Then rearrange for $S_{n}$                                        |
|                                                                   |
| $S_{n} =$ ..........................................              |
|                                                                   |
| By multiplying the numerator and denominator, we can get an       |
| alternate formula:                                                |
|                                                                   |
| $S_{n} =$ ..........................................              |
+-------------------------------------------------------------------+

+-----------------------------------------------------------------------------------------------------------+
| - **The Partial Sum of a Geometric Series.**                                                              |
+=====================================================+=====================================================+
| $$S_{n} = \frac{a\left( r^{n} - 1 \right)}{r - 1}$$ | $$S_{n} = \frac{a\left( 1 - r^{n} \right)}{1 - r}$$ |
+-----------------------------------------------------+-----------------------------------------------------+
| - Usually $n$ is not given, you need to find it using $a_{n} = ar^{n - 1}$                                |
+-----------------------------------------------------------------------------------------------------------+

+-------------------------------------------------------------------+
| - **Example** Finding the partial sum of an arithmetic series.    |
+===================================================================+
| Consider the geometric series: $2 - 6 + 18 - \ldots$              |
|                                                                   |
| Find $S_{6}$                                                      |
|                                                                   |
| $$a = 2,\ r = - 3$$                                               |
|                                                                   |
| $$S_{6} = \frac{2\left( ( - 3)^{6} - 1 \right)}{( - 3) - 1}$$     |
|                                                                   |
| $= - 364$                                                         |
