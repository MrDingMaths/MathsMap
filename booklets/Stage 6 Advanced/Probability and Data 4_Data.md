# Syllabus Content

**MAV-11-10** displays and analyses datasets using summary statistics
and graphical representations

**Data**

- Define a random variable as a variable whose value is the outcome of a
  random experiment

- Compare discrete random variables with continuous random variables,
  describe their differences, and give practical examples of each

- Organise finite datasets using a table or a spreadsheet, listing the
  values, frequency, relative frequency, cumulative frequency, and
  cumulative relative frequency

- Graph the frequency, relative frequency, and cumulative frequency
  histograms and polygons of datasets, using spreadsheets or graphing
  applications, and identify the mode and median from the graphs, and
  from tables

- Use the relative frequency to estimate the probability of results in
  experiments

# Relative Frequency and the Law of Large Numbers

+--------------------------------------------------------------------+
| - **Relative Frequency**                                           |
+====================================================================+
| **Relative frequency** of an outcome is how often the outcome      |
| happens compared to the total number of trials.                    |
|                                                                    |
| $$\begin{matrix}                                                   |
| Relative \\                                                        |
| frequency                                                          |
| \end{matrix} = \frac{Frequency\ of\ outcome\ }{Total\ frequency}$$ |
|                                                                    |
| - The relative frequency of all outcomes must add to 1.            |
|                                                                    |
| - When graphing, it is a good idea to express relative frequency   |
|   as a decimal.                                                    |
+--------------------------------------------------------------------+
