# Radians

- Ancient Babylonians invented degrees as an arbitrary unit.\
  To connect degrees to their circular origin, we use **radians**.

- A **radian** is the **angle formed** when an **arc equals exactly 1
  radius** in length.

- Radians represent a **dimensionless ratio** comparing **arc length**
  to **radius**.

- If an angle does not have the degree symbol, assume it is in radians.

> $$\therefore\angle ADB = \frac{2\pi}{3}$$
>
> $$\tan\frac{\pi}{6} = \frac{4}{BC};\ \ \ \ \ BC = 4\sqrt{3}$$

>
> $$AC = \sqrt{13^{2} - \left( 4\sqrt{3} \right)^{2}} = 11$$

>

> on a straight line and $AB = AD = 3$ m.

a.  Show that $\angle DAB =$ $\frac{2\pi}{3}$

> $$\angle DBA = \pi - \frac{5\pi}{6} = \frac{\pi}{6}\ (angles\ on\ a\ straight\ line)$$
>
> $$\angle BDA = \angle DBA = \frac{\pi}{6}\ (angles\ opposite\ equal\ sides\ of\ isosceles\ triangle)$$
>
> $$\angle DAB = \pi - \frac{\pi}{6} - \frac{\pi}{6}\ (angle\ sum\ of\ a\ triangle)$$
>
> $$= \frac{2\pi}{3}$$

b.  Find the length of BD.

Using cosine rule

$$BD = \sqrt{3^{2} + 3^{2} - 2(3)(3)\cos\frac{2\pi}{3}}$$

$$= \sqrt{27}\ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ $$

$= 3\sqrt{3}\ m\ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \$

c.  Hence find the area of the garden.

$$Area\ \Delta ABD = \frac{1}{2}(3)(3)\sin\frac{2\pi}{3} = \frac{9\sqrt{3}}{4}$$

Area of sector
$= \frac{1}{2}r^{2}\theta = \frac{1}{2}\left( \sqrt{27} \right)^{2}\left( \frac{5\pi}{6} \right)$

$$= \frac{45}{4}\pi\ m^{2}\ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ $$
