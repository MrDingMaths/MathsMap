+-----------------------------------+-----------------------------------------------------------------------+

+-----------------------------------+-----------------------------------------------------------------------+

+-----------------------------------+-----------------------------------------------------------------------+

Mastery

3.  If $y\  = \ eˣ\ \sin\ x$, find $y'$ and $y''$, and show that
    $y''\  - \ 2y'\  + \ 2y\  = \ 0$.

$$y' = \ eˣ\sin x + \ eˣ\cos x$$

$$= \ eˣ(sin\ x\  + \ cos\ x)$$

$$y'' = \ eˣ\left( \sin x + \cos x \right) + \ eˣ\left( \cos x - \sin x \right)$$

$$= \ eˣ(2\ cos\ x)$$

$$y'' - \ 2y' + \ 2y\  = \ 2eˣ\ cos\ x\  - \ 2eˣ(sin\ x\  + \ cos\ x) + \ 2eˣ\ sin\ x$$

$$= \ 2eˣ\ cos\ x\  - \ 2eˣ\ sin\ x\  - \ 2eˣ\ cos\ x\  + \ 2eˣ\ sin\ x\ $$

$$= \ 0$$

4.  If $y\  = \ e⁻ˣ\ \cos\ x$, find $y'$ and $y''$, and show that
    $y''\  + \ 2y'\  + \ 2y\  = \ 0$.

$$y'\  = \  - e⁻ˣ\ cos\ x\  + \ e⁻ˣ( - sin\ x)\  = \  - e^{-}ˣ(cos\ x\  + \ sin\ x)$$

$$y''\  = \ e⁻ˣ(cos\ x\  + \ sin\ x)\  + \ \left( - e^{-}ˣ \right)( - sin\ x\  + \ cos\ x)$$

$$= \ e⁻ˣ(cos\ x\  + \ sin\ x)\  - \ e^{-}ˣ(cos\ x\  - \ sin\ x)$$

$$= \ e^{-}ˣ(2\ sin\ x)$$

$$y'' + \ 2y' + \ 2y\  = \ 2e^{-}ˣ\ sin\ x\  + \ 2\left( - e^{-}ˣ \right)(cos\ x\  + \ sin\ x) + \ 2e^{-}ˣ\ cos\ x$$

$$= \ 2e^{-}ˣ\ sin\ x\  - \ 2e^{-}ˣ\ cos\ x\  - \ 2e^{-}ˣ\ sin\ x\  + \ 2e^{-}ˣ\ cos\ x$$

$$= \ 0$$

5.  If
    $f(x)\  = \ \log_{e}\left( \frac{1\  + \ \sin\ x}{\cos\ x} \right)$,
    show that $f’(x)\  = \ \sec\ x$.

$$f(x)\  = \ ln(1\  + \ sin\ x)\  - \ ln(cos\ x)$$

$$f'(x) = \frac{cos\ x}{1\  + \ sin\ x} - \frac{- sin\ x}{cos\ x}$$

$$= \frac{cos\ x}{1\  + \ sin\ x} + \frac{sin\ x}{cos\ x}$$

$$= \frac{cos^{2}x\  + \ sin\ x(1\  + \ sin\ x)}{cos\ x(1\  + \ sin\ x)}$$

$$= \frac{cos^{2}x\  + \ sin\ x\  + \ sin^{2}x}{cos\ x(1\  + \ sin\ x)}$$

$$= \frac{1\  + \ sin\ x}{cos\ x(1\  + \ sin\ x)}$$

$$= \frac{1}{cos\ x}$$

$$= \ sec\ x$$

6.  If $y\  = \ \ln\ (\tan\ 2x)$, show that $\frac{dy}{dx}$
    $= \ 2\ \sec\ 2x\ cosec\ 2x$.

$$\frac{dy}{dx} = \ \left( \frac{1}{tan\ 2x} \right) \times \ sec^{2}2x\  \times \ 2$$

$$= \frac{2\ sec^{2}2x}{tan\ 2x}$$

$$= \frac{2\ sec^{2}2x}{\left( \frac{sin\ 2x}{cos\ 2x} \right)}$$

$$= \frac{2\ sec^{2}2x\  \times \ cos\ 2x}{sin\ 2x}$$

$$= \frac{2\ sec\ 2x}{sin\ 2x}$$

$$= \ 2\ sec\ 2x\ cosec\ 2x$$

7.  The point $P\left( \frac{\pi}{6},\frac{1}{2} \right)$ lies on the
    curve $y\  = \ 2\ \sin\ x\  - \ \cos\ 2x$.

    a.  Show that the tangent at $P$ has equation
        $2\sqrt{3}x\  - \ y\  = \frac{\pi\sqrt{3}}{3} - \frac{1}{2}$.

$$\frac{dy}{dx} = \ 2\ cos\ x\  + \ 2\ sin\ 2x$$

$$At\ x\  = \frac{\pi}{6}:$$

$$\frac{dy}{dx} = \ 2\ cos\left( \frac{\pi}{6} \right) + \ 2\ sin\left( \frac{\pi}{3} \right) = \ 2\left( \frac{\sqrt{3}}{2} \right) + \ 2\left( \frac{\sqrt{3}}{2} \right) = \ 2\sqrt{3}$$

$$Tangent:\ y\  - \frac{1}{2} = \ 2\sqrt{3}\left( x\  - \frac{\pi}{6} \right)\ $$

$$y\  = \ 2\sqrt{3}x\  - \frac{\pi\sqrt{3}}{3} + \frac{1}{2}$$

$$2\sqrt{3}x\  - \ y\  = \frac{\pi\sqrt{3}}{3} - \frac{1}{2} = \frac{\pi\sqrt{3}}{3} - \frac{1}{2}$$

b.  Show that the normal at $P$ has
    equation$\ x\  + \ 2\sqrt{3}y\  = \frac{\pi}{6} + \ \sqrt{3}$.

$$Normal\ has\ gradient\  - \frac{1}{2\sqrt{3}}$$

$$Normal:\ y\  - \frac{1}{2} = \ \left( - \frac{1}{2\sqrt{3}} \right)\left( x\  - \frac{\pi}{6} \right)$$

$$2\sqrt{3}y\  - \ \sqrt{3} = \  - x\  + \frac{\pi}{6}\ $$

$$x\  + \ 2\sqrt{3}y\  = \frac{\pi}{6} + \ \sqrt{3}$$
