**Piecewise-defined functions**

- Interpret piecewise-defined functions, where the function is defined
  differently in different parts of the domain

- Graph piecewise-defined functions involving functions covered in the
  scope of the Mathematics Advanced course, test if they are even or

  paper

- Identify points where piecewise-defined functions and other functions
  are not continuous

- Define a discontinuity of a function informally as a point where the
  function is not continuous

## Piecewise-Defined Function

- A **piecewise** **function** combines more than one function in
  different parts of its domain.

$$f(x) = \left\{ \begin{array}{r}
g(x),\ \ \ \ \ \ \ \ \ x \leq a \\
\ \ h(x),\ \ x > a\ 
\end{array} \right.\ $$

- $f(x)$ is called the piecewise-defined function.

- $g(x)$ and $h(x)$ are called subfunctions.

- To sketch a piecewise-defined function:

1.  Identify the graphs of each subfunction and their domains.

2.  Calculate function values at the boundaries

3.  Sketch each subfunction separately.\
    Decide whether to connect the curves or use open/closed circles.

# Continuous At a Point

- A function is **continuous** at a point if the curve can be drawn
  through the point without lifting the pen off the paper.

- A point where the function is not continuous is called a
  **discontinuity**.

+------------------------------------+
| **Continuity of Piecewise-Defined  |
| Functions**                        |
+====================================+
| Given a piecewise function:        |
|                                    |
| $$f(x) = \left\{ \begin{array}{r}  |
| g(x),\ \ \ \ \ \ \ \ \ x \leq a \\ |
| \ \ h(x),\ \ x > a\                |
| \end{array} \right.\ $$            |
|                                    |
| The function is **continuous** at  |
| $x = a$ if $g(a) = h(a)$.          |
+------------------------------------+
