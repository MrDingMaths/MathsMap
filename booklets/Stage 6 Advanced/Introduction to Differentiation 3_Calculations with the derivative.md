# Syllabus Content

**MAV-11-06** interprets the meaning of the derivative and determines
the derivative of functions to solve problems

**Calculations with the derivative**

- Use the notation $\frac{dy}{dx}$ and $y'$ for the derivative of $y$
  when $y$ is a function of $x$

- Use the notation $\frac{d}{dx}\left( f(x) \right)$ and $f'(x)$ for the
  derivative of a function $f(x)$

- Use the formula $\frac{d}{dx}(x^{n}) = nx^{n - 1}$ for all real values
  of $n$

- Apply the fact that the derivative of a sum is the sum of the
  derivatives: $\frac{d}{dx}\left( f(x) + g(x) \right) = f'(x) + g'(x)$,
  and the derivative of a multiple of a function is the multiple of its
  derivative: $\frac{d}{dx}(k \cdot f(x)) = k \cdot f'(x)$

- Use the rules for differentiation to find equations of tangents and
  normals to a curve at points on the curve

- Find points on a curve where the tangent or normal has a given
  gradient

- Examine and use the relationship between the angle of inclination of a
  line or tangent to a curve, $\theta,$ with the positive $x$-axis, and
  the gradient, $m$, of that line or tangent, and establish that
  $\tan\theta = m$

- Apply the product rule: if $y = uv$, where $u$ and $v$ are both
  differentiable functions of $x$, then
  $\frac{dy}{dx} = u\frac{dv}{dx} + v\frac{du}{dx}$ or if
  $h(x) = f(x)g(x)$ for differentiable functions $f(x)$ and $g(x)$ then
  $h'(x) = f(x)g'(x) + f'(x)g(x)$ 

- Apply the quotient rule: if $y = \frac{u}{v}$, where $u$ and $v$ are
  both functions of $x,$ then
  $\frac{dy}{dx} = \frac{v\frac{du}{dx} - u\frac{dv}{dx}}{v^{2}}$ or if
  $h(x) = \frac{f(x)}{g(x)}$ then
  $h'(x) = \frac{g(x)f'(x) - f(x)g'(x)}{\lbrack{g(x)\rbrack}^{2}}$

- Apply the chain rule: if$\ y$ is a differentiable function of $u$, and
  $u$ is a differentiable function of $x$, then
  $\frac{dy}{dx}\  = \ \frac{dy}{du} \times \frac{du}{dx}$ or if
  $h(x) = f\left( g(x) \right)$ for differentiable functions $f(x)$ and
  $g(x)$ then $h'(x) = f'\left( g(x) \right)g'(x)$

- Identify and apply the product, quotient or chain rule, or a
  combination of the rules, as appropriate to differentiate a given
  function

# Derivative Notation

- Calculus uses different notation systems depending on the situation.

- All systems are acceptable.

- **Lagrange's** notation is the most concise.

- **Leibniz\'s** notation works best when your equations have multiple
  variables, which is common in application problems.

- Leibniz\'s notation appears on your reference sheet.

+-------------------------------------------------------------------+
| - **Chain Rule**                                                  |
+===================================================================+
| **Chain rule:** Multiply the derivative of the outer function by  |
| the derivative of the inner function.                             |
|                                                                   |
| For a composite function $y = g\left( f(x) \right)$, let          |
| $u = f(x)$ and $y = g(u),$ then:                                  |
|                                                                   |
| $$\frac{dy}{dx} = \frac{dy}{du} \times \frac{du}{dx}$$            |
|                                                                   |
| 1.  Identify the inner function $u(x)$ and find $\frac{du}{dx}$.  |
|                                                                   |
| 2.  Identify the outer function $y(u)\$and find $\frac{dy}{du}$.  |
|                                                                   |
| 3.  Multiply the derivatives                                      |
|     $\frac{dy}{dx} = \frac{dy}{du} \times \frac{du}{dx}$.         |
|                                                                   |
| 4.  Substitute all $u$ terms back to $x$.                         |
|                                                                   |
| - When finding $\frac{du}{dx}$ and $\frac{dy}{du}$, write your    |
|   working on the right-hand side to keep the main working area    |
|   clear.                                                          |
+-------------------------------------------------------------------+

+-------------------------------------------------------------------------------------------------+
| - **Example** Differentiate composite functions using the chain rule.                           |
+=============================================================+===================================+
| Differentiate $y = (1 - 6x)^{5}$                            | 1\. Let $u = 1 - 6x$              |
|                                                             |                                   |
| $${3.\ \ \frac{dy}{dx} = \frac{dy}{du} \times \frac{du}{dx} | $\frac{du}{dx}$ $= - 6$           |
| }{= 5u^{4} \times - 6                                       |                                   |
| }{= - 30u^{4}}$$                                            | 2\. Let $y = u^{5}$               |
|                                                             |                                   |
| 4\. $= - 30(1 - 6x)^{4}$                                    | $\frac{dy}{du}$ $= 5x^{4}$        |
+-------------------------------------------------------------+-----------------------------------+
| Differentiate $y = 3\left( 1 - 5x^{2} \right)^{7}$          | 1\. Let $u = 1 - 5x^{2}$          |
|                                                             |                                   |
| $${3.\ \ \frac{dy}{dx} = \frac{dy}{du} \times \frac{du}{dx} | $\frac{du}{dx}$ $= - 10x$         |
| }{= 21u^{6} \times - 10x                                    |                                   |
| }{= - 210xu^{6}}$$                                          | 2\. Let $y = {3u}^{7}$            |
|                                                             |                                   |
| 4\. $= - 210x\left( 1 - 5x^{2} \right)^{6}$                 | $\frac{dy}{du}$ $= 21u^{6}$       |
+-------------------------------------------------------------+-----------------------------------+

# General Power Rule

+-------------------------------------------------------------------------------------------+
| - **General Power Rule**                                                                  |
+===========================================================================================+
| We shortcut chain rule for a function raised to a power $\left( f(x) \right)^{n}\$using   |
| the general power rule.                                                                   |
|                                                                                           |
| $$\frac{d}{dx}\left( f(x)^{n} \right) = nf'(x)\ \left\lbrack f(x) \right\rbrack^{n - 1}$$ |
|                                                                                           |
| 1.  Multiply by the power.                                                                |
|                                                                                           |
| 2.  Reduce the power by 1.                                                                |
|                                                                                           |
| 3.  Multiply by the derivative of the inner function.                                     |
|                                                                                           |
| This is the same as chain rule:\                                                          |
| Multiply the derivative of the outer function by the derivative of the inner function.    |
+-------------------------------------------------------------------------------------------+

+-------------------------------------------------------------------------------------------------------------------------------+
| - **Example** Differentiate using the general power rule.                                                                     |
+=========================================================+=====================================================================+

|                                                         |                                                                     |
| $$\ \ \ \ \  = 200x^{3}\left( 5x^{4} + 2 \right)^{9}$$  |                                                                     |

34. We can bypass the quotient rule by splitting a fraction.

    a.  Find constants $a$ and $b$ such that $\frac{x}{x + 1}$ $= a +$
        $\frac{b}{x + 1}$

$$\frac{x}{x + 1} = \frac{ax + a + b}{x + 1}$$

$$x = ax + a + b$$

By equating coefficients, $a = 1,\ a + b = 0$

$$\therefore a = 1,\ b = - 1$$

b.  Hence differentiate $y =$ $\frac{x}{x + 1}$

From part **a**, this is the same as differentiating

$$y = 1 + \frac{- 1}{x + 1}$$

$$y' = \frac{1}{(x + 1)^{2}}$$

35. Consider $f(x) = \frac{\sqrt{x} + 1}{x - 1}$

    a.  Show that $\frac{\sqrt{x} + 1}{x - 1} = \frac{1}{\sqrt{x} - 1}$.

$$\frac{\sqrt{x} + 1}{x - 1} = \frac{\sqrt{x} + 1}{\left( \sqrt{x} - 1 \right)\left( \sqrt{x} + 1 \right)}$$

$$= \frac{1}{\sqrt{x} - 1}$$

b.  Hence differentiate $f(x).$

$$f'(x) = \frac{- 1}{2\sqrt{x}\left( \sqrt{x} - 1 \right)^{2}}$$

36. Consider the function $f(x) =$ $\frac{x^{2}}{x - 1}$.

    a.  Show that $f(x) = x + 1 +$ $\frac{1}{x - 1}$.

$$\frac{x^{2}}{x - 1} = \frac{x^{2} - 1 + 1}{x - 1}$$

$$= \frac{x^{2} - 1}{x - 1} + \frac{1}{x - 1}$$

$$= \frac{(x - 1)(x + 1)}{x - 1} + \frac{1}{x - 1}$$

$$= x + 1 + \frac{1}{x - 1}$$

b.  Hence, differentiate $f(x).$

$$f'(x) = 1 - \frac{1}{(x - 1)^{2}}$$

37. By first simplifying the expression, differentiate $f(x) =$
    $\frac{x^{4} - 1}{x - 1}$.

$$\frac{x^{4} - 1}{x - 1} = \frac{\left( x^{2} - 1 \right)\left( x^{2} + 1 \right)}{x - 1}$$

$$= \frac{(x - 1)(x + 1)\left( x^{2} + 1 \right)}{x - 1}$$

$$= (x + 1)\left( x^{2} + 1 \right)$$

$$= x^{3} + x^{2} + x + 1$$

$$f'(x) = 3x^{2} + 2x + 1$$

38. Differentiate $f(x) =$ $\frac{x - 1}{\sqrt{x} + 1}$ by first
    rationalising the denominator.

$$\frac{x - 1}{\sqrt{x} + 1} = \frac{(x - 1)\left( \sqrt{x} - 1 \right)}{\left( \sqrt{x} + 1 \right)\left( \sqrt{x} - 1 \right)}$$

$$= \frac{(x - 1)\left( \sqrt{x} - 1 \right)}{x - 1}$$

$$= \sqrt{x} - 1$$

$$f'(x) = \frac{1}{2\sqrt{x}}$$

23. Let $n$ be any positive integer. Show that
    $f(x) = x\left( x^{2} + 1 \right)^{n}$ will never be horizontal at
    any points on the curve.

$$f'(x) = \left( x^{2} + \ 1 \right)ⁿ^{- 1}\left\lbrack x^{2}(1\  + \ 2n) + \ 1 \right\rbrack$$

Tangent horizontal when
$\left( x^{2} + \ 1 \right)ⁿ^{- 1}\left\lbrack x^{2}(1\  + \ 2n) + \ 1 \right\rbrack = 0$

By null factor law, either

$$\left( x^{2} + \ 1 \right)ⁿ^{- 1} = 0\ or\ x^{2}(1\  + \ 2n) + \ 1 = 0$$

$$\left( x^{2} + \ 1 \right)ⁿ^{- 1} \neq 0\ as\ x^{2} + 1 > 0\ $$

$$x^{2}(1\  + \ 2n) + \ 1 \neq 0\ as\ x^{2}(1\  + \ 2n) \geq 0\ $$

Therefore, no real numbers exist for
$\left( x^{2} + \ 1 \right)ⁿ^{- 1}\left\lbrack x^{2}(1\  + \ 2n) + \ 1 \right\rbrack = 0$,
tangent can never be horizontal.

If you check the sketch of $f'(x)$, there are no turning points or
points of inflection.
