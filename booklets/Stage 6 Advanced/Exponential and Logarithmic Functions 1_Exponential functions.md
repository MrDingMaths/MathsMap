# Syllabus Content

**MAV-11-08** analyses graphs of exponential and logarithmic functions

**Exponential functions**

- Graph the exponential functions $y = k\left( a^{x} \right)$ and
  $y = k\left( a^{- x} \right)$ for constants $a$ and $k$ where $a > 0$,
  $a \neq 1$ and $k \neq 0$, and identify its asymptote, $y$-intercept,
  domain and range

- Describe the behaviour of $y = k\left( a^{x} \right)$ and
  $y = k\left( a^{- x} \right)$ as $x \rightarrow \infty$ and
  $x \rightarrow - \infty$

- Examine the gradient of the tangent to the curve $y = a^{x}$ at its
  $y$-intercept for varying values of $a$, and verify using graphing
  applications that there is a unique number
  $e \approx 2.71828182845$\..., such that the gradient of the tangent
  to $y = e^{x}$ at $x = 0$ is 1, and call this number $e$ Euler's
  number

- Examine the gradient function of $y = a^{x}$ with graphing
  applications and identify that the gradient function is
  $\frac{d}{dx}\left( a^{x} \right) = ka^{x}$, for some constant $k$
  depending only on $a$

- Conclude that Euler's number, $e$, is a unique number such that
  $\frac{d}{dx}\left( e^{x} \right) = e^{x}$, that is, $e^{x}$ is its
  own derivative

# Properties of the Exponential Curve

+--------------------------------------------------------------------------------------------------------------------------------------------------------------------+
| - **Exponential Curves**                                                                                                                                           |
+========================================================================================================================+===========================================+
| - An **exponential relationship** $\mathbf{y =}\mathbf{a}^{\mathbf{x}}$ produces **exponential curve** graphs.                                                     |
|                                                                                                                                                                    |
| - Exponential curves have these features:                                                                                                                          |
|                                                                                                                                                                    |
|   - An **asymptote**: a line the graph approaches but never reaches.                                                                                               |
|                                                                                                                                                                    |
|   - The **nature** of the graph (**growth** or **decay**) depends on the coefficient of $x$.                                                                       |
|                                                                                                                                                                    |
|   - Consecutive $y$ values change by a **common ratio**, which is the base number $a$.                                                                             |
