# Syllabus Content

**MAV-11-03** analyses and solves algebraic and graphical problems
involving transformations of functions and relations

- Establish using graphing applications that replacing $x$ by $- x$ in
  the equation of a graph corresponds to the reflection of the graph in
  the $y$-axis, and that replacing $y$ by $- y$  corresponds to a
  reflection in the $x$-axis

- Establish using graphing applications that replacing $x$ by $x - a$ in
  the equation of a graph corresponds to a horizontal translation of a
  graph by $a$, that replacing $y$ by $y - b$ corresponds to a vertical
  translation by $b$, and recognise that the sign of $a$ and $b$
  determines the direction of the translation

- Describe a horizontal dilation as a stretch from the $y$-axis and a
  vertical dilation as a stretch from the $x$-axis

- Establish using graphing applications that replacing $x$ by
  $\frac{x}{k}$ in the equation of a graph corresponds to a horizontal
  dilation of a graph by a factor of $k$, that replacing $y$ by
  $\frac{y}{l}$ corresponds to a vertical dilation by a factor of $l$,
  and determine the effects on the graph when the magnitude of the
  dilation factor lies between 0 and 1

- Establish, using graphing applications, replacing $x$ by $\frac{x}{k}$
  and replacing $y$ by $\frac{y}{k}$ in the equation of a graph
  corresponds to a dilation by a factor of $k$, that is, an enlargement
  of the graph when $k > 1$ and a reduction of the graph when
  $0 < k < 1$

- Apply reflections, translations and dilations to functions within the
  scope of the Mathematics Advanced course, excluding trigonometric
  functions, determine the function rule and the graph of the
  transformed function, the domain and range of the transformed graph,
  any intercepts, asymptotes and discontinuities where appropriate, and
  describe which transformations have been applied

- Use the principles of translations to identify the centre, radius,
  domain and range of graphs of circles in the form
  ${(x - a)}^{2} + {(y - b)}^{2} = r^{2}$, where $a$, $b$ and $r$ are
  constants

- Find the centre and radius of circles of the form
  $x^{2} + y^{2} + ax + by + c = 0$, where $a$, $b$ and $c$ are
  constants, by completing the squares

- Graph circles given their equations and find the equation of a circle
  from its graph

- Recognise that the order in which individual transformations are
  applied to functions is important

# Combining Transformations

+---------------------------------------------------------------------------------------------------------------------------------------------------------------------+
| - **Investigation** Order of transformations                                                                                                                        |
+==================================================================================+==================================================================================+
| We have learned three types of graph **transformations**:                                                                                                           |
|                                                                                                                                                                     |
| 1\. **Translations**                                                                                                                                                |
|                                                                                                                                                                     |
| 2\. **Reflections** (dilation of factor $- 1$)                                                                                                                      |
|                                                                                                                                                                     |
| 3\. **Dilations**                                                                                                                                                   |
|                                                                                                                                                                     |
| We will investigate which transformations are **commutative**, i.e. the order doesn't matter.                                                                       |
|                                                                                                                                                                     |
| Consider an arbitrary function $y = f(x)$.                                                                                                                          |
+---------------------------------------------------------------------------------------------------------------------------------------------------------------------+
| **Horizontal and Vertical Transformations**                                                                                                                         |
+----------------------------------------------------------------------------------+----------------------------------------------------------------------------------+
| Dilate horizontally by factor $2,$\                                              | ![](media/Graph Transformations/media/image191.png){width="2.0054604111986003in" |
| then translate downwards by $3$.                                                 | height="1.9685028433945757in"}Translate downwards by $3$,\                       |
|                                                                                  | then dilate horizontally by factor $2.$                                          |
| ![](media/Graph Transformations/media/image190.png){width="2.0054615048118984in" |                                                                                  |
| height="1.9685028433945757in"}$y = f(x)$                                         | $$y = f(x)$$                                                                     |
|                                                                                  |                                                                                  |
| $$y = f\left( \frac{x}{2} \right)$$                                              | $$y + 3 = f(x)$$                                                                 |
|                                                                                  |                                                                                  |
| $$y + 3 = f\left( \frac{x}{2} \right)$$                                          | $$y + 3 = f\left( \frac{x}{2} \right)$$                                          |
+----------------------------------------------------------------------------------+----------------------------------------------------------------------------------+
| The $x$ and $y$ coordinates are independent of each other.                                                                                                          |
|                                                                                                                                                                     |
| We can apply vertical transformations first, or horizontal first.\                                                                                                  |
| The result is the same. Vertical and horizontal transformations are commutative.                                                                                    |
+---------------------------------------------------------------------------------------------------------------------------------------------------------------------+
| **Two Dilations**                                                                                                                                                   |
+----------------------------------------------------------------------------------+----------------------------------------------------------------------------------+
| ![](media/Graph Transformations/media/image192.png){width="2.004861111111111in"  | ![](media/Graph Transformations/media/image193.png){width="2.004861111111111in"  |
| height="1.9680555555555554in"}Dilate horizontally by factor $2,$\                | height="1.9680555555555554in"}Dilate horizontally by factor $\frac{1}{3}$,$$then |
| then dilate horizontally by factor $\frac{1}{3}$.                                | dilate horizontally by factor $2$.                                               |
