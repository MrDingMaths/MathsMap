# Linear Modelling

- Linear relationships model many real-life scenarios.

  - The $y$-axis is the dependent variable. It changes based on another
    variable.

  - The $x$-axis is the independent variable. It causes the change.

  - The **gradient is the rate;** it represents the **change in**
    $\mathbf{y}$ **per unit increase in** $\mathbf{x}$**.**

  - The $\mathbf{y}$**-intercept** is the **initial** or **fixed value
    at** $\mathbf{x = 0}$**.**

## Constructing Linear Models

1.  Read the scenario in its entirety. Highlight the variables and the
    numbers.

2.  Identify the dependent variable $y$. It changes based on another
    variable.

3.  Identify the independent variable $x$. It causes the change.

4.  Determine the gradient $m.$ This is the rate given by the question.
    "$m$ **per** $x$"

5.  Determine the $y$-intercept $c$. This is the initial or fixed value
    given by the question.

# Applications of Simultaneous Equations

- When a scenario has two variables that aren't directly related to each
  other, we need to set up two equations and solve them simultaneously.

1.  Define the two variables.

2.  Set up two equations using information from the scenario.

3.  Solve the two equations simultaneously.

4.  Answer the question in words.

# Break-even Analysis

- **Break-even analysis** is an application of simultaneous equations
  where one equation is **cost,** and the other is **revenue**.

<!-- -->

- ![](media/Working with Functions 6_Constructing and using functions/media/image130.png)The
  **break-even point** occurs when costs match income, meaning:

  - You make no profit or loss

  - Below this point, you operate at a loss

  - Above this point, you generate profit
