**Differentiation with exponential functions**

- Apply the rules of differentiation to find the derivative of
  $f(x) = ke^{ax}$, where $k$ and $a$ are constants

- Use the chain rule to prove
  $\frac{d}{dx}\left( e^{ax + b} \right) = ae^{ax + b}$, where $a$ and
  $b$ are constants and $a \neq 0$

- Prove and use the formula
  $\frac{d}{dx}\left( a^{x} \right) = (\ln a)a^{x}$, where $a$ is a
  constant and $a > 0$

- Use the chain rule to differentiate functions of the form $e^{f(x)}$

8.  Define the **hyperbolic** cosine and sine functions:

$$\cosh x = \frac{e^{x} + e^{- x}}{2}\ \ and\ \sinh x = \frac{e^{x} - e^{- x}}{2}$$

a.  Show that $\frac{d}{dx}\cosh x = \sinh x\$and
    $\frac{d}{dx}\sinh x = \cosh x$.

$$\frac{d}{dx}\cosh\ x\  = \frac{e^{x} - \ e^{- x}}{2} = \ \sinh\ x$$

$$\frac{d}{dx}\sinh\ x\  = \frac{e^{x} - - e^{- x}}{2} = \ \cosh\ x$$

b.  Show that $\cosh^{2}x - \sinh^{2}x = 1$.

$$\cos h^{2}x\  - \ \sin h^{2}x\  = \ \left( \frac{e^{x} + \ e^{- x}}{2} \right)^{2} - \ \left( \frac{e^{x} - \ e^{- x}}{2} \right)^{2}$$

$$= \frac{e^{2x} + \ 2\  + \ e^{- 2x}}{4} - \frac{e^{2x} - \ 2\  + \ e^{- 2x}}{4}$$

$$= \frac{e^{2x} + \ 2\  + \ e^{- 2x} - \ e^{2x} + \ 2\  - \ e^{- 2x}}{4}$$

$$= \frac{4}{4} = \ 1$$

# Tangents and Normals of Exponential Functions

**Foundation**

1.  a.  Use calculus to find the gradient of the tangent to $y = e^{x}$
        at Q(0, 1)

$$\frac{dy}{dx} = \ e^{x}$$

$$at\ Q(0,1):\frac{dy}{dx}\  = \ e^{0} = \ 1$$

b.  Hence find the equation of the tangent at $Q$.

$$y\  - \ 1\  = \ 1(x\  - \ 0)$$

$$y\  = \ x\  + \ 1$$

c.  Find the $x$-intercept of the tangent.

At $y = 0$,

$$x = - 1$$

2.  a.  Use calculus to find the gradient of the tangent to $y = e^{x}$
        at P(1, e)

$$\frac{dy}{dx} = \ e^{x}$$

$$at\ P(1,e):\frac{dy}{dx}\  = \ e^{1} = \ e$$

b.  Hence find the equation of the tangent at $P$.

$$y\  - \ e\  = \ e(x\  - \ 1)$$

$$y\  = \ ex\  - \ e\  + \ e$$

$$y\  = \ ex$$

c.  Find the $x$-intercept of the tangent.

At $y = 0$,

$$x = 0$$

3.  a.  Use calculus to find the gradient of the tangent to
        $y\  = \ e^{x}$ at $R\left( - 1,\frac{1}{e} \right)$

$$\frac{dy}{dx} = \ e^{x}$$

$$at\ R\left( - 1,\frac{1}{e} \right):\frac{dy}{dx}\  = \ e^{- 1} = \frac{1}{e}$$

$$y\  - \frac{1}{e} = \ \left( \frac{1}{e} \right)\left( x\  - \ ( - 1) \right)$$

$$y\  = \ \left( \frac{1}{e} \right)x\  + \frac{1}{e} + \frac{1}{e}$$

$$y\  = \ \left( \frac{1}{e} \right)x\  + \frac{2}{e}$$

$$at\ y = 0,\ $$

$$\frac{1}{e}x = - \frac{2}{e}$$

$$x = - 2$$
