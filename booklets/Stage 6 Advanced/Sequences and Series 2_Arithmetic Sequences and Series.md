# Syllabus Content

**MAV-12-03** uses arithmetic and geometric sequences and series to
model and solve problems

**Arithmetic sequences and series**

- Define a sequence $a_{n}$ to be an arithmetic sequence, or arithmetic
  progression (AP), if every difference $a_{n} - a_{n - 1}$ of
  successive terms is the same where $n \geq 2$, that is 
  $a_{n} - a_{n - 1} = d$ for some constant $d$ called the common
  difference

- Develop the formula $a_{n} = a + (n - 1)d$ for the $n$th term of an
  AP, where $a$ is the first term and $n \geq 1$, and use it to solve
  problems

- Recognise that $a_{n}$ is a linear function of $n$ in an AP

- Develop the formula $S_{n} = \frac{n}{2}\left( a + a_{n} \right)$ for
  the $n$th partial sum of an AP, and use the formula to solve problems

- Develop the formula
  $S_{n} = \frac{n}{2}\left\lbrack 2a + (n - 1)d \right\rbrack$ for the
  $n$th partial sum of an AP, and use the formula to solve problems

- Apply the formulas for arithmetic sequences and their partial sums to
  model and solve growth and decay problems involving a quantity that is
  a linear function of time

# Arithmetic Sequences

+-------------------------------------------------------------------+
| - **Arithmetic Sequences**                                        |
+===================================================================+
| An **arithmetic sequence**, also called an **arithmetic           |
| progression (AP)** or **linear sequence**, **\**                  |
| is a sequence where there is a **common difference** between each |
| pair of consecutive terms.                                        |
|                                                                   |
| For example:                                                      |
|                                                                   |
| $$3,\ 7,\ 11,\ 15,\ 19,\ 23,\ \ldots$$                            |
|                                                                   |
| The common difference ($d$) is the same value that you add to     |
| each term to get the next term.                                   |
|                                                                   |
| It can be:                                                        |
|                                                                   |
| - Positive (sequence increases)                                   |
|                                                                   |
| - Negative (sequence decreases)                                   |
|                                                                   |
| - Zero (sequence stays constant)                                  |
|                                                                   |
| We check whether a sequence is arithmetic by finding the          |
| differences between consecutive terms and seeing if they are      |
| constant.                                                         |
|                                                                   |
| $$a_{n} - a_{n - 1} = d$$                                         |
+-------------------------------------------------------------------+

+-----------------------------------------------------------------------------------------------------------------+
| - **Identify** whether a sequence is arithmetic.                                                                |
+:===============:+:===============:+:==================:+:==================:+:===============:+:===============:+
| ✖               | ✔                                    | ✔                                    | ✖               |
|                 |                                      |                                      |                 |
| 1, 4, 9, 16,    | 1, 4, 7, 10, 13...                   | 55, 48, 41, 34, 27...                | 56, 48, 41, 34, |

+===================================================================+
| Let the first term of a sequence be $a$.                          |
|                                                                   |
| Let the common difference be $d.$                                 |
|                                                                   |
| An arithmetic sequence is                                         |
| $a,\ \ a + d,\ \ a + 2d,\ \ a + 3d\ldots$                         |

| - $\mathbf{n}$**^th^ term Formula for an Arithmetic Sequence**    |
+===================================================================+

|                                                                   |

+-------------------------------------------------------------------+

+---------------------------------------------------------------------+

|   sequence.                                                         |

|                                                                     |

# Partial Sum of an Arithmetic Series

+-------------------------------------------------------------------------------------------------------------------+
| - **Investigation** Partial sum of a Geometric Series.                                                            |
+=================================================+=================================================================+
| Reminder that the partial sum of a sequence is:                                                                   |
|                                                                                                                   |
| $$S_{n} = a_{1} + a_{2} + a_{3} + \ldots a_{n}$$                                                                  |
|                                                                                                                   |
| Consider the partial sum of the arithmetic sequence below:                                                        |
|                                                                                                                   |
| $$S_{6} = 5 + 15 + 25 + 35 + 45 + 55$$                                                                            |
|                                                                                                                   |
| A quick way of adding up everything is by **reversing the sum**, then pairing up the terms.                       |
|                                                                                                                   |
| $$S_{6} = 55 + 45 + 35 + 25 + 15 + 5$$                                                                            |
|                                                                                                                   |
| $2S_{6} =$ ..........................................                                                             |
|                                                                                                                   |
| $\therefore S_{6} =$ ..........................................                                                   |
+-------------------------------------------------------------------------------------------------------------------+
| Use the above method to evaluate:*\*                                                                              |
| $$S_{8} = 9 + 14 + 19 + 24 + 29 + 34 + 39 + 44$$                                                                  |
+-------------------------------------------------------------------------------------------------------------------+
| We will generalise to form an equation; we will first define let the $n$^th^ term be $a_{n}$.                     |
|                                                                                                                   |
| $$S_{n} = a + (a + d) + (a + 2d) + (a + 3d) + \ldots a_{n}$$                                                      |
|                                                                                                                   |
| We can write an alternate form of the formula by substituting $a_{n} = a + (n - 1)d$                              |
+-------------------------------------------------------------------------------------------------------------------+
| - **The Partial Sum of an Arithmetic Series**                                                                     |
+-------------------------------------------------+-----------------------------------------------------------------+
| $$S_{n} = \frac{n}{2}\left( a + a_{n} \right)$$ | $$S_{n} = \frac{n}{2}\left\lbrack 2a + (n - 1)d \right\rbrack$$ |
+-------------------------------------------------+-----------------------------------------------------------------+
| - Usually $n$ is not given, you need to find it using $a_{n} = a + (n - 1)d$                                      |
+-------------------------------------------------------------------------------------------------------------------+

+-------------------------------------------------------------------+
| - **Example** Finding the partial sum of an arithmetic series.    |
+===================================================================+
| Consider the arithmetic series: $100 + 94 + 88 + 82\ldots$        |
|                                                                   |
| Find $S_{10}$                                                     |
|                                                                   |
| $$a = 100,\ d = - 6$$                                             |
|                                                                   |
| $$S_{10} = \frac{10}{2}\left( 2(100) + (10 - 1)( - 6) \right)$$   |
|                                                                   |
| $= 730$                                                           |
+-------------------------------------------------------------------+

    logarithms.

    a.  $\log_{a}2\  + \ \log_{a}4\  + \ \log_{a}8\  + \ ...\  + \ \log_{a}1024$

$$\log_{a}1024\  = \ \log_{a}2^{10} = \ 10\ \log_{a}2$$

$$n\  = \ 10$$

$$= \ 5(11\ log\_ a\ 2)$$

$$= \ 55\ \log_{a}2$$
