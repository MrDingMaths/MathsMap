- Determine the formulas
  $\frac{d}{dx}\left( \sin x \right) = \cos x\$and

5.  Find the equation of the tangent to $y = \sin(\pi - x)$ at the point
    $\left( \frac{\pi}{6},\frac{1}{2} \right)$ in exact general form.

$$\frac{dy}{dx} = \  - \cos(\pi\  - \ x)$$

$$At\ x\  = \frac{\pi}{6}:\frac{dy}{dx} = \  - \cos\left( \pi\  - \frac{\pi}{6} \right) = \  - \cos\left( \frac{5\pi}{6} \right) = \frac{\sqrt{3}}{2}$$

$$Tangent:\ y\  - \frac{1}{2} = \ \left( \frac{\sqrt{3}}{2} \right)\left( x\  - \frac{\pi}{6} \right)$$

This simplifies to

$$6\sqrt{3x} - \ 12y\  + \ 6\  - \ \pi\sqrt{3} = \ 0$$

6.  Find the equation of the normal to $y\  = \ 3sin(2x)\$at the point
    where $x\  = \frac{\pi}{8}$ in exact general form.

$$\frac{dy}{dx} = \ 6cos(2x)$$

$$At\ x\  = \frac{\pi}{8}:\frac{dy}{dx} = \ 6cos\left( \frac{\pi}{4} \right) = \ 6\left( \frac{\sqrt{2}}{2} \right) = \ 3\sqrt{2}$$

$$Normal\ gradient\  = \  - \frac{1}{3\sqrt{2}} = \  - \frac{\sqrt{2}}{6}$$

$$At\ x\  = \frac{\pi}{8}:\ y\  = \ 3sin\left( \frac{\pi}{4} \right) = \frac{3\sqrt{2}}{2}$$

$$Normal:\ y\  - \frac{3\sqrt{2}}{2} = \ \left( - \frac{\sqrt{2}}{6} \right)\left( x\  - \frac{\pi}{8} \right)$$

This simplifies to

$$8x + 24\sqrt{2y} - 72 - \pi = 0$$

13. a.  By writing $\sec\ x$ as $(\cos\ x)⁻¹,$ show that
        $\frac{d}{dx}\left( \sec\ x \right) = \ \sec\ x\ \tan\ x$.

$$\frac{d}{dx}(sec\ x) = \  - (cos\ x)^{- 2}( - sin\ x)$$

$$= \ \frac{\sin x}{cos^{2}x}$$

$$= \ \left( \frac{1}{\cos x} \right)\left( \frac{\sin x}{\cos x} \right)$$

$$= \ sec\ x\ tan\ x$$

    $\frac{d}{dx}(cosec\ x) = \  - cosec\ x\ \cot\ x$.

$$\frac{d}{dx}(cosec\ x) = \  - (sin\ x)^{- 2}(cos\ x)$$

$$= \  - \frac{\cos x}{sin^{2}\ x}$$

$$= \  - \left( \frac{1}{\sin x} \right)\left( \frac{\sin x}{\cos x} \right)$$

$$= \  - cosec\ x\ cot\ x\ $$

    $\frac{d}{dx}\left( \cot\ x \right) = \  - cosec^{2}x$.

$$= \  - \frac{1}{sin^{2}x}$$

$$= \  - cosec^{2}x$$
