# Syllabus Content

**MAV-12-04** selects and applies differentiation methods to solve
problems

**Differentiation with logarithmic functions**

- Use chain rule on the identity $e^{\ln x} = x$ to prove the formula
  $\frac{d}{dx}\left( \ln x \right) = \frac{1}{x}$ for the derivative of
  the natural logarithm function where $x > 0$

- Prove and use the formula
  $\frac{d}{dx}\left( \log_{a}x \right) = \frac{1}{x\ln a}$, where $a$
  is a constant and $a > 0$

- Use the chain rule to differentiate functions of the form $\ln{f(x)}$

# Derivative of Natural Logarithmic Functions

+-------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------+
| - **Investigation** Derivative of natural logarithmic functions.                                                                                                                          |
+===========================================================================================================================================================================================+
| In this investigation we will derive the derivative of $y = \ln x$                                                                                                                        |
|                                                                                                                                                                                           |
| We will start with $y = e^{\ln x},$ differentiate by letting $u = \ln x:$                                                                                                                 |
|                                                                                                                                                                                           |
| $$\ \ \ y = e^{\ln x}$$                                                                                                                                                                   |
|                                                                                                                                                                                           |
| $$\frac{dy}{dx} = \frac{dy}{du} \times \frac{du}{dx}$$                                                                                                                                    |
|                                                                                                                                                                                           |
| Let $u = \ln x$                                                                                                                                                                           |
|                                                                                                                                                                                           |
| $$\ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \frac{du}{dx} = u'(x)$$ |
|                                                                                                                                                                                           |
| (where $u'(x)$ is the derivative of $\ln x$, which we want to find)                                                                                                                       |
|                                                                                                                                                                                           |
| Let $y = e^{u}$                                                                                                                                                                           |
|                                                                                                                                                                                           |
| $$\ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \frac{dy}{du} =$$       |
|                                                                                                                                                                                           |
| Now, we apply the identity $e^{\ln x} = x$.                                                                                                                                               |
|                                                                                                                                                                                           |
| +------------------------------------+--------------------------------+                                                                                                                   |
| | From above, we can rewrite:        | And at the same time, using    |                                                                                                                   |
| |                                    | $y = e^{\ln x} = x$,           |                                                                                                                   |
| | $$\frac{dy}{dx} = u'(x)e^{\ln x}$$ |                                |                                                                                                                   |
| |                                    | $$\frac{dy}{dx} =$$            |                                                                                                                   |
| | $=$                                |                                |                                                                                                                   |
| +====================================+================================+                                                                                                                   |
|                                                                                                                                                                                           |
| Equating the two expressions we have for $\frac{dy}{dx},$ we can find an expression for $u'(x),$ which is the derivative of $\ln x$.                                                      |
+-------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------+

+------------------------------------------------------------------------------------------------------+
| - **The Log Laws**                                                                                   |
+===================================+==================================================================+
| **Inverse Relationship**          | ${\log_{a}a}^{x} = a^{\log_{a}x} = x$                            |
+-----------------------------------+------------------------------------------------------------------+
| **Log Law of Multiplication**     | $$\log_{a}(xy) = \log_{a}x + \log_{a}y$$                         |
+-----------------------------------+------------------------------------------------------------------+
| **Log Law of Division**           | $$\log_{a}{\left( \frac{x}{y} \right) = \log_{a}x - \log_{a}y}$$ |
+-----------------------------------+------------------------------------------------------------------+
| **Log Law of Indices**            | $$\log_{a}{(x^{n})} = n\log_{a}x$$                               |
+-----------------------------------+------------------------------------------------------------------+
