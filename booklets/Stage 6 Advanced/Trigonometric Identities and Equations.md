# Overview

## Syllabus Content

**MAV-11-05** uses periodic functions to solve trigonometric equations
and prove trigonometric identities

- Define the trigonometric functions $\sec\theta$, $cosec\theta$ and
  $\cot\theta$ for acute angles $\theta$ using ratios of sides in a
  right-angled triangle

- Find the exact secant, cosecant and cotangent ratios for angles of
  $30{^\circ}$, $45{^\circ}$ and $60{^\circ}$

- Justify the values of $\sec\theta$, $cosec\theta$ and $\cot\theta$ at
  $0{^\circ}$ and $90{^\circ}$ by examining the ratios of sides in a
  right-angled triangle as $\theta$ tends to $0{^\circ}$ and
  $90{^\circ}$

- Develop the definitions $\sec{\theta = \frac{r}{x}}$,
  $cosec\theta = \frac{r}{y}$ and $\cot\theta = \frac{x}{y}$ using a
  circle of radius $r$ centred at the origin

- Establish the reciprocal ratios $\sec A = \ \frac{1}{cos\ A}$,
  $cosec\ A\  = \ \frac{1}{sin\ A}$, and $\cot A = \ \frac{1}{tan\ A}$,
  identifying the angles to be excluded in each identity

- Determine the exact value of the secant, cosecant and cotangent ratios
  for angles that are integer multiples of $\frac{\pi}{6}$ and
  $\frac{\pi}{4}$, if they exist

- Establish the identities $\tan\theta = \frac{\sin\theta}{\cos\theta}$
  and $\cot\theta = \frac{\cos\theta}{\sin\theta}$, identifying the
  angles to be excluded in each identity

- Prove the complementary angle identities
  $\sin{(90{^\circ} -}\theta) = \cos\theta$,
  ${\cos{(90{^\circ} - \theta) = \sin}}\theta$,
  ${\tan{(90{^\circ} - \theta) = \cot}}\theta$,
  ${\cot{(90{^\circ} - \theta) = \tan}}\theta$,
  ${\sec{(90{^\circ} - \theta) = cosec}}\theta$ and
  ${{cosec}{(90{^\circ} - \theta) = \sec}}\theta$ identifying the angles
  to be excluded in each identity

- Evaluate trigonometric expressions using angles of any magnitude and
  complementary angle identities

- Solve equations involving trigonometric ratios of angles, specified in
  degrees or radians, on a restricted domain

- Prove the Pythagorean identity $\cos^{2}x + \sin^{2}x = 1$, and the
  identities ${1 + \tan}^{2}x = \sec^{2}x$ and
  $1 + \cot^{2}x = {cosec}^{2}x$

- Apply trigonometric identities to solve problems, simplify expressions
  and prove further trigonometric identities using substitution and/or
  reduction to $\sin x$ and $\cos x$

- Solve problems involving trigonometric equations, including those that
  reduce to quadratic equations, on a restricted domain

# The Reciprocal Trigonometric Functions

  ---------------------------------------------------------------------------------------------------------------------
  **Reciprocal Trig Functions**
  ---------------------------------------------------------------------------------------------------------------------
  $${co\mathbf{s}ec}\theta = \frac{1\ }{\mathbf{s}in\ \theta} = \frac{Hyp}{Opp}$$

  $${\ \ \ \ se\mathbf{c}}\theta = \frac{1}{\mathbf{c}os\ \theta} = \frac{Hyp}{Adj}$$

  $${\ \ \ \ co\mathbf{t}}\theta = \frac{1}{\mathbf{t}an\ \theta} = \frac{Adj}{Opp} = \frac{\cos\theta}{\sin\theta}$$
  ---------------------------------------------------------------------------------------------------------------------

$$= (2)^{2} - \left( \sqrt{3} \right)^{2}\ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ $$

$$= 1\ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ $$

6.  Without using a calculator, prove that
    $\frac{1}{\sec{30{^\circ}} - \tan{30{^\circ}}}$ $= \sqrt{3}$

$$\frac{1}{\left( \frac{1}{\cos{30{^\circ}}} \right) - \tan{30{^\circ}}}$$

$$= \frac{1}{\frac{2}{\sqrt{3}} - \frac{1}{\sqrt{3}}}$$

$$= \frac{1}{\left( \frac{1}{\sqrt{3}} \right)} = \sqrt{3}$$

# Deriving a Trigonometric Ratio

- When you are given a trigonometric ratio, you can find another without
  finding the angle.

+-------------------------------------------+

13. If $a > 0$ and $\sec\theta = a +$ $\frac{1}{4a}$, prove that
    $\sec\theta + \tan\theta = 2a\ or$ $\frac{1}{2a}$

$$\sec\theta = \frac{4a^{2} + 1}{4a}$$

$$\cos\theta = \frac{4a}{4a^{2} + 1}$$

$since\ a > 0,\cos\theta > 0;\ \theta$ is in Q1 or 4

$$adj = 4a,\ hyp = 4a^{2} + 1$$

$$opp = \sqrt{\left( 4a^{2} + 1 \right)^{2} - (4a)^{2}} = \sqrt{16a^{4} - 8a^{2} + 1} = \sqrt{\left( 4a^{2} - 1 \right)^{2}} = 4a^{2} - 1$$

**Tips for proving identities**

- You **CANNOT** apply operations to both sides when proving statements.

- **Convert to sine and cosine**: Simplify mixed functions by rewriting
  everything in terms of sinθ and cosθ.

- **Use Pythagorean identities**: Replace squared terms using the
  identities

- **Start with the more complex side**: It gives more options for
  simplification.

- **Expand brackets**: Reveals terms that may cancel or combine.

- **Factor expressions**: Useful when one side is a product and the
  other a sum or difference.

- **Split fractions**: Break up a single fraction into separate terms if
  the other side has multiple terms.

- **Combine fractions**: Use a common denominator to merge terms.

- **Multiply by the conjugate**: For expressions with binomials in the
  denominator, this can reveal a difference of two squares.

- **Simplify both sides**: Complex examples will require you to simplify
  both LHS and RHS separately until they match.

**The key is recognising patterns through practice. You will likely
struggle at first.**

15. **HSC Advanced Sample Question Band 5**

> Solve
> $\left( \tan\theta - 1 \right)\left( \sin^{2}\theta - 3\cos^{2}\theta \right) = 0$
> where $0 \leq \theta \leq \pi$.

$$\left( \tan\theta - 1 \right)\left( \sin^{2}\theta - 3\cos^{2}\theta \right) = 0$$

$$\left( \tan\theta - 1 \right)\left( \sin\theta - \sqrt{3}\cos\theta \right)\left( \sin\theta + \sqrt{3}\cos\theta \right) = 0$$

Consider $\sin\theta - \sqrt{3}\cos\theta = 0$, divide all terms by
$\cos\theta$:

$$\tan\theta - \sqrt{3} = 0$$

$$\tan\theta = \sqrt{3}$$

Similarly,
$\sin\theta + \sqrt{3}\cos\theta \rightarrow \tan\theta = - \sqrt{3}$

$$\therefore\tan\theta = 1\ or\tan\theta = \pm \sqrt{3}$$

$$\therefore\theta = \frac{\pi}{4},\frac{\pi}{3},\frac{2\pi}{3}$$

16. a.  Prove that $\frac{1}{\sin x\cos x}$ $- \cot x = \tan x$

LHS

$$= \frac{1}{\sin x\cos x} - \frac{\cos x}{\sin x}$$

$$= \frac{1 - \cos^{2}x}{\sin x\cos x}$$

$$= \frac{\sin^{2}x}{\sin x\cos x}$$

$$= \frac{\sin x}{\cos x}$$

$$= \tan x$$

= RHS

b.  Hence solve in the domain $x \in \lbrack 0,2\pi\rbrack:\$

$$\frac{\cos^{2}x - 1}{\sin x\cos x} = \sqrt{3}$$

$$\frac{- \left( - \cos^{2}x + 1 \right)}{\sin x\cos x} = \sqrt{3}$$

$$\frac{- \sin^{2}x}{\sin x\cos x} = \sqrt{3}$$

$$- \tan x = \sqrt{3}$$

$$\tan x = - \sqrt{3}$$

$$x = \frac{2\pi}{3},\frac{5\pi}{3}$$
