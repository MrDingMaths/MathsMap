# Absolute Value

- The symbol $|x|$ represents the **absolute value** of $x$.

- A number's **absolute value**, or **magnitude**, is its distance from
  zero on a number line.

## Square Root Definition of Absolute Value

- To find $|x|$, we can square the number, then take the positive square
  root $\sqrt{x^{2}}$.

+------------------------+
| **Absolute Value as    |
| the Positive Square    |
| Root**                 |
+========================+
| For all real numbers:  |
|                        |
| $$|x|^{2} = x^{2}$$    |
|                        |
| $$|x| = \sqrt{x^{2}}$$ |
+------------------------+

# Absolute Value as a Piecewise Function

+------------------------------------------------+
| **Absolute Value as a\                         |
| Piecewise-Defined Function**                   |
+================================================+
| $$|x| = \left\{ \begin{matrix}                 |
| x,\ \ \ \ \ \text{for}\ x \geq 0, \\           |
|  - x,\ \text{for}\ x < 0                       |
| \end{matrix} \right.\ $$                       |
|                                                |
| More generally:                                |
|                                                |
| $$\left| f(x) \right| = \left\{ \begin{matrix} |
| f(x),\ \ \ \ \ \text{for}\ f(x) \geq 0, \\     |
|  - f(x),\ \text{for}\ f(x) < 0                 |
| \end{matrix} \right.\ $$                       |
+------------------------------------------------+

To write an absolute value expression as a piecewise function:

1.  Write $\left| f(x) \right| = \left\{ \begin{matrix}
    f(x),\ \ \ \ \ \text{for}\ f(x) \geq 0, \\
     - f(x),\ \text{for}\ f(x) < 0
    \end{matrix} \right.\$

2.  Simplify as much as possible.

# Solving Absolute Value Equations

- An absolute value equation has the form $|ax + b| = k$

- Using the fact that

$|ax + b| = k$ and $\left| - (ax + b) \right| = k$

we remove the absolute value signs by considering two cases:

+-----------------------------------------------+
| **Solving Absolute Value Equations**          |
+=======================+=======================+
| **Case 1:**           | **Case 2:**           |
| $\mathbf{ax + b}$     | $\mathbf{ax + b}$     |
| **is positive**       | **is negative**       |
|                       |                       |
| $$ax + b = k$$        | $$- (ax + b) = k$$    |
+-----------------------+-----------------------+
