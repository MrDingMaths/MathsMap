# Overview

## Syllabus Content

**MAV-11-02** uses functions and relations to model, analyse and solve
problems

**Properties of functions, relations and graphs**

- Extend the definitions of domain and range to relations

- Recognise domains and ranges of functions and relations given in
  interval notation, as inequalities and as worded descriptions

- Determine and describe the domain and range of functions and
  relations, using interval notation, inequalities or worded
  descriptions

- Define a function to be even if its graph is unchanged under
  reflection in the $y$-axis, and odd if its graph is unchanged under
  rotation of $180{^\circ}$ about the origin

- Develop and use the tests that a function $f(x)$ is odd if
  $f( - x) = - f(x)$ and a function $f(x)$ is even if $f( - x) = f(x)$

- Solve problems involving even and odd functions

- Use the composite function $f\left( g(x) \right)$, where the output of
  $g(x)$ becomes the input of $f(x)$

- Determine the equations of composite functions

# Testing for Even or Odd Functions Algebraically

## Function Notation and Negatives

- Testing for odd and evenness involves looking for symmetry.

+-------------------------------------------------------------------------------------------------------------------+----------------------------------------------------------------------------------------------------------------------------------+
| **Even Function**                                                                                                 | **Odd Function**                                                                                                                 |
+:=================================================================================================================:+:================================================================================================================================:+
| Line symmetry across $y$ axis.                                                                                    | $180{^\circ}$ rotational symmetry about the origin.                                                                              |
|                                                                                                                   |                                                                                                                                  |
| $$f( - x) = f(x)$$                                                                                                | $$f( - x) = - f(x)$$                                                                                                             |
|                                                                                                                   |                                                                                                                                  |
| ![](media/Working with Functions 9_Properties of functions, relations and graphs/media/image25.png){width="1.6in" | ![](media/Working with Functions 9_Properties of functions, relations and graphs/media/image26.png){width="1.6162073490813649in" |

+-------------------------------------------------------------------------------------------------------------------+----------------------------------------------------------------------------------------------------------------------------------+

- To check whether a function equation is even, odd, or neither:

1.  Substitute in $f( - x)$.

2.  Simplify as much as possible.

3.  If $f( - x) = f(x)$, even.

> If $f( - x) = - f(x)$, odd.
>
> If $f( - x)$ does not match $f(x)$ or $- f(x)$, neither.

# Composite Functions

- A composite function is where the output of one function is inputted
  into another function.

- In the composite function $f\left( g(x) \right)$, the output of $g(x)$
  becomes the input of $f(x)$.

- Another way to write $f\left( g(x) \right)$ is $f \circ g$, read as
  '$f$ of $g$'.

### **Example** Determine the equation of a composite function.

# Domain and Range of Combined Functions

- To find the domain of a **combined function** $f(x) \pm g(x)$,
  $f(x) \times g(x)$ or $f(x) \div g(x)$:

1.  Identify domain of $f(x)$, call this set $F$.

2.  Identify domain of $g(x)$, call this set $G$.

3.  The domain of the combined function is $F \cap G$. ($x$ values in
    both $F$ and $G$).

4.  For division, there is an extra restriction: $g(x) \neq 0$
