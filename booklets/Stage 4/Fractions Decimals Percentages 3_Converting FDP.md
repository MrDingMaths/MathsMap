# Syllabus Content

**MA4-FRC-C-01** represents and operates with fractions, decimals and
percentages to solve problems

**Identify and make use of the relationship between fractions, decimals
and percentages to carry out simple conversions**

- Define rational numbers as numbers that can be written in the form
  $\frac{a}{b}$, where $a$ and $b$ are integers and $b \neq 0$

- Classify fractions and percentages as rational numbers

- Recognise and explain that numbers with terminating or recurring
  decimals are rational

- Represent fractions as decimals (terminating and recurring) and
  percentages

- Represent terminating decimals as fractions and percentages

- Represent improper fractions as mixed numbers and decimals, and vice
  versa

- Represent percentages as fractions and decimals

# Converting Mixed Numbers and Improper Fractions

+-------------------------------------------------------------------+
| - **Mixed to Improper using Visuals**                             |
+===================================================================+
| To convert **mixed numbers** to **improper fractions**:           |
|                                                                   |
| 1.  Split each whole into parts based on the **denominator**.     |
|                                                                   |
| 2.  Count the number of shaded parts.                             |
+-------------------------------------------------------------------+

+----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------+

+-------------------------------------------------------------------+
| - **Comparing Fractions Decimals Percentages**                    |
+===================================================================+
| To compare the value of fractions, decimals, and percentages, it  |
| is most efficient to:                                             |
|                                                                   |
| 1.  Convert everything to decimal form.                           |
|                                                                   |
| 2.  Order the decimals based on place value.                      |
|                                                                   |
| 3.  Rewrite the decimals in their original form.                  |
+-------------------------------------------------------------------+

+-------------------------------------------------------------------+
| - **Example** Order fractions decimals percentages                |
+===================================================================+
|   ---------------------------------------------------------       |
|   $$\frac{1}{5}$$   $$75\%$$   $$\frac{4}{8}$$   $$18\%$$         |
|   ----------------- ---------- ----------------- ----------       |
|                                                                   |
|   ---------------------------------------------------------       |
|                                                                   |
| 1\. Convert to decimal $\downarrow$                               |
|                                                                   |
|   -----------------------------------------                       |
|   $$0.2$$   $$0.75$$   $$0.5$$   $$0.18$$                         |
|   --------- ---------- --------- ----------                       |
|                                                                   |
|   -----------------------------------------                       |
|                                                                   |
| 2\. Ascending order $\downarrow$                                  |
|                                                                   |
|   -----------------------------------------                       |
|   $$0.18$$   $$0.2$$   $$0.5$$   $$0.75$$                         |
|   ---------- --------- --------- ----------                       |
|                                                                   |
|   -----------------------------------------                       |
|                                                                   |
| 3\. Rewrite $\downarrow$                                          |
|                                                                   |
|   ---------------------------------------------------------       |
|   $$18\%$$   $$\frac{1}{5}$$   $$\frac{4}{8}$$   $$75\%$$         |
|   ---------- ----------------- ----------------- ----------       |
|                                                                   |
|   ---------------------------------------------------------       |
