+-------------------------------------------------------------------+
| - **Expanding Brackets**                                          |
+===================================================================+
| We **group** terms using **brackets**.                            |
|                                                                   |
| **Expanding brackets** means simplifying an expression by         |
| **writing** **without** **brackets**.                             |
|                                                                   |
| To expand brackets, we multiply each term inside the brackets by  |
| the term outside.\                                                |
| This is called the **distributive law**.                          |
|                                                                   |
| $$a(b + c) = ab + ac$$                                            |
+-------------------------------------------------------------------+

+--------------------------------------------------------------------------------+
| - **Example** Expand brackets using the distributive law                       |
+===================================+============================================+
| Expand:                                                                        |
+-----------------------------------+--------------------------------------------+
| $$\ \ \ \ \ 3(2x + 5)$$           | To expand a bracket, multiply              |
|                                   | e............ term inside the brackets by  |
| $$= 3(2x) + 3(5)$$                | the term o...............                  |
|                                   |                                            |
| $$= 6x + 15$$                     |                                            |
+-----------------------------------+--------------------------------------------+

|                                                                   |
| 2.  When there is subtraction inside the brackets, such as        |
|     $4(3a - 2)$, treat the two terms inside the bracket as        |
|     ......... and .........                                       |

| - **Expanding Brackets**                                          |
+===================================================================+
| To expand brackets, we multiply each term inside the brackets by  |
| the term outside.                                                 |
|                                                                   |
| $$a(b + c) = ab + ac$$                                            |

+--------------------------------------------------------------------------------+
| - **Example** Expand brackets using the distributive law                       |
+===================================+============================================+
| Expand $x(2y + 3)$                | To expand a bracket, multiply              |
|                                   | e............ term inside the brackets by  |

|                                   |                                            |
| $= 2xy + 3x$                      |                                            |
+-----------------------------------+--------------------------------------------+
| Expand $x(x + 4)$                 | When something is multiplied by itself, we |
|                                   | write $x \times x =$ ......                |
| $= x(x) + x(4)$                   |                                            |
|                                   |                                            |
| $= x^{2} + 4x$                    |                                            |
