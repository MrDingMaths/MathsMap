# Finding Lengths in Scale Drawings

- A **scale drawing** is a **similar figure** of a real-life object.\
  The **original** is the real-life object, and the **image** is the
  drawing.

- Examples include maps, house plans, and scientific diagrams.

- Scales for scale drawings are written in **simplified ratio
  notation:** $\mathbf{1\ :\ n}$

<!-- -->

- A scale of 1:500 means every 1 unit on the drawing is 500 units in
  real life.

  -----------------------------
  **Scales in Scale Drawings**
  -----------------------------
  $$image\ :original$$

  -----------------------------

1.  Set up a ratio table ***image : original***

    a.  The first row should be the given scale.

    b.  The second row should be the given length and the length you
        want to find.

2.  Find the unknown side by multiplying or dividing.

# Solving Problems using the Unitary Method

- If a problem gives a ratio $\mathbf{m:n}$, one part's value, and asks
  for the value of the other,\
  we can solve it using the **unitary method**.

1.  Identify the number of parts the given quantity represents.

2.  Divide to find 1 part's value.

3.  Multiply to find the required number of parts.
