# Best Buys

- The **best buy** offers the best value for money.

1.  Convert both rates to the same units if different.

2.  Simplify to unit rates using **\$** per **quantity**.

3.  The **lowest price per unit** is the best buy.

### **Example** Determine the best buy.

# Rates Problems

- We will solve rates problems using **equivalent rates**.

- Solve problems involving rates using algebraic fractions:

1.  Write the given information as an equation. Let the quantity you
    want to find be $x$.

2.  Solve for the unknown.

> If the unknown is in the denominator, take the reciprocal of both
> sides.

### **Example** Solve rates problems.

+--------------------------------------------------+------------------------------------------------------------+
| A car travels 360 km on 30 L of petrol.\         | Bella can type 240 words every 3 minutes. How long does it |
| How far does it travel on 120 L?                 | take to write 2800 words?                                  |
|                                                  |                                                            |
| 1\.                                              | 1\.                                                        |
|                                                  |                                                            |
| $$\frac{360\ km}{30\ L} = \frac{x\ km}{120\ L}$$ | $$\frac{240\ words}{3\ min} = \frac{2800\ words}{x\ min}$$ |
|                                                  |                                                            |
| 2\. $\times 120\ \ \ \  \times 120$              | 2\. reciprocal both sides                                  |
|                                                  |                                                            |
| $$1440\ km = x$$                                 | $$\frac{3\ min}{240\ words} = \frac{x\ min}{2800\ words}$$ |
|                                                  |                                                            |
|                                                  | $$\times 2800\ \ \ \ \  \times 2800$$                      |
|                                                  |                                                            |
|                                                  | $35\ min = x$                                              |
+==================================================+============================================================+

# Related Rates

- Usually when working with rates, we only use two units.

- Sometimes a question will give a *third* unit (or more).

- To solve these problems:

1.  Identify the three quantities in the question.

2.  Set up a **related rate equation**.

> $$\frac{Quantity\ 1}{Quantity\ 2} = \frac{Quantity\ 1}{Quantity\ 3} \times \frac{Quantity\ 3}{Quantity\ 2}$$

3.  Substitute given information.\
    The scenario will always give two of the rates.

4.  Convert units to be the same, and to the unit wanted by the
    question.

5.  Simplify and solve the equation.

- A common application is drip rates for medication, important for
  nurses.
