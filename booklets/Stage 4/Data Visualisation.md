# Syllabus Content

**MA4-DAT-C-01** classifies and displays data using a variety of
graphical representations

**Classify data as either numerical (discrete or continuous) or
categorical (nominal or ordinal) variables**

- Define a variable in the context of statistics as any characteristic,
  number or quantity that can be measured or counted

- Classify and describe variables as numerical or categorical

- Describe a numerical variable as either discrete or continuous

- Describe a categorical variable as nominal or ordinal

- Distinguish between and compare numerical (discrete or continuous) and
  categorical (nominal or ordinal) variables

# Classifying Data

+-------------------------------------------------------------------+
| - **Variables**                                                   |
+===================================================================+
| Data has **variables** and **values**.                            |
|                                                                   |
| - A **variable** is anything we can describe, measure, or count.  |
|                                                                   |
| - **Data values** are the measurements.                           |
|                                                                   |
|   --------------------------------                                |
|   **Variables**   **Data Values**                                 |
|   --------------- ----------------                                |
|   Age             11, 12, 15, 21,                                 |
|                   86                                              |
|                                                                   |
|   Grade           A, B, C, D, E                                   |
|                                                                   |
|   Colour          Red, Green,                                     |
|                   Orange                                          |
|   --------------------------------                                |
+-------------------------------------------------------------------+

+-------------------------------------------------------------------+
| - **Numerical and Categorical Variables**                         |
+===================================================================+
| **Categorical** data is **qualitative**. It involves              |
| descriptions, categories, and names.                              |
|                                                                   |
| **Numerical** data is **quantitative**. It involves numbers.      |
+-------------------------------------------------------------------+

+-----------------------------------------------------------------------+

+-----------------------------------+-----------------------------------+
| e.  Favourite makeup brand.       | f.  Time taken to complete a      |
|                                   |     test.                         |
+-----------------------------------+-----------------------------------+

+-------------------------------------------------------------------+
| - **Discrete and Continuous Variables**                           |
+===================================================================+
| - **Numerical** data can be **discrete** or **continuous**.       |
|                                                                   |
| <!-- -->                                                          |
|                                                                   |
| - **Discrete** data:                                              |
|                                                                   |
|   - **increase in steps**.\                                       |
|     usually comes from **counting.**                              |
|                                                                   |
| - **Continuous** data:                                            |
|                                                                   |
|   - can have **any value** in a range**.\**                       |
|     usually comes from **measuring**.                             |
+-------------------------------------------------------------------+

+-----------------------------------------------------------------------+
| - **Classify** variables as numerical discrete or numerical           |
|   continuous                                                          |
+===================================+===================================+

+-----------------------------------+-----------------------------------+

|                                   |     school today.                 |
+-----------------------------------+-----------------------------------+
| e.  Number of people at a         | f.  Height of a year 9 student.   |
|     concert.                      |                                   |
+-----------------------------------+-----------------------------------+

+-------------------------------------------------------------------+
| - **Nominal and Ordinal Variables**                               |
+===================================================================+
| - **Categorical** variables can be **nominal** or **ordinal**.    |
|                                                                   |
| <!-- -->                                                          |
|                                                                   |
| - Nominal data has no natural order, e.g. colours, flavours,      |
|   locations etc.                                                  |
|                                                                   |
| - Ordinal data has a natural order, e.g. sizes, level of          |
|   education, etc.                                                 |
