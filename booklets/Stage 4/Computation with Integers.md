# Syllabus Content

**MA4-INT-C-01** compares, orders and calculates with integers to solve
problems

**Compare and order integers**

- Recognise and describe the direction and magnitude of integers

- Identify and represent integers on a number line

- Compare the relative value of integers using the less than (\<) and
  greater than (\>) symbols

- Order integers

**Add and subtract positive and negative integers**

- Add and subtract integers with and without the use of digital tools

- Construct a directed number sentence to model a situation

- Examine different meanings (position or operation) for the + and -
  signs, depending on context

**Multiply and divide positive and negative integers**

- Represent multiples of negative integers as repeated addition

- Multiply and divide positive and negative integers with and without
  the use of digital tools

**Apply the 4 operations to integers**

- Apply the 4 operations to integers

- Solve problems involving grouping symbols with integers

- Apply the order of operations to evaluate expressions involving
  integers, with and without the use of digital tools

| - **Integers**                                                                                                          |
+-------------------------------------------------------------------------------------------------------------------------+
| An **integer** is a whole number.                                                                                       |
|                                                                                                                         |
| Integers include positive whole numbers, negative whole numbers, and zero.                                              |
|                                                                                                                         |
| ![](media/Computation with Integers/media/image7.png){width="2.053152887139108in" height="2.7559055118110236in"}        |
|                                                                                                                         |
| **Positive integers** are greater than zero                                                                             |
|                                                                                                                         |
| Positive integers may be written with a $+$ sign or no sign.\                                                           |
| (e.g. $+ 5$ or $5$)                                                                                                     |
|                                                                                                                         |
| **Negative integers** are less than zero.                                                                               |
|                                                                                                                         |
| Negative integers are always written with a $-$ sign.\                                                                  |
| (e.g. $- 5$)                                                                                                            |
|                                                                                                                         |
| We use negative numbers to represent temperature,\                                                                      |
| owing money (debt), altitude, etc.                                                                                      |

# The Negative Symbol

+-------------------------------------------------------------------+
| - **Meaning of the Minus Sign**                                   |
+===================================================================+
| The minus symbol: $-$ has two different meanings in mathematics:  |
|                                                                   |
| - Tells us that a number has **negative direction**, when it is   |
|   'attached' to a number.\                                        |
|   $- 5$ means 'negative 5'                                        |
|                                                                   |
| <!-- -->                                                          |
|                                                                   |
| - To avoid confusion, we place negative numbers inside brackets   |
|   to indicate that a negative is 'attached' to a number.          |
|                                                                   |
| $8 + ( - 5)$ means 8 add 'negative 5'                             |
|                                                                   |
| - Tells us to **subtract,** when it is in between two numbers.    |
|                                                                   |
| $8 - 5$ means '8 subtract 5'                                      |
+-------------------------------------------------------------------+

+-----------------------------------------------------------------------------+
| - **Interpret** the negative symbol                                         |
+=========================+=========================+=========================+
| $$10 - 4$$              | $$- 7$$                 | $$5 + ( - 2)$$          |
|                         |                         |                         |

+-------------------------+-------------------------+-------------------------+
| $$- 15 + 3$$            | $$6 - ( - 3)$$          | $$12 - 8$$              |
|                         |                         |                         |
| negative 15 add 3       | 6 subtract negative 3   | 12 subtract 8           |

# Multiplying and Dividing Integers

+-----------------------------------------------------------------------+
| - **Investigation** Sign rules for multiplication and division        |
+===================================+===================================+
| We will investigate the sign rules for multiplying integers:          |
+-----------------------------------+-----------------------------------+
| $3 \times 2 =$                    | $( - 3) \times 2 =$               |
|                                   |                                   |
| $3 \times 1 =$                    | $( - 3) \times 1 =$               |
|                                   |                                   |
| $3 \times 0 =$                    | $( - 3) \times 0 =$               |
|                                   |                                   |
| $3 \times ( - 1) =$               | $( - 3) \times ( - 1) =$          |
|                                   |                                   |
| $3 \times ( - 2) =$               | $( - 3) \times ( - 2) =$          |
|                                   |                                   |
| $3 \times ( - 7) =$               | $( - 3) \times ( - 7) =$          |
+-----------------------------------+-----------------------------------+

+-------------------------------------------------------------------+
| - **Sign Rules for Multiplication and Division**                  |
+===================================================================+
| **Same signs** $\mathbf{\rightarrow}$ **positive**                |
|                                                                   |
| When multiplying/dividing two numbers with the same sign (both    |
| positive or both negative), the answer is positive.               |
|                                                                   |
| **Different signs** $\mathbf{\rightarrow}$ **negative**           |
|                                                                   |
| When multiplying/dividing two numbers with different signs (one   |
| positive, one negative), the answer is negative.                  |
+-------------------------------------------------------------------+

+-----------------------------------------------------------------------------+
| - **Identify** Whether the answer will be positive or negative              |
+:=======================:+:=======================:+:=======================:+
| negative $\times$       | positive $\times$       | positive $\div$         |
| negative                | negative                | negative                |
|                         |                         |                         |
| = positive              | = negative              | = negative              |
+-------------------------+-------------------------+-------------------------+
| Decide whether the answer will be positive or negative.                     |
+-------------------------+-------------------------+-------------------------+
| a.  negative $\times$   | b.  negative $\times$   | c.  positive $\times$   |
|     positive            |     positive            |     positive            |
|                         |                         |                         |
| - Positive              | - Positive              | - Positive              |
|                         |                         |                         |
| - Negative              | - Negative              | - Negative              |
+-------------------------+-------------------------+-------------------------+
| d.  positive $\div$     | e.  negative $\div$     | f.  negative $\times$   |
|     negative            |     negative            |     negative            |
|                         |                         |                         |
| - Positive              | - Positive              | - Positive              |
|                         |                         |                         |
| - Negative              | - Negative              | - Negative              |
+-------------------------+-------------------------+-------------------------+
| - **Multiplying and Dividing Integers**                                     |
+-----------------------------------------------------------------------------+
| When multiplying integers, we work out the sign separately from the         |
| magnitude.                                                                  |
|                                                                             |
| 1.  Determine the sign of the answer                                        |
|                                                                             |
| Same signs $\rightarrow$ positive                                           |
|                                                                             |
| Different signs $\rightarrow$ negative                                      |
|                                                                             |
| 2.  Multiply the magnitudes, ignoring the signs.                            |
|                                                                             |
| 3.  Write the correct sign with the magnitude.                              |
+-----------------------------------------------------------------------------+

+-----------------------------------------------------------------------+
| - **Example** Multiply and divide integers                            |
+===================================+===================================+
| $$\ \ \ \ \ 5 \times ( - 6)$$     | $- 18 \div ( - 9)$                |
|                                   |                                   |
| 1\. answer is negative            | 1\. answer is positive            |
|                                   |                                   |
| 2\. $5 \times 6 = 30$             | 2\. $18 \div 9 = 2$               |
|                                   |                                   |
| 3\. $5 \times ( - 6) = - 30$      | 3\. $- 18 \div ( - 9) = 2$        |
