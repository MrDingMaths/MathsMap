# Properties of the Exponential Curve

- An **exponential relationship** $\mathbf{y =}\mathbf{a}^{\mathbf{x}}$
  produces **exponential curve** graphs.

- Exponential curves have these features:

  - An **asymptote**: a line the graph approaches but never reaches.

  - The **nature** of the graph (**growth** or **decay**) depends on the
    coefficient of $x$.

  - Consecutive $y$ values change by a **common ratio**, which is the
    base number $a$.
