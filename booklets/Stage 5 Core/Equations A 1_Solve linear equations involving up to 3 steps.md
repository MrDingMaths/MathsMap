# Multi-Step Equations with Pronumerals On Both Sides

- If an equation has pronumerals on both sides,\
  then **move all pronumerals to one side** by adding or subtracting a
  multiple of $x$.

1.  Move all pronumerals to one side.\
    You should try to keep the coefficient of $x$ positive, if possible.

2.  Solve for $x$ by applying inverse operations to both sides.

# Equations with Brackets and Unknown on Both Sides

- If an equation has brackets and pronumerals on both sides, you should
  expand the brackets.

1.  Expand any brackets.

2.  Simplify, if possible.

3.  Move $x$ to one side.

4.  Solve for $x$ by applying inverse operations to both sides.
