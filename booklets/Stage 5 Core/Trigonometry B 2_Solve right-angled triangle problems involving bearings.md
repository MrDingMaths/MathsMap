# True Bearings

- ![Diagram Description automatically
  generated](media/Trigonometry B 2_Solve right-angled triangle problems involving bearings/media/image6.png){width="2.109027777777778in"
  height="2.0027777777777778in"}Bearings:
  https://www.geogebra.org/m/er3jqacn

<!-- -->

- **Bearings** give a travel direction from one point to another.

- With bearings, we are always taking a **top-down view**.

- **True bearing** measures the angle to turn clockwise from north.

- True bearings **always use three digits**.\
  If a navigator only hears two digits, then they know they need it to
  be repeated.
