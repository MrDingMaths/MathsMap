# Scientific Notation (Standard Form)

- **Scientific notation**, or **standard form**, shows very large or
  very small numbers.

- Scientific notation expresses numbers as a multiple of a **power of
  10**.

  -----------------------------------------------------------------------------------------------------------------------------------
  **Scientific Notation (Standard Form)**
  -----------------------------------------------------------------------------------------------------------------------------------
  ![A black and white text Description automatically
  generated](media/Numbers of Any Magnitude 4_Express numbers in scientific notation/media/image4.png){width="3.8472222222222223in"
  height="1.1204385389326335in"}

  -----------------------------------------------------------------------------------------------------------------------------------

- Identify the largest place column to have value (i.e. the first
  significant figure),\
  then treat that as the new counting unit.

  - 6 500 000 "6.5 million" $6.5 \times 10^{6}$

  - 300 000 "3 hundred thousands" $3 \times 10^{5}$

  - 0.004 35 "4.35 thousandths" $4.35 \times 10^{- 3}$

# Converting to Scientific notation

1.  Move the decimal point after the first non-zero digit.

2.  Count digits between the new and old decimal points. This is the
    power of 10.

3.  Write the power of 10:

    a.  Positive for large numbers,

    b.  Negative for small numbers.

# Converting to Decimal Form

- For $a \times 10^{n}$

1.  Move the decimal left or right depending on the value of $n$.

    - If $n$ is positive, move the decimal $n$ places to the right.

    - If $n$ is negative, move the decimal $n$ places to the left.

- **Remember**: big numbers have positive power, small numbers have
  negative power.

### **Example** Represent numbers as decimal form.

+------------------------+---------------------------------------------+
| $6.48 \times 10^{5}$   | How do we know how many places to move the  |
|                        | decimal?                                    |
| $=$ $6\ 48000.$        |                                             |
|                        | *The ............... of 10 tells us how     |
| $$= 648\ 000$$         | many places to move the decimal.*           |
|                        |                                             |
|                        | How do we know if it will be a very small   |
|                        | or very large number?                       |
|                        |                                             |
|                        | $n$ *positive: ............ number*         |
|                        |                                             |
|                        | $n$ *negative: ............ number*         |
+------------------------+                                             |
| $3.51 \times 10^{- 6}$ |                                             |
|                        |                                             |
| $=$ $0.000003\ 51$     |                                             |
|                        |                                             |
| $= 0.000\ 003\ 51$     |                                             |

# Scientific Notation and Significant Figures

- For a number written in scientific notation $a \times 10^{n}$, every
  digit of $a$ is significant.

  - $1.02 \times 10^{5}$ has 3 significant digits.

  - $6.0 \times 10^{- 3}$ has 2 significant digits.

<!-- -->

- Rounding to a significant figure and scientific notation:

1.  Write the number in scientific notation $a \times 10^{n}$

2.  Round $a$ to the required significant figure.
