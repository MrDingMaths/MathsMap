# Quadratic Relationships

- **Linear relationships** are in the form $\mathbf{y = mx + c}$.\
  The graph is a straight line; so, the relationship is **linear.**

- **Quadratic relationships** are in the form
  $\mathbf{y = a}\mathbf{x}^{\mathbf{2}}\mathbf{+ bx + c}$.\
  Their graph will not be a straight line; **non-linear.**

- We graph quadratic relationships the same way as linear relationships:

  1.  Construct a table of values using the equation.

  2.  Graph the table by using the columns as coordinates.

  3.  Join the points to form a smooth curve.

# Exponential Relationships

- **Exponential relationships** are in the form
  $\mathbf{y =}\mathbf{a}^{\mathbf{x}}$.\
  Their graph will not be a straight-line; **non-linear.**

<!-- -->

- We graph exponential relationships the same way as before:

  1.  Construct a table of values using the equation.

  2.  Graph the table by using the columns as coordinates.

  3.  Join the points to form a smooth curve.

<!-- -->

- Consecutive $y$ values change by a **common ratio**, which is the base
  number $a$.
