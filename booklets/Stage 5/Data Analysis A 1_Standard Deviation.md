# Standard Deviation

+------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------+
| - **Standard Deviation**                                                                                                                                                           |
+=========================+====================================================+=====================================================================================================+
| **Standard deviation** is a number that describes how far data is spread from the **mean**.                                                                                        |
|                                                                                                                                                                                    |
| It measures the 'average' distance of each data value from the mean using a special formula.                                                                                       |
|                                                                                                                                                                                    |
| A **larger** standard deviation means **more** **spread** in the data.                                                                                                             |
|                                                                                                                                                                                    |
| A **lower** standard deviation means **less** **spread** and **more consistency**.                                                                                                 |

| - **Two Types of Standard Deviation**                                                                |
+------------------------------------------------------------------------------------------------------+
| $\sigma_{x}$ **population** standard deviation. ($\sigma$ is the Greek letter lowercase sigma)       |
|                                                                                                      |
| $s_{x}$ **sample** standard deviation.                                                               |
|                                                                                                      |
| **Always calculate population standard deviation** $\mathbf{\sigma}_{\mathbf{x}}$**,**\              |
| unless you are specifically asked to calculate sample standard deviation.                            |

4.  During Term 2, Rachel did 9 spelling tests. The tests were scored
    out of 20.\
    Her mean score is 15 and her standard deviation is 2.54.

    a.  If Rachel had scored 2 extra marks on every test,

        i.  What would happen to her mean?

Mean increases to 17

ii. Explain why her standard deviation would be unchanged.

The spread of the distribution has not changed.

b.  If Rachel had scored 15 on every test,

    i.  What would happen to her mean?

unchanged

ii. Explain why her standard deviation would be zero.

> No spread means 0 standard deviation

c.  What will happen to Rachel's standard deviation if she scores 15 on
    her next test?

15 is the mean. A value at the mean is 0 distance from the mean, so the
standard deviation decreases

d.  What will happen to Rachel's standard deviation if she scores 0 on
    her next test?

0 is 15 below the mean, which is much greater than the current SD of
2.54, so adding this value increases the spread and therefore the SD
