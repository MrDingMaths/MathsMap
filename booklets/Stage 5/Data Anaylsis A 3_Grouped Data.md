# Grouped Frequency Tables

+-------------------------------------------------------------------+
| - **Grouped Data**                                                |
+===================================================================+
| When handling many numerical values, we group them into intervals |
| called **classes** to spot patterns more easily.                  |
|                                                                   |
| Grouping data can reveal trends, but individual data values are   |
| lost.                                                             |

# Mean and Median from Grouped Data

+-------------------------------------------------------------------+
| - **Mean from Grouped Data**                                      |
+===================================================================+
| When grouping data, individual data points are lost.              |
|                                                                   |
| Therefore, any calculation of mean and median will be an          |
| estimate.                                                         |
|                                                                   |
| Estimation of mean is the same as calculation from ungrouped data |
| in tables, except we use the class centre.                        |
|                                                                   |
| 1.  Find the class centres.                                       |
|                                                                   |
| 2.  Multiply the class centres and the frequencies $cc \times f$  |
|                                                                   |
| 3.  Estimate the mean by dividing the sum of $cc \times f$ by the |
|     total frequency                                               |
|     $\overline{x} = \frac{\Sigma(cc \times f)}{\Sigma f}$         |
+-------------------------------------------------------------------+

+-------------------------------------------------------------------------------+
| - **Example** Estimate mean from a grouped frequency table                    |
+===============================================================================+
| Estimate the mean from this grouped frequency table:                          |
|                                                                               |
|   --------------------------------------------------------------------------- |
|         **Mass**        **Frequency**    **Class    $$\mathbf{cc \times f}$$  |
|                                         Centre**                              |
|   -------------------- --------------- ----------- -------------------------- |
|    $$40 < m \leq 50$$        12            45                 540             |
|                                                                               |
|    $$50 < m \leq 60$$        18            55                 990             |
|                                                                               |
|    $$60 < m \leq 70$$        20            65                 1300            |
|                                                                               |
|    $$70 < m \leq 80$$        13            75                 975             |
|                                                                               |
|          Total               63                               3805            |
|   --------------------------------------------------------------------------- |
|                                                                               |
| $$\overline{x} = \frac{\Sigma ccf}{\Sigma f} = \frac{3805}{63} = 60.4$$       |
+-------------------------------------------------------------------------------+

17. **2018 HSC Standard 2 Band 5\**
    A cumulative frequency table for a data set is shown. What is the
    interquartile range?

![](media/Data Anaylsis A 3_Grouped Data/media/image53.png){width="2.066666666666667in"
height="2.0909722222222222in"}

There are $42$ scores in total. The data can be split into two halves
of $21$ scores each.\
\
The lower quartile ($Q_{1}$) is the median of the first $21$ scores,
which is the $11\text{th}$ score.\
the $11\text{th}$ score falls in the score group $3$ (which contains
the $10\text{th}$ to $16\text{th}$ scores).\
So, $Q_{1} = 3$.\
\
The upper quartile ($Q_{3}$) is the median of the second half of
scores,\
which is the $11\text{th}$ score of the second half, or
the $32\text{nd}$ score overall.\
the $32\text{nd}$ score falls in the score group $5$ (which contains
the $21\text{st}$ to $34\text{th}$ scores).\
So, $Q_{3} = 5$.\
\
$$\begin{matrix}
\text{Interquartile range (IQR)} & = Q_{3} - Q_{1} \\
 & = 5 - 3 \\
 & = 2
\end{matrix}
$$
