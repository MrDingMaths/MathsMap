# Syllabus Content

**MA5-TRG-P-02** establishes and applies the properties of trigonometric
functions and finds solutions to trigonometric equations

**Use the unit circle to define trigonometric functions and represent
them graphically**

- Redefine the sine and cosine ratios in terms of the unit circle

- Verify that the tangent ratio can be expressed as a ratio of the sine
  and cosine ratios

- Use graphing applications to examine the sine, cosine and tangent
  ratios for (at least) $0^{{^\circ}} \leq x \leq 360^{{^\circ}}$, and
  graph the results

- Use graphing applications to examine graphs of the sine, cosine and
  tangent functions for angles of any magnitude, including negative
  angles

- Use the unit circle or graphs of trigonometric functions to establish
  and apply the relationships ${\sin{A = \sin}}(180{^\circ} - A)$,
  ${\cos{A = - \cos}}(180{^\circ} - A)$, and
  ${\tan{A = - \tan}}(180{^\circ} - A)$ for obtuse angles when
  $0{^\circ} \leq A \leq 90{^\circ}$

- Establish and apply the relationship $m = \tan\theta,\$where $m$ is
  the gradient of the line and $\theta$ is the angle of inclination of a
  line with the $x$ -axis on the Cartesian plane

**Solve trigonometric equations using exact values and the relationships
between supplementary and complementary angles**

- Derive and apply the exact sine, cosine and tangent ratios for angles
  of $30{^\circ}$, $45{^\circ}$ and  $60{^\circ}$

- Verify and use the relationships between the sine and cosine ratios of
  complementary angles in right-angled triangles:
  $\sin A = \cos{(90{^\circ} - A)}$ and $\cos A = \sin(90{^\circ} - A)$

- Find the possible acute and/or obtuse angles, given a trigonometric
  ratio

- Apply the sine rule and area rule to find angles involving the
  ambiguous case

11. Prove, using exact values, that:

    a.  $\sin{330{^\circ}}\cos{150{^\circ}} - \cos{390{^\circ}}\sin{390{^\circ}} = 0$

$$LHS = - \sin{30{^\circ}}\left( - \cos{30{^\circ}} \right) - \cos{30{^\circ}}\sin 30$$

$$\ \ \ \ \ \ \ \ \  = \sin{30{^\circ}}\cos{30{^\circ}} - \cos{30{^\circ}}\sin{30{^\circ}}$$

$$\ \ \ \ \ \ \ \  = 0 = RHS$$

b.  $\sin{420{^\circ}}\cos{405{^\circ}} + \cos{420{^\circ}}\sin{405{^\circ}} =$
    $\frac{\sqrt{3} + 1}{2\sqrt{2}}$

$$LHS = \sin{60{^\circ}}\cos{45{^\circ}} + \cos{60{^\circ}}\sin{45{^\circ}}$$

$$\ \ \ \ \ \ \ \ \  = \frac{\sqrt{3}}{2}\left( \frac{1}{2} \right) + \frac{1}{2}\left( \frac{1}{\sqrt{2}} \right)$$

$$\ \ \ \ \ \ \ \ \  = \frac{\sqrt{3}}{2\sqrt{2}} + \frac{1}{2\sqrt{2}}$$

$$\ \ \ \ \ \ \ \ \  = \frac{\sqrt{3} + 1}{2\sqrt{2}} = RHS$$

c.  $\frac{\sin{135{^\circ}} - \cos{120{^\circ}}}{\sin{135{^\circ}} + \cos{120{^\circ}}}$
    $= 3 + 2\sqrt{2}$

$$LHS = \frac{\sin{45{^\circ}} - \left( - \cos{60{^\circ}} \right)}{\sin{45{^\circ}} + \left( - \cos{60{^\circ}} \right)}$$

$$\ \ \ \ \ \ \ \ \  = \frac{\left( \frac{1}{\sqrt{2}} + \frac{1}{2} \right)}{\left( \frac{1}{\sqrt{2}} - \frac{1}{2} \right)}$$

$$\ \ \ \ \ \ \ \ \  = \frac{\left( \frac{2 + \sqrt{2}}{2\sqrt{2}} \right)}{\left( \frac{2 - \sqrt{2}}{2\sqrt{2}} \right)}$$

$$\ \ \ \ \ \ \ \ \  = \frac{2 + \sqrt{2}}{2 - \sqrt{2}}\ \frac{\times 2 + \sqrt{2}}{\times 2 + \sqrt{2}}$$

$$\ \ \ \ \ \ \ \ \  = \frac{4 + 4\sqrt{2} + 2}{4 - 2}$$

$$\ \ \ \ \ \ \ \ \  = \frac{6 + 4\sqrt{2}}{2}$$

$$\ \ \ \ \ \ \ \ \  = 3 + 2\sqrt{2} = RHS$$

d.  $\left( \sin{150{^\circ}} + \cos{270{^\circ}} + \tan{315{^\circ}} \right)^{2} = \sin^{2}{135{^\circ}}\cos^{2}{225{^\circ}}$

$$LHS = \left( \sin{30{^\circ}} + 0 + \left( - \tan{45{^\circ}} \right) \right)^{2}$$

$$\ \ \ \ \ \ \ \ \  = \left( \frac{1}{2} - 1 \right)^{2}$$

$$\ \ \ \ \ \ \ \ \  = \frac{1}{4}$$

$$RHS = \sin^{2}{45{^\circ}}\left( - - \cos^{2}{45{^\circ}} \right)$$

$$\ \ \ \ \ \ \ \ \ \  = \left( \frac{1}{\sqrt{2}} \right)^{2}\left( - \frac{1}{\sqrt{2}} \right)^{2}$$

$$\ \ \ \ \ \ \ \ \ \  = \frac{1}{4}$$

$$\therefore LHS = RHS$$

e.  $\frac{\sin{120{^\circ}}}{\tan{300{^\circ}}}$ $-$
    $\frac{\cos{240{^\circ}}}{\left( \frac{1}{\tan{315{^\circ}}} \right)}$
    $= \tan^{2}{240{^\circ}} -$ $\frac{1}{\sin^{2}{330{^\circ}}}$

$$LHS = \frac{\sin{60{^\circ}}}{- \tan{60{^\circ}}} - \frac{- \cos{60{^\circ}}}{\left( \frac{1}{- \tan{45{^\circ}}} \right)}$$

$$\ \ \ \ \ \ \ \  = \frac{\left( \frac{\sqrt{3}}{2} \right)}{- \sqrt{3}} - \frac{\left( - \frac{1}{2} \right)}{- 1}$$

$$\ \ \ \ \ \ \ \  = - \frac{1}{2} - \frac{1}{2}$$

$$\ \ \ \ \ \ \ \  = - 1$$

$$RHS = \tan^{2}{60{^\circ}} - \frac{1}{\sin^{2}{30{^\circ}}}$$

$$\ \ \ \ \ \ \ \ \  = \left( \sqrt{3} \right)^{2} - \frac{1}{\left( \frac{1}{2} \right)^{2}}$$

$$\ \ \ \ \ \ \ \ \  = 3 - \frac{1}{\left( \frac{1}{4} \right)}$$

$$\ \ \ \ \ \ \ \  = 3 - 4$$

$= - 1$

$$\therefore LHS = RHS$$

12. Prove that

$$\frac{\cos{(90{^\circ} - \theta)}\ \sin{(180{^\circ}} - \theta)\ }{\cos(180{^\circ} + \theta)\ \cos( - \theta)} = - \tan^{2}\theta$$

$$LHS = \frac{\sin\theta\sin\theta}{{- cos}\theta\cos\theta}$$

$$\ \ \ \ \ \ \ \ \  = \frac{\sin^{2}\theta}{{- \cos^{2}}\theta}$$

$$\ \ \ \ \ \ \ \ \  = - \tan^{2}\theta = RHS$$

+===================================================================================================================================+

|                                                                                                                                   |

+------------------------------------------------------------------------------+
| - **Finding the Reference Angle from a Calculator**                          |
+==============================================================================+
| To find the acute reference angle $A$:                                       |
|                                                                              |
| - Use the relevant inverse trigonometric function:                           |
|   $\sin^{- 1}{(\ \ \ \ )},\ \cos^{- 1}{(\ \ \ \ )},\ \tan^{- 1}{(\ \ \ \ )}$ |
|                                                                              |

|                                                                              |
| - The calculator returns the acute reference angle $A$.                      |
|                                                                              |
| The calculator\'s inverse functions give results in restricted ranges that   |
| may include negative angles.\                                                |
| By entering the positive value, we always get a positive acute angle and     |
| this leads to fewer mistakes.\                                               |
| You will learn more about the inverse trigonometric functions in the Year 12 |
| Advanced course.                                                             |

+-------------------------------------------------------------------+
| - **Solving Ambiguous Case Sine Rule Questions**                  |
+===================================================================+
| If the question doesn't tell you whether a triangle has an acute  |
| or obtuse angle:                                                  |
|                                                                   |
| 1.  Use sine rule to find the unknown acute angle $\theta.$       |
|                                                                   |
| 2.  Find the obtuse angle $\theta' = 180{^\circ} - \theta$.       |
|                                                                   |
| 3.  Add $\theta'$ to the given angle. If the sum $> 180{^\circ}$, |
|     it is not a valid possibility\                                |
|     (angle sum of triangle cannot exceed 180°).                   |
|                                                                   |
| - Diagrams are never drawn to scale. In an exam, if you see a     |
|   sine rule question worth 3 marks or more, it is likely the      |
|   ambiguous case and marks will be awarded for justifying all     |
|   possible answers.                                               |
