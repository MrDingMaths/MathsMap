# Syllabus Content

**MA5-FNC-P-01** uses function notation to describe and graph functions
of one variable and graphs inequalities in one and 2 variables

**Graph regions corresponding to linear inequalities in one and 2
variables**

- Graph linear inequalities of the form $ax + by > c$, testing whether
  the points satisfy the given inequality and shading appropriate
  regions
