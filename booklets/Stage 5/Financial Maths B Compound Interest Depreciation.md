# Syllabus Content

**MA5-FIN-C-02** solves financial problems involving compound interest
and depreciation

**Solve problems involving compound interest and depreciation**

- Examine compound interest for up to 3 time periods using repetition of
  the formula for simple interest

- Associate the calculation of the total value of a compound interest
  investment with repeated multiplication, using digital tools

- Establish and use the formula $FV = PV(1 + r)^{n}$ to find compound
  interest where $FV$ $=$ future value of the investment, $PV =$ present
  value of the investment, $r = \$interest rate per time period and
  $n =$ number of time periods

- Solve problems involving compound interest

- Compare simple interest with compound interest in practical situations

- Use the compound interest formula to establish the depreciation
  formula $S = V_{0}(1 - r)^{n}\$where $S =$ salvage value,
  $V_{0} = \$initial value of the asset, $r =$ depreciation rate per
  time period and $n =$ number of periods

- Solve problems involving the depreciation of an asset

# Compound Interest Formula

+--------------------------------------------------------------------+
| - **Investigation** the compound interest formula                  |
+====================================================================+
| \$1000 is invested at a **compound** interest rate of 10% p.a.     |
|                                                                    |
| Complete this table.                                               |
|                                                                    |
|   ---------------------------------------------------------------- |
|    **Years**    **Starting    **Interest paid that     **Ending    |
|                 Balance**            year**           Balance**    |
|   ----------- -------------- ---------------------- -------------- |
|        1       $$\$ 1000$$    $$1000 \times 10\%$$   $$\$ 1100$$   |
|                                                                    |
|        2       $$\$ 1100$$    $$1100 \times 10\%$$                 |
|                                                                    |
|        3                                                           |
|   ---------------------------------------------------------------- |
|                                                                    |
| Every year, the ending balance increases by 10%, which means       |
| multiplying by 1.1.                                                |
|                                                                    |
| After 1 year, $A = \$ 1000 \times 1.1$                             |
|                                                                    |
| After 2 years, $A = \$ 1000 \times 1.1 \times 1.1$                 |
|                                                                    |
| After 3 years, $A = \$ 1000 \times$ .... $\times$ .... $\times$    |
| ....                                                               |
|                                                                    |
| If you were to write the ending balance after 6 years, it would    |
| be:                                                                |
|                                                                    |
| $A = \$ 1000 \times$ ...... $\times$ ...... $\times$ ......        |
| $\times$ ...... $\times$ ...... $\times$ ......                    |
|                                                                    |
| $= \$ 1000 \times (\ \ \ \ \ \ \ \ )^{\ldots}$                     |
|                                                                    |
| If you were to write the ending balance after $n$ years, it would  |
| be:                                                                |
|                                                                    |
| $A = \$ 1000 \times (\ \ \ \ \ \ \ \ )^{\ldots}$                   |
|                                                                    |
| For any initial amount $P$, and interest rate $r$, the ending      |
| balance after $n$ years would be:                                  |
|                                                                    |
| $A =$ ......... $\times$ $(1 + \ \ \ \ \ )^{\ldots}$               |

| - **Compound Interest Future Value Formula**                      |
+===================================================================+
| The initial amount (**principal**) is called the **present        |
| value** (PV).                                                     |
|                                                                   |
| The **final** **amount** after growth is the **future value**     |
| (FV).                                                             |
|                                                                   |
| $\begin{matrix}                                                   |
| present\  \\                                                      |
| value                                                             |
| \end{matrix}\$ $\begin{matrix}                                    |
| number\ of \\                                                     |
| periods                                                           |
| \end{matrix}$                                                     |
|                                                                   |
| $$FV = PV(1 + r)^{n}$$                                            |
|                                                                   |
| $\begin{matrix}                                                   |
| future\  \\                                                       |
| value                                                             |
| \end{matrix}$ $\begin{matrix}                                     |
| interest\ rate\  \\                                               |
| per\ period                                                       |
| \end{matrix}$                                                     |

15. \$16 000 is invested for 5 years at 8% compounded annually.

$$I = Prn$$

$$7509.25 = 16000 \times r \times 5$$

$$\frac{170\ 814.45}{100\ 000} \times 100 = 70.81\%$$
