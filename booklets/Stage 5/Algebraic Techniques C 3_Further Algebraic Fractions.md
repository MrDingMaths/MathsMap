| - **Identify** whether a fraction can be simplified                                                                                                            |
+======================+==============+==============+==================================+==================+==================+==================================+
| A fraction $\frac{A}{B}$ can be simplified if and only if $A$ and $B$ share a common factor other than 1.                                                      |
+----------------------+-----------------------------+----------------------------------+-------------------------------------+----------------------------------+
| ✖                    | ✔                           | ✔                                | ✔                                   | ✖                                |
|                      |                             |                                  |                                     |                                  |
| $$\frac{2x + 3}{3}$$ | $$\frac{4x + 8}{2}$$        | $$\frac{(x - 3)(x + 3)}{x + 3}$$ | $$\frac{(x + 2)(x + 3)}{x + 2}$$    | $$\frac{(x + 2)(x + 3)}{x - 2}$$ |
+----------------------+-----------------------------+----------------------------------+-------------------------------------+----------------------------------+
| Decide whether these fractions can be simplified.                                                                                                              |
+-------------------------------------+--------------------------------------------------------------------+-----------------------------------------------------+

| d.                                  | e.                                                                 | f.                                                  |
|                                     |                                                                    |                                                     |
| $$\frac{x^{2} + 3x}{x}$$            | $$\frac{(x + 1)(x + 2)}{x + 5}$$                                   | $$\frac{(x + 4)(x + 2)}{x + 2}$$                    |

10. Kylie simplifies $\frac{4x + 6}{2}$ and gets $2x + 6$. Explain her
    error and give the correct answer.

She divided only the first term by 2. Every term in the numerator must
be divided: $\frac{4x + 6}{2} = \frac{4x}{2} + \frac{6}{2} = 2x + 3$.

Equivalently, factorise first: $\frac{2(2x + 3)}{2} = 2x + 3$.

15. A student writes $\frac{x + 3}{x + 5} = \frac{3}{5}$ by "cancelling
    the $x$".

Explain why this is wrong and show a numerical counterexample.

Only common factors of the entire numerator and denominator can be
cancelled, not individual terms.

Counterexample: let $x = 2$. Left side: $\frac{5}{7}$. Right side:
$\frac{3}{5}$. These are not equal.

Mastery

16. The expression $\frac{6x + k}{3}$ simplifies to $2x + 4$. Find $k$.

$\frac{6x + k}{3} = 2x + \frac{k}{3}$*.* So $\frac{k}{3} = 4$*,* giving
$k = 12$

17. For what value of $n$ does $\frac{nx - 6}{x - 2} = 3$?

The numerator must equal $3(x - 2) = 3x - 6$*.* So $n = 3$*.*

18. Is the following statement always, sometimes, or never true?
    Explain.

$$\frac{a + b}{a} = 1 + \frac{b}{a}$$

Always true for when $a \neq 0$.

Splitting the fraction:
$LHS = \frac{a + b}{a} = \frac{a}{a} + \frac{b}{a} = 1 + \frac{b}{a} = RHS$

19. A path has area $(10x - 15)$ m² and length $(2x - 3)$ m.\
    Show that the width of the path is constant and state its value.

Width $= \frac{10x - 15}{2x - 3} = \frac{5(2x - 3)}{2x - 3} = 5$ m. The
width is 5 m regardless of $x$.

11. Explain why $\frac{x^{2} - 9}{x - 3}$ is not defined when $x = 3$.

The original expression has $(x - 3)$ in the denominator, so it is
undefined at $x = 3$ (division by zero).

The simplified form $x + 3$ does not carry this restriction, so the two
expressions are equivalent only for $x \neq 3$.

12. A square has area $(x^{2} - 12x + 36)$ cm². Write an expression for
    its side length.

Area of square $= \text{side}^{2}$, so side
$= \sqrt{x^{2} - 12x + 36} = \sqrt{(x - 6)^{2}} = x - 6$ cm (for
$x > 6$).

15. Find values of $a$ and $b$ such that $\frac{x^{2} + ax + b}{x + 3}$
    $= x - 4$ for all valid $x$.

Multiply both sides by $(x + 3)$:
$x^{2} + ax + b = (x + 3)(x - 4) = x^{2} - x - 12$.

So $a = - 1$, $b = - 12$.

16. The expression $\frac{x^{2} + 5x + 6}{x^{2} + ax + b}$ simplifies to
    $\frac{x + 2}{x + 5}$. Find $a$ and $b$.

Numerator $= (x + 2)(x + 3)$. The factor $(x + 3)$ cancels, so the
denominator must have $(x + 3)$ as a factor and the remaining factor is
$(x + 5)$. Denominator $= (x + 3)(x + 5) = x^{2} + 8x + 15$. So $a = 8$,
$b = 15$.

17. For which integer values of $k$ does $\frac{x^{2} + kx + 12}{x + 3}$
    simplify to a linear expression?

$(x + 3)$ must be a factor of $x^{2} + kx + 12$. Write
$x^{2} + kx + 12 = (x + 3)(x + c)$ where $3c = 12$, so $c = 4$ and
$k = 3 + 4 = 7$.

13. Without fully simplifying, explain why
    $\frac{x^{2} - 9}{x + 3} \times \frac{x + 3}{x^{2} - 9} = 1$ for
    valid values of $x$.

The two fractions are reciprocals of each other. Any non-zero quantity
multiplied by its reciprocal equals 1.

14. The product
    $\frac{x^{2} + bx + c}{x^{2} - 9} \times \frac{x - 3}{x + 2}$
    simplifies to $\frac{x + 5}{x + 2}$. Find $b$ and $c$.

After simplification the $(x - 3)$ in the numerator must cancel with a
factor in $x^{2} - 9 = (x - 3)(x + 3)$, leaving $(x + 3)$ in the
denominator.

Then the numerator $x^{2} + bx + c$ divided by $(x + 3)$ must give
$(x + 5)$,

so $x^{2} + bx + c = (x + 3)(x + 5) = x^{2} + 8x + 15$.

Thus $b = 8$, $c = 15$.

+---------------------------------------------------------------------------------------------------------------------------------------------------------------------+
| - **Identify** main numerator and main denominator                                                                                                                  |
+=============================================+=====================================================+=================================================================+
| A compound fraction has a fraction in its numerator, denominator, or both.                                                                                          |
|                                                                                                                                                                     |
| Dealing with these fractions requires identifying the 'main' numerator and 'main' denominator.                                                                      |
+---------------------------------------------+-----------------------------------------------------+-----------------------------------------------------------------+
| a.                                          | b.                                                  | c.                                                              |
|                                             |                                                     |                                                                 |
| $$\frac{\frac{1}{x}}{\frac{2}{x + 1}}$$     | $$\frac{\frac{3}{x}}{1 - \frac{5}{x}}$$             | $$\frac{1 - \frac{2}{x}}{3}$$                                   |
+---------------------------------------------+-----------------------------------------------------+-----------------------------------------------------------------+
| d.                                          | e.                                                  | f.                                                              |

| $$\frac{1 + \frac{1}{x}}{1 - \frac{1}{x}}$$ | $$\frac{2 + \frac{1}{x + 1}}{3 - \frac{1}{x + 1}}$$ | $$\frac{\frac{1}{x} + \frac{1}{y}}{\frac{1}{x} - \frac{1}{y}}$$ |

+-------------------------------------------------------------------+
| - **Simplifying Compound Fractions**                              |
+===================================================================+
| 1.  Identify the main numerator and main denominator.             |
|                                                                   |
| 2.  Multiply every single term in the main numerator and          |
|     denominator by the LCD.                                       |
|                                                                   |
| 3.  Simplify.                                                     |
|                                                                   |
| This is the fastest method. An alternate method is:               |
|                                                                   |
| 1.  Create a single fraction in the main numerator and            |
|     denominator.                                                  |
|                                                                   |
| 2.  Divide the numerator by the denominator.                      |
|                                                                   |
| 3.  Simplify.                                                     |
+-------------------------------------------------------------------+
