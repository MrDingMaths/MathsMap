# Syllabus Content

**MA5-EQU-P-02** solves linear equations of more than 3 steps, monic and
non-monic quadratic equations, and linear simultaneous equations

**Solve quadratic equations using a variety of methods**

- Solve equations of the form ${ax}^{2} + bx + c = 0$ by factorisation
  and by completing the square

- Apply the quadratic formula
  $x = \frac{- b \pm \sqrt{b^{2}\  - \ 4ac}}{2a}$ to solve quadratic
  equations

- Apply the most appropriate method to solve a variety of quadratic
  equations

- Use substitution to verify solutions to quadratic equations

- Identify whether a given quadratic equation has real solutions, and if
  there are real solutions, whether or not they are equal

- Solve quadratic equations resulting from substitution into formulas in
  various contexts

- Model and solve word problems using quadratic equations in various
  contexts

- Solve equations that are reducible to quadratics

# Non-Monic Quadratic Equations

+----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------+
| - **Null Factor Law**                                                                                                                                                                  |
+===================================+====================================================================================================================================================+
| **The null factor law** says that if two factors multiply to zero, one or both factors must be zero.                                                                                   |
+-----------------------------------+----------------------------------------------------------------------------------------------------------------------------------------------------+
| If $a \times b = 0$               | If $(ax + b)(cx + d) = 0$                                                                                                                          |
|                                   |                                                                                                                                                    |
| Then $a = 0$ or $b = 0$           | Then $ax + b = 0\ \ \ \ \ \ \ \ or\ \ \ \ \ \ \ \ cx + d = 0$                                                                                      |
|                                   |                                                                                                                                                    |
|                                   | $$\ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ x = - \frac{b}{a}\ \ \ or\ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ x = - \frac{d}{c}$$ |
+-----------------------------------+----------------------------------------------------------------------------------------------------------------------------------------------------+

# Quadratic Modelling

+-------------------------------------------------------------------+
| - **Quadratic Modelling**                                         |
+===================================================================+
| Many scenarios can be modelled and solved using a quadratic       |
| equation.                                                         |
|                                                                   |
| 1.  Let a variable represent the unknown number.                  |
|                                                                   |
| 2.  Write an equation to describe the problem. (A diagram is      |
|     helpful).                                                     |
|                                                                   |
| 3.  Solve the equation.                                           |
|                                                                   |
| 4.  Check your answer makes sense and has correct units.          |
+-------------------------------------------------------------------+

+-----------------------------------------------------------------------+
| - **Example** Model and solve problems using quadratic equations      |
+===================================+===================================+
| The area of a rectangle is fixed at 28 m² and its length is 3 m more  |
| than its breadth.\                                                    |
| Find the width of the rectangle.                                      |
+-----------------------------------+-----------------------------------+
| 1\. Let $x$ be the width.         | 3\. $x^{2} + 3x = 28$             |
|                                   |                                   |

|                                   | $(x + 7)(x - 4) = 0$              |
|                                   |                                   |
|                                   | $x = - 7\$or 4                    |

|                                   | $(since\ x > 0)$                  |

7.  Without completing the square fully, explain how you can tell
    immediately that $x^{2} + 4x + 10$ cannot be written as a difference
    of two squares.

Completing the square gives $(x + 2)^{2} + c$. Since $a = 2$, we get

A difference of two squares requires $c < 0$, so the expression cannot
be factorised over the reals.

8.  The expression $x^{2} + bx + 16$ completes to give a perfect square
    $(x + a)^{2}$ with no $c$ term.\
    Find all possible integer values of $b$.

Completing the square gives
$\left( x + \frac{b}{2} \right)^{2} + \left( 16 - \frac{b^{2}}{4} \right)$.

For no $c$ term: $16 - \frac{b^{2}}{4} = 0$, so $b^{2} = 64$, giving
$b = \pm 8$.

# Solving Quadratic Equations by Completing the Square

+-------------------------------------------------------------------+
| - **Solving Quadratic Equations by Completing the Square**        |
+===================================================================+
| When we can't use the PSF method to factorise a quadratic, we can |
| solve a quadratic equation by completing the square.              |
|                                                                   |
| 1\. Write the quadratic in the form $(x + a)^{2} + c$ by          |
| completing the square.                                            |
|                                                                   |
| 2\. Solve by isolating $(x + a)^{2}.$                             |
|                                                                   |
| - Remember that a quadratic equation can have **no real           |
|   solution.**                                                     |

+-------------------------------------------------------------------------------+

+===================================+===========================================+

| 1\. $+ 4^{2}\ \ \ \ \  + 4^{2}$   | 1\.                                       |
|                                   | $+ {( - 3)}^{2}\ \ \ \ \  + {( - 3)}^{2}$ |
| $x^{2} + 8x + 16 = 36$            |                                           |
|                                   | $x^{2} - 6x + 9 + 2 = 9$                  |
| 2\. $(x + 4)^{2} = 36$            |                                           |
|                                   | 2\. $(x - 3)^{2} + 2 = 9$                 |
| 3\. $x + 4 = \pm 6$               |                                           |
|                                   | 3\. $(x - 3)^{2} = 7$                     |
| $x = - 4 \pm 6$                   |                                           |

|                                   | $x = 3 \pm \sqrt{7}$                      |

10. Consider $x^{2} - 2ax + 4 = 0$, where $a$ is a constant.

    a.  Find the solutions in terms of $a$.

$${x^{2} - 2ax + a^{2} - a^{2} + 4 = 0
}{(x - a)^{2} = a^{2} - 4
}{x - a = \pm \sqrt{a^{2} - 4}
}{x = a \pm \sqrt{a^{2} - 4}}$$

b.  Show that the product of the solutions is always a constant.

$$\ \ \ \ \ \left( a + \sqrt{a^{2} - 4} \right)\left( a - \sqrt{a^{2} - 4} \right)$$

+----------------------------------------------------------------------------------------------------------------------------------------+
| - **Derive** the quadratic formula                                                                                                     |
+========================================================================================================================================+
| The quadratic formula comes from completing the square on the general equation:                                                        |
|                                                                                                                                        |
| $$ax^{2} + bx + c = 0$$                                                                                                                |
|                                                                                                                                        |
| a.  Make the quadratic monic by dividing both sides by $a$.                                                                            |
|                                                                                                                                        |
| $$x^{2} + \frac{b}{a}\ x + \frac{c}{a} = 0$$                                                                                           |
|                                                                                                                                        |
| b.  Complete the square.                                                                                                               |
|                                                                                                                                        |
| $$x^{2} + \frac{b}{a}\ x + \left( \frac{b}{2a} \right)^{2} - \left( \frac{b}{2a} \right)^{2} + \frac{c}{a} = 0$$                       |
|                                                                                                                                        |
| $$\left( x + \frac{b}{2a} \right)^{2} - \left( \frac{b}{2a} \right)^{2} + \frac{c}{a} = 0$$                                            |
|                                                                                                                                        |
| c.  Solve for $x$.\                                                                                                                    |
|     Use two methods: Isolating the squared term and factorising by completing the square.                                              |
|                                                                                                                                        |
| Method 1:                                                                                                                              |
|                                                                                                                                        |
| $$\left( x + \frac{b}{2a} \right)^{2} = \left( \frac{b}{2a} \right)^{2} - \frac{c}{a}$$                                                |
|                                                                                                                                        |
| $$\left( x + \frac{b}{2a} \right)^{2} = \frac{b^{2}}{4a^{2}} - \frac{c}{a}$$                                                           |
|                                                                                                                                        |
| $$\left( x + \frac{b}{2a} \right)^{2} = \frac{b^{2} - 4ac}{4a^{2}}$$                                                                   |
|                                                                                                                                        |
| $$x + \frac{b}{2a} = \frac{\pm \sqrt{b^{2} - 4ac}}{2a}$$                                                                               |
|                                                                                                                                        |
| $$x = \frac{- b \pm \sqrt{b^{2} - 4ac}}{2a}$$                                                                                          |
|                                                                                                                                        |
| Method 2:                                                                                                                              |
|                                                                                                                                        |
| $$\left( x + \frac{b}{2a} \right)^{2} - \frac{b^{2}}{4a^{2}} + \frac{c}{a} = 0$$                                                       |
|                                                                                                                                        |
| $$\left( x + \frac{b}{2a} \right)^{2} - \frac{b^{2} - 4ac}{4a^{2}} = 0$$                                                               |
|                                                                                                                                        |
| Using difference of two squares                                                                                                        |
|                                                                                                                                        |
| $$\left( x + \frac{b}{2a} - \frac{\sqrt{b^{2} - 4ac}}{2a} \right)\left( x + \frac{b}{2a} + \frac{\sqrt{b^{2} - 4ac}}{2a} \right) = 0$$ |
|                                                                                                                                        |

13. Show that $x^{2} - 4px + \left( 4p^{2} - 0 \right) = 0$ always has

$$= 2p \pm 3$$

So, integer solutions for all integer $p$

Since the discriminant is always negative, this equation never has any

16. Show that for all $m$, the equation
    $x^{2} - \left( m^{2} + m \right)x + m^{3} = 0$ always yields two
    solutions where one solution is the perfect square of the other.

$$\Delta = \left( - \left( m^{2} + m \right) \right)^{2} - 4(1)\left( m^{3} \right)$$

$$= m^{4} - 2m^{3} + m^{2}$$

$$= m^{2}(m - 1)^{2}$$

The solutions are

$$x_{1} = \frac{m^{2} + m + m(m - 1)}{2} = m^{2}$$

$$x_{2} = \frac{m^{2} + m - m(m - 1)}{2} = m$$

17. The equation
    $\left( k^{2} - 1 \right)x^{2} - 2\left( k^{2} + 1 \right)x + \left( k^{2} - 1 \right) = 0$
    can yield distinct, rational solutions that are reciprocals of each
    other. For what values of $k$ is this true?

$$\Delta = \left( - 2\left( k^{2} + 1 \right) \right)^{2} - 4\left( k^{2} - 1 \right)\left( k^{2} - 1 \right)$$

$$= 16k^{2}$$

The solutions are

$$x = \frac{2\left( k^{2} + 1 \right) \pm 4k}{2\left( k^{2} - 1 \right)}$$

$$= \frac{k^{2} \pm 2k + 1}{k^{2} - 1}$$

$$= \frac{(k + 1)^{2}}{(k - 1)(k + 1)} = \frac{k + 1}{k - 1}$$

Or

$$\frac{(k - 1)^{2}}{(k - 1)(k + 1)} = \frac{k - 1}{k + 1}$$

The solutions would be undefined for $k = \pm 1$

Suppose $\frac{k - 1}{k + 1} = \frac{k + 1}{k - 1}$

$$(k - 1)^{2} = (k + 1)^{2}$$

$$- 2k = 2k$$

$\therefore$ roots are equal when $k = 0$

So distinct, rational solutions as long as $k$ is a rational number and
$k \neq - 1,0,1$
