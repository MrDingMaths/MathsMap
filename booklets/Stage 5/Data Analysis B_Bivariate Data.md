# Syllabus Content

**MA5-DAT-C-02** displays and interprets datasets involving bivariate
data

Identify and describe numerical datasets involving 2 variables

- Distinguish between situations involving 1-variable and 2-variable
  (bivariate) data and explain when each is needed

- Explain the difference between variables that have an association and
  variables that have a causal relationship

- Identify and describe the independent variable and dependent variable
  in relationships with possible cause and effect

**Represent datasets involving 2 numerical variables, using a scatter
plot and a line of best fit, by eye**

- Gather data on a topic of interest involving 2 numerical variables

- Represent the data using a scatter plot

- Create a line of best fit, by eye, on an existing scatter plot

**Interpret data involving 2 numerical variables, using graphical
representations**

  apply terminology about form (linear), strength (strong, moderate or
  weak) and direction (positive or negative)

  data values (interpolation) and what might happen beyond known data
  values (extrapolation)

- Explain the limitations of the model when making predictions

# Bivariate Data

+-------------------------------------------------------------------+
| - **Independent and Dependent Variables**                         |
+===================================================================+
| **Univariate** **data** counts the frequency of a single          |
| variable.                                                         |
|                                                                   |
| **Bivariate data** explores relationships between two variables.  |
|                                                                   |
| The **independent variable** often causes changes in the          |
| **dependent variable,** but not always.                           |
|                                                                   |
| For example, for a relationship between maths test scores and     |
| hours spent on social media, Mr Ding's hypothesis might be that   |
| more hours on social media mean lower test scores.                |
|                                                                   |
| - The independent variable would be hours on social media.        |
|                                                                   |
| - The dependent variable would be maths test score.               |

# Drawing a Line of Best Fit

+-------------------------------------------------------------------+
| - **Line of Best Fit**                                            |
+===================================================================+
| A line of best fit:                                               |
|                                                                   |
| - Represents most of the points as closely as possible.           |
|                                                                   |
| - Goes through as many points as possible.                        |
|                                                                   |
| - Has roughly the same number of points above and below it.       |
|                                                                   |
| - Is drawn so that the distances of points from the line are as   |
|   small as possible.                                              |
|                                                                   |
| <!-- -->                                                          |
|                                                                   |
| - Do **NOT** extend the line past existing data points, as the    |
|   relationship may not continue outside of this range.            |
|                                                                   |
| - A line of best fit may not "fit" the data nicely if there is an |
|   outlier.                                                        |
|                                                                   |
| A line of best fit is also referred to as a **regression** line,  |
| if it was generated using a special method using a computer.      |

# Interpolating and Extrapolating using the Equation

+-------------------------------------------------------------------------------------------+
| - ![](media/Data Analysis B_Bivariate Data/media/image88.png){width="2.942361111111111in" |
|   height="2.1368055555555556in"}**Interpolating and Extrapolating using the Equation**    |
+===========================================================================================+
| We can use the equation of the line to interpolate and extrapolate without using the line |
| itself.                                                                                   |
|                                                                                           |
| 1.  Identify the given variable: independent ($x$) or dependent ($y$).                    |
|                                                                                           |
| 2.  Substitute the known value into the equation.                                         |
|                                                                                           |
| 3.  Solve to find the unknown variable.                                                   |
+-------------------------------------------------------------------------------------------+

+-----------------------------------------------------------------------+
| - **Example** Interpolate and extrapolate using the linear regression |
|   equation                                                            |
+===================================+===================================+
| The equation of a line of best fit is given by $y = 1.6x + 121$.\     |
| The independent variable $x$ is humerus length (cm).\                 |
| The dependent variable $y$ is height (cm).                            |
+-----------------------------------+-----------------------------------+
| Predict the height of a student   | Predict the humerus length of a   |
| with humerus length of 20 cm      | student with a height of 165 cm   |
|                                   |                                   |
| 1\. $Let\ x = 20$                 | 1\. Let $y = 165$                 |
|                                   |                                   |
| 2\. $y = 1.6(40) + 120$           | 2\. $165 = 1.6x + 121$            |
|                                   |                                   |
| 3\. $\ \ \ \ \ \ \  = 185\ cm$    | 3\. $165 - 121 = 1.6x$            |
|                                   |                                   |
|                                   | $\frac{165 - 121}{1.6}$ $= x$     |
|                                   |                                   |
|                                   | $27.5\ cm = x$                    |
+-----------------------------------+-----------------------------------+

| In the late 1800s, researchers noticed something odd: places with more  |

| The confounding variable was area. Larger countries have more rural     |
| land, storks are more common in rural areas, and rural families tend to |
| have more children. This is a classic example of a spurious             |
| correlation: two things that appear related but have no                 |
| cause-and-effect link.                                                  |
|                                                                         |
| Correlation does not imply causation. Two things can occur together     |
| without either one causing the other.                                   |
|                                                                         |
| A common modern example is the claim that violent video games cause     |
| aggressive behaviour. There is a correlation between playing these      |
| games and aggression, but other factors such as temperament or home     |
| environment may explain both the game choice and the behaviour. The     |
| correlation alone doesn\'t establish cause.                             |
|                                                                         |
| In the 1950s, tobacco companies used the same logic when scientists     |
| first linked smoking to lung cancer. They argued the relationship might |
| be spurious, just like storks and birth rates. This delayed public      |
| understanding of the risk for years.                                    |
|                                                                         |
| So how did scientists eventually establish that smoking does cause lung |
| cancer?                                                                 |
|                                                                         |
| They used several independent lines of evidence:                        |
|                                                                         |
| \- Large-scale studies showed smokers had far higher rates of lung      |
| cancer than non-smokers.                                                |
|                                                                         |
| \- Laboratory experiments showed cigarette smoke damages DNA and causes |
| mutations.                                                              |
|                                                                         |
| \- Lung cancer risk fell in people who quit smoking.                    |
|                                                                         |
| \- Animals exposed to cigarette smoke developed lung cancer.            |
|                                                                         |
| \- A clear *dose-response relationship* existed: more smoking meant     |
| higher risk.                                                            |
|                                                                         |
| No single piece of evidence was enough on its own. Taken together, they |
| established a causal relationship.                                      |
|                                                                         |
