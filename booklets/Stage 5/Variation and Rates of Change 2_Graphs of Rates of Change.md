# Syllabus Content

**MA5-RAT-P-02** analyses and constructs graphs relating to rates of
change

**Analyse graphs that are decreasing or increasing at a constant rate**

- Describe direct variation graphs as increasing at a constant rate and
  interpret the gradient of the line as the rate of change

- Interpret and analyse further graphs that exhibit a constant rate of
  change, including those that are decreasing at a constant rate

**Analyse the relationship between graphs and variable rates of change**

- Interpret distance--time graphs when the speed is variable, describing
  the rate of increase or decrease, the initial and final points,
  constant relationships represented by straight lines and variable
  relationships represented by curved lines

- Describe the rate of change between variables in a variety of contexts
  including direct and inverse variation

- Describe qualitatively the rate of change of a graph, using terms such
  as *increasing at a decreasing rate*

**Construct graphical representations of rates of change **

- Represent a given description of a variable rate of change of one
  quantity over time

# Speed-Time Graphs

+-------------------------------------------------------------------+
| - **Acceleration and Deceleration**                               |
+===================================================================+
| **Speed** is the rate of **change of** **distance over time**,\   |
| E.g. A car travelling on a highway at 90 km per h (km/h).         |
|                                                                   |
| **Acceleration** is the rate of **change of speed over time**,\   |
| E.g. An object falling due to gravity accelerating at 9.8 metres  |
| per second, per second (m/s²).                                    |
|                                                                   |
| **Acceleration** is the gradient of a speed-time graph.           |
|                                                                   |
| **Negative acceleration** is called **deceleration**.             |
+-------------------------------------------------------------------+

+------------------------------------------------------------------------------------------------------------------------------------------------------------+
| - **Example** Calculate acceleration and deceleration from a speed-time graph                                                                              |
+====================================================================================================================+=======================================+
| Consider the **speed-time** graph of a car.                                                                        | a.  Calculate the acceleration during |
|                                                                                                                    |     the first 8 seconds               |
| ![](media/Variation and Rates of Change 2_Graphs of Rates of Change/media/image88.png){width="2.726849300087489in" |                                       |
| height="2.623128827646544in"}                                                                                      | $$m = \frac{16}{8} = 2\ m/s^{2}$$     |
|                                                                                                                    |                                       |
|                                                                                                                    | b.  Explain why the acceleration is 0 |
|                                                                                                                    |     between 8 and 14 seconds.         |
|                                                                                                                    |                                       |
|                                                                                                                    | The gradient is 0, this means the     |
|                                                                                                                    | speed is constant, no change in speed |
|                                                                                                                    | means no acceleration.                |
|                                                                                                                    |                                       |
|                                                                                                                    | c.  What is the acceleration during   |
|                                                                                                                    |     14 to 16 seconds?                 |
|                                                                                                                    |                                       |
|                                                                                                                    | $$m = \frac{- 16}{2} = - 8\ m/s^{2}$$ |
