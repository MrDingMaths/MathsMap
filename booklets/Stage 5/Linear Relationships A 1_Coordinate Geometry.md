# Syllabus Content

**MA4-LIN-C-01** determines the midpoint, gradient and length of an
interval, and graphs linear relationships, with and without digital
tools

**Find the midpoint and gradient of a line segment (interval) on the
Cartesian plane**

- Plot and join 2 points to form an interval on the Cartesian plane and
  use the interval as the hypotenuse of a right-angled triangle

- Apply the relationship gradient $\ m = \frac{rise}{run}$ to find the
  gradient/slope of the interval joining the 2 points

- Distinguish between intervals with positive and negative gradients
  from a diagram

- Explain why horizontal intervals have a gradient of 0 and vertical
  intervals have undefined gradients using the gradient relationship

- Determine the midpoint of horizontal and vertical intervals on the
  Cartesian plane

- Apply the process for calculating the mean to find the midpoint, $M$
  of the interval joining 2 points on the Cartesian plane

- Use graphing applications to find the midpoint and gradient/slope of
  an interval  

**Find the distance between 2 points located on the Cartesian plane**

- Use the interval between 2 points as the hypotenuse of a right-angled
  triangle on the Cartesian plane and apply Pythagoras' theorem to
  determine the length of the interval joining the 2 points

- Use graphing applications to find the distance between 2 points on the
  Cartesian plane

| - **Gradient Formula**                                                                                  |
+=========================================================================================================+
| Gradient, $m$, is the ratio of **rise** to **run**.                                                     |
|                                                                                                         |
| Gradient shows **how far up or down for each unit to the right**.                                       |
|                                                                                                         |
| > ![](media/Linear Relationships A 1_Coordinate Geometry/media/image3.png){width="1.5555555555555556in" |
| > height="1.2497517497812773in"}                                                                        |
| >                                                                                                       |
| > $$m = \frac{rise}{run}                                                                                |
| > $$                                                                                                    |
|                                                                                                         |
| To calculate gradient:                                                                                  |
|                                                                                                         |
| 1.  Find the **rise**: vertical change between two points, from left to right.                          |
|                                                                                                         |
|     a.  Positive if line slopes upwards from left to right.                                             |
|                                                                                                         |
|     b.  Negative if line slopes downwards from left to right.                                           |
|                                                                                                         |
| 2.  Find the **run**: horizontal change between two points, moving from left to right.                  |
|                                                                                                         |
| 3.  Calculate gradient $m =$ $\frac{rise}{run}$.                                                        |
|                                                                                                         |
| 4.  Simplify the fraction, if you can.                                                                  |

| - **Midpoint Formula**                                                                                 |
+========================================================================================================+
| **Midpoint** is halfway between two endpoints.                                                         |
|                                                                                                        |
| - Midpoint: <https://www.geogebra.org/m/uepqnx7k>                                                      |
|                                                                                                        |
| ![](media/Linear Relationships A 1_Coordinate Geometry/media/image38.png){width="1.3307633420822398in" |
| height="1.1811023622047243in"}To find the midpoint, calculate the average, or mean, of the endpoints:  |
| $mean = \frac{a + b}{2}$                                                                               |
|                                                                                                        |
| $$M = (Mean\ of\ x,\ Mean\ of\ y)\ $$                                                                  |
|                                                                                                        |
| $= \left( \overline{x},\overline{y} \right)$                                                           |
|                                                                                                        |
| To find the midpoint of an interval:                                                                   |
|                                                                                                        |
| 1.  Find $x$ midpoint, $\overline{x}$, the mean of the $x$-coordinate endpoints.                       |
|                                                                                                        |
| 2.  Find $y$ midpoint, $\overline{y}$, the mean of the $y$-coordinate endpoints.                       |
|                                                                                                        |
| 3.  The midpoint is $M = \left( \overline{x},\overline{y} \right)$                                     |
