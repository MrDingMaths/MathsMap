# Syllabus Content

**MA5-ALG-P-02** selects and applies appropriate algebraic techniques to
operate with algebraic fractions, and expands, factorises and simplifies
algebraic expressions

**Expand, factorise and simplify algebraic expressions including special
products**

- Prove and apply these special products
  $(a - b)(a + b) = a^{2} - b^{2}$, $(a + b)^{2} = a^{2} + 2ab + b^{2}$,
  $(a - b)^{2} = a^{2} - 2ab + b^{2}$

- Expand and simplify a variety of algebraic expressions including
  binomial products and the special products

- Factorise algebraic expressions involving the special products and
  strategies, including common factors, grouping in pairs for 4-term
  expressions and quadratic trinomials (monic and non-monic)

7.  Explain why $(3 - x)^{2}$ and $(x - 3)^{2}$ is the same.

$(3 - x)^{2} = 9 - 6x + x^{2}$ and $(x - 3)^{2} = x^{2} - 6x + 9$, which
are the same.

This is because you can factorise a negative out of the inner terms,
which disappears when squared.

$$(3 - x)^{2} = \left( - (x - 3) \right)^{2} = (x - 3)^{2}$$

# Factorising Perfect Squares

+------------------------------------------------------------------------------------------------------------------------------+
| - **Identify** a perfect square trinomial                                                                                    |
+=====================+===============+=============+======================+=============+==============+======================+
| A **perfect square trinomial** is the expansion of a perfect square.                                                         |
|                                                                                                                              |
| It is of the form $a^{2} + 2ab + b^{2}$.                                                                                     |
|                                                                                                                              |
| 1.  Find the square roots of the first term $\sqrt{a^{2}} = a$ and last term $\sqrt{b^{2}} = b$.                             |
|                                                                                                                              |
| 2.  Check that the middle term is $2ab$.                                                                                     |
+---------------------+-----------------------------+----------------------+----------------------------+----------------------+
| ✖                   | ✔                           | ✔                    | ✔                          | ✖                    |
|                     |                             |                      |                            |                      |
| $$x^{2} + 6x + 16$$ | $$x^{2} + 6x + 9$$          | $$x^{2} - 10x + 25$$ | $$4x^{2} + 12x + 9$$       | $$4x^{2} + 10x + 9$$ |
+---------------------+-----------------------------+----------------------+----------------------------+----------------------+
| Decide if these are perfect square trinomials.                                                                               |
+-------------------------------------+--------------------------------------------------+-------------------------------------+
| a.  $x²\  + \ 6x\  + \ 9$           | b.  $x^{2} + 7x + 9$                             | c.  $x^{2} - 8x + 16$               |
+-------------------------------------+--------------------------------------------------+-------------------------------------+
| d.  $x^{2} - 8x + 9$                | e.  $9x^{2} - 12x + 4$                           | f.  $4x^{2} + 20x + 25$             |
+-------------------------------------+--------------------------------------------------+-------------------------------------+
| g.  $x^{2} + 10x + 16$              | h.  $x^{2} - 14x + 49$                           |                                     |
+-------------------------------------+--------------------------------------------------+-------------------------------------+

+-----------------------------------------------------------------------+
| - **Factorising Perfect Square Trinomials**                           |
+===================================+===================================+
| $${= a^{2} + 2ab + b^{2}          | $${= a^{2} - 2ab + b^{2}          |
| }{= (a + b)^{2}}$$                | }{= (a - b)^{2}}$$                |
+-----------------------------------+-----------------------------------+
| 1.  Find the square roots of the first term $\sqrt{a^{2}} = a$ and    |
|     last term $\sqrt{b^{2}} = b$.                                     |
|                                                                       |
| 2.  Check whether the middle term is $2ab$.                           |
|                                                                       |
| 3.  Write $(a + b)^{2}$ or $(a - b)^{2}$ based on the sign of the     |
|     middle term.                                                      |
+-----------------------------------------------------------------------+

+-----------------------------------------------------------------------+
| - **Example** Factorise perfect square trinomials                     |
+===================================+===================================+

|                                   |                                   |

| 2\. Check:                        | 2\. Check:                        |
| $2 \times x \times 3 = 6x$ ✓      | $2 \times 3x \times 5 = 30x$ ✓    |
|                                   |                                   |
| 3\. $= (x - 3)^{2}$               | 3\. $= (3x + 5)^{2}$              |

# Factorising by Grouping in Pairs

+-------------------------------------------------------------------+
| - **Factorise by Grouping in Pairs**                              |
+===================================================================+
| $$ac + ad + bc + bd$$                                             |
|                                                                   |
| Factorise expressions with **four terms** with no common factor   |
| using **grouping in pairs**:                                      |
|                                                                   |
| 1.  Pair the terms.                                               |
|                                                                   |
| 2.  Factorise each pair separately.                               |
|                                                                   |
| 3.  Factorise using the binomial common factor.                   |
|                                                                   |
| - You may need to take out a negative factor to get a binomial    |
|   common factor.                                                  |
+-------------------------------------------------------------------+

+-----------------------------------------------------------------------+
| - **Example** Factorise by grouping in pairs                          |
+===================================+===================================+
| $6xy + 9x + 4y + 6$               | The first step is to ............ |

|                                   |                                   |
| 2\. $3x(2y + 3) + 2(2y + 3)$      | From the first pair, factor out   |
|                                   | ...............                   |
| 3\. $(2y + 3)(3x + 2)$            |                                   |
|                                   | From the second pair, factor out  |
|                                   | ............                      |
|                                   |                                   |
|                                   | The binomial common factor is     |
|                                   | ...............                   |

# Factorising Non-Monic Quadratic Trinomials

+-------------------------------------------------------------------+
| - **Factorising Non-Monic Quadratic Trinomials**                  |
+===================================================================+
| **Non-monic** means the coefficient of $x^{2}$ is not 1.          |
|                                                                   |
| **Non-monic quadratic trinomials** are of the form                |
| $ax^{2} + bx + c$.                                                |
|                                                                   |
| To factorise a non-monic quadratic trinomial:                     |
| $ax^{2} + bx + c = 0$                                             |
|                                                                   |
| 1.  Identify the product: $ac$                                    |
|                                                                   |
| 2.  Identify the sum: $b$                                         |
|                                                                   |
| 3.  Find a factor pair, $\alpha$ and $\beta$, that has product of |
|     $ac$ and sum to $b$.                                          |
|                                                                   |
| 4.  Split $bx$ to $\alpha x + \beta x$                            |
|                                                                   |
| 5.  Factorise by grouping in pairs.                               |
+-------------------------------------------------------------------+

+-------------------------------------------------------------------+
| - **Example** Factorise non-monic quadratic trinomials            |
+===================================================================+
| $5x^{2} - 13x + 6$                                                |
+-------------------------------------------------------------------+
| 1\. P: $5 \times 6 = 30$                                          |
|                                                                   |
| 2\. S: $- 13$                                                     |
|                                                                   |
| 3\. F: $10,\  - 3$                                                |
|                                                                   |
| 4\. Split $= 5x^{2} - 10x - 3x + 6$                               |
|                                                                   |
| 5\. Factorise $= 5x(x - 2) - 3(x - 2)$                            |
|                                                                   |
| $= (x - 2)(5x - 3)$                                               |
+-------------------------------------------------------------------+

$n^{4} + 2n^{3} - n^{2} - 2n = n(n + 1)(n - 1)(n + 2);$ a product of 4
consecutive integers.

Two of the numbers are even, and one of them must be divisible by 4.

One of the numbers must be divisible by 3.

Therefore, divisible by 24 as it is divisible by 8 (from the two even
numbers, one divisible by 4) and 3.

Note: it is not enough to prove divisibility by 4 and 6, as they are not
coprime (e.g. 12 is divisible by 4 and 6 but not 24)

8.  Prove that every square number divided by 4 leaves a remainder of 1
    or 0.

Every integer is either even ($2n$) or odd ($2n + 1$).

$(2n)^{2} = 4n^{2}$, remainder 0.

$(2n + 1)^{2} = 4n^{2} + 4n + 1$, remainder 1.

9.  $N$ is a number divisible by 5. Prove that
    $(N + 1)^{2} + (N + 2)^{2}$ is also divisible by 5.

Write $N = 5L$. $(N + 1)^{2} + (N + 2)^{2}$

$$= 2N^{2} + 6N + 5$$

$$= 2(5L)^{2} + 6(5L) + 5$$

$= 5(2 \times 5L^{2} + 6L + 1)$.

Therefore $(N + 1)^{2} + (N + 2)^{2}$ is divisible by 5.

10. Prove the following algebraically.

    a.  Any even number added to another even number is an even number.

Write even numbers as $2k$ and $2m$.

$2k + 2m = 2(k + m)$, which is even.

b.  The sum of two odd numbers is an even number.

Write odd numbers as $2k + 1$ and $2m + 1$.

$(2k + 1) + (2m + 1) = 2(k + m + 1)$, which is even.

c.  The product of two odd numbers is an odd number.

$$(2k + 1)(2m + 1)$$

$$= 4km + 2k + 2m + 1$$

$= 2(2km + k + m) + 1$, which is odd.

d.  The sum of an integer and its square is even. Hint: consider
    integers either as odd or even.

Case 1: integer is odd, $2k + 1$.

$$(2k + 1) + (2k + 1)^{2}$$

$$= 4k^{2} + 6k + 2$$

$= 2(2k^{2} + 3k + 1)$, which is even.

Case 2: integer is even, $2k$.

$$2k + (2k)^{2}$$

$$= 2k + 4k^{2}$$

$= 2(k + 2k^{2})$, which is even.

11. If $p^{2} = p + 1$, find $p^{3}$ in terms of $p$.

$$p^{3} = p^{2} \times p$$

$$= (p + 1)p$$

$$= p^{2} + p$$

$$= (p + 1) + p$$

$= 2p + 1$.

12. If $x + y = 1$ and $x^{2} + y^{2} = 5$, find the value of:

    a.  $(x + y)^{2}$

$(x + y)^{2} = 1^{2} = 1$.

b.  $xy$

$(x + y)^{2} = x^{2} + y^{2} + 2xy$,

$\therefore 1 = 5 + 2xy$,

$\therefore xy = - 2$.

13. Look for a pattern with each of the following scenarios by trying a
    few numbers, come up with a hypothesis and prove your findings
    algebraically.

    a.  Difference between the squares of two odd numbers.

Let odd numbers be $2n + 1$ and $2m + 1$.
$(2n + 1)^{2} - (2m + 1)^{2} = 4(n^{2} + n - m^{2} - m)$.

The difference is always a multiple of 4.

Alternatively,
$(2n + 1)^{2} - (2m + 1)^{2} = (\text{odd} - \text{odd})(\text{odd} + \text{odd}) = \text{even} \times \text{even}$,
a multiple of 4.

b.  Difference between the squares of consecutive even numbers.

Let consecutive even numbers be $2m$ and $2m + 2$.
$(2m + 2)^{2} - (2m)^{2} = 4(2m + 1)$.

The difference is always a multiple of 4.

c.  Difference between consecutive square numbers.

Let consecutive square numbers be $m^{2}$ and $(m + 1)^{2}$.
$(m + 1)^{2} - m^{2} = 2m + 1$.

The difference is always an odd number.

14. If $k^{2} = k + 5$, write an expression in terms of $k$ and a

    a.  $k^{3}$

$$= k(k + 5)$$

$$= k^{2} + 5k$$

$= 6k + 5$.

b.  $k^{4}$

15. If $x + y = 9$ and $xy = 5$, find the value of:

    a.  $x^{2} + y^{2}$

16. Using the fact that $x^{2} - y^{2} = (x - y)(x + y)$, factorise the
    following:

    a.  $x^{4} - 16$

$$= \left( x^{2} - 4 \right)\left( x^{2} + 4 \right)$$

$$= (x - 2)(x + 2)(x^{2} + 4)$$

b.  $(x - 2y)^{2} - 9$

c.  $4 - (x - 5)^{2}$

$$= \lbrack 2 - (x - 5)\rbrack\lbrack 2 + (x - 5)\rbrack$$

$$= \lbrack 7 - x\rbrack\lbrack x - 3\rbrack$$

$$= \left( x^{2} - 1 \right)\left( x^{2} + 1 \right)\left( x^{4} + 1 \right)$$

$$= (x - 1)(x + 1)(x^{2} + 1)(x^{4} + 1)$$

17. Find the four prime factors of $2^{16} - 1$ by factorising.
