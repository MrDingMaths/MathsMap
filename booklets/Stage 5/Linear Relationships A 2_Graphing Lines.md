# Syllabus Content

**MA4-LIN-C-01** determines the midpoint, gradient and length of an
interval, and graphs linear relationships, with and without digital
tools

**Recognise and graph equations**

- Recognise that equations of the form  $y = mx + c$ represent linear
  relationships or straight lines

- Construct tables of values and use coordinates to graph a variety of
  linear relationships on the Cartesian plane, with and without digital
  tools

- Identify the $x$- and $y$- intercepts of lines

- Determine whether a point lies on a line using substitution

**Examine parallel, horizontal and vertical lines**

- Explain that parallel lines have equal gradients/slopes

- Explain why the $x$-axis has the equation $y = 0$ and the $y$-axis has
  the equation $x = 0$

- Recognise $y = c$  as a line parallel to the $x$-axis and $x = k$ as a
  line parallel to the $y$-axis

- Graph vertical and horizontal lines

+===================================================================+

+-------------------------------------------------------------------+
| - **Sketch a Line from a Table of Values**                        |
+===================================================================+
| Some equations are called **linear** **relationships** because    |
| when we plot their $x$-$y$ pairs, it forms a straight line.       |
|                                                                   |
| 1.  Construct a table of values by substituting $x$ values into   |
|     the equation to find $y$.                                     |
|                                                                   |
| 2.  Plot the $(x,y)$ points on a Cartesian plane.                 |
|                                                                   |
| 3.  Draw a straight line through the points and extend it.        |

# Horizontal and Vertical Lines

+---------------------------------------------------------------------------------------------------------------------------------------------+
| - **Horizontal and Vertical Lines**                                                                                                         |
+===================================================================================================+=========================================+
| $$y = c$$                                                                                         | $$x = k$$                               |
|                                                                                                   |                                         |
| Every $y$ coordinate is $c$.                                                                      | Every $x$ coordinate is $k$.            |
|                                                                                                   |                                         |

|                                                                   |
| 1.  Separate the **left-hand side** **LHS** and **right-hand      |
|     side** **RHS** of the equation.                               |
|                                                                   |
| 2.  Substitute $x$ and $y$ values of the point into both sides.   |
|                                                                   |
| 3.  Compare: If LHS = RHS, it lies on the line. If LHS ≠ RHS, it  |
|     does not lie on the line.                                     |
+-------------------------------------------------------------------+

+----------------------------------------------------------------------------+
| - **Example** Check whether a point lines on a line by substitution        |
+=====================================+======================================+
| Decide whether $(2,5)$ lies on the  | Decide whether $( - 2, - 5)$ lies on |
| line                                | the line                             |
|                                     |                                      |
| $$y = 2x + 1$$                      | $$y = 2x + 1$$                       |
|                                     |                                      |
| +----------------+----------------+ | +----------------+-----------------+ |
| | $LHS = y$      | $RHS = 2x + 1$ | | | $LHS = y$      | $RHS = 2x + 1$  | |
| |                |                | | |                |                 | |
| | $= 5$          | $= 2(2) + 1$   | | | $= - 5$        | $= 2( - 2) + 1$ | |
| |                |                | | |                |                 | |
| |                | $= 5$          | | |                | $= - 3$         | |
| +================+================+ | +================+=================+ |
| | $$LHS = RHS$$                   | | | $$LHS \neq RHS$$                 | |
| |                                 | | |                                  | |
| | $(2,5)$ lies on the line        | | | $( - 2,5)$ does not lie on the   | |
| | $y = 2x + 1$                    | | | line $y = 2x + 1$                | |
| +---------------------------------+ | +----------------------------------+ |

8.  A straight line has equation $y = 5x + 2a + 3$ . The line passes
    through the point $(0,19).$\
    Work out the value of $a$.

Substitute into equation

$$19 = 5(0) + 2a + 3$$

$$2a = 16$$

$$a = 8$$

9.  A line has gradient 3 and passes through the point $(1,7)$. Find its
    equation $y = mx + c$.

$$y = 3x + c$$

$$7 = 3(1) + c$$

$$c = 4$$

$$\therefore y = 3x + 4$$
