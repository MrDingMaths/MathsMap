**Budgeting**

- A budget plans your money coming in and going out.

- It helps you save for future purchases and avoid overspending.

- Income is money you receive, while expenses are what you spend.

- At the end of each period (weekly, monthly, or yearly), you either:

  - Build your savings

  - Use up savings/go into debt.

$$Budget\ Balance\  = \ Income\ –\ Expenses$$

$Saving:\ Income > Expenses$ $Dissaving:\ Expenses > Income$

- Budgets are made for a specific period of time (week, fortnight, month
  year).

- Read the questions carefully and make sure the amounts are all in the
  same time period.

> **Example** Calculate a budget balance.

+---------------------------------------------------------------------+
| Isabella has just started work and lives at home with her parents.\ |
| Her weekly take-home pay is \$480.\                                 |
| Each week she pays \$49 for transport fares and \$35 for lunches.\  |
| She spends \$120 per fortnight on entertainment and \$330 per month |
| on clothes.                                                         |
|                                                                     |
| Determine the annual budget balance and state whether she is saving |
| or dissaving.                                                       |
|                                                                     |
| +--------------------------------+--------------------------------+ |
| | **Income**                     | **Expenses**                   | |
| +================================+================================+ |
| | Pay:                           | Fares:                         | |
| | $\$ 480 \times 52 = \$ 24960$  | $\$ 49 \times 52 = \$ 2548$    | |
| |                                |                                | |
| |                                | Lunches:                       | |
| |                                | $\$ 35 \times 52 = \$ 1820$    | |
| |                                |                                | |
| |                                | Entertainment:                 | |
| |                                | $\$ 120 \times 26 = \$ 3120$   | |
| |                                |                                | |
| |                                | Clothes:                       | |
| |                                | $\$ 330 \times 12 = \$ 3960$   | |
| +--------------------------------+--------------------------------+ |
| | Total: \$24 960                | Total: \$11 448                | |
| +--------------------------------+--------------------------------+ |
|                                                                     |
| Budget balance $= \$ 24960 - \$ 11448 = \$ 13512$ savings.          |
