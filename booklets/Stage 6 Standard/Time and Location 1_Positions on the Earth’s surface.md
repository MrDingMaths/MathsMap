  differences between locations on Earth given the relevant longitudes

# Latitude and Longitude

- Imagine wrapping a Cartesian Plane around the surface of the Earth.

- In this way, it is possible to describe the location of any point on
  the surface of Earth with absolute accuracy.

<!-- -->

- Coordinates are always $(Latitude,\ Longitude)$.

+---------------------------------------------------------------------------------------------------------------+---------------------------------------------------------------------------------------------------------------+
| **Latitude**                                                                                                  | **Longitude**                                                                                                 |
+===============================================================================================================+===============================================================================================================+
| ![](media/Time and Location 1_Positions on the Earth’s surface/media/image1.png){width="3.4322637795275592in" | ![](media/Time and Location 1_Positions on the Earth’s surface/media/image1.png){width="3.4319444444444445in" |
| height="1.4722222222222223in"}                                                                                | height="1.4722222222222223in"}                                                                                |
|                                                                                                               |                                                                                                               |
| - **Latitude** is the north-south coordinate.                                                                 | - **Longitude** is the east-west coordinate.                                                                  |
|                                                                                                               |                                                                                                               |
| - It consists of horizontal slices parallel to the **Equator**.                                               | - It forms vertical slices from the North Pole to the South Pole.                                             |
|                                                                                                               |                                                                                                               |
| - The equator has a latitude of $0{^\circ}$                                                                   | - The **Prime Meridian** is a slice through Greenwich, England.                                               |
|                                                                                                               |                                                                                                               |
| - **Latitude** measures north or south from the Equator.                                                      | - It has a longitude of $0{^\circ}$                                                                           |
|                                                                                                               |                                                                                                               |
|                                                                                                               | - **Longitude** measures east or west from the Prime Meridian.                                                |
+---------------------------------------------------------------------------------------------------------------+---------------------------------------------------------------------------------------------------------------+

- Latitude and longitude are **angles** because of Earth's spherical
  shape.

- "Straight lines" on the surface of Earth are arcs of circles, and
  angles are the natural way to measure arc-length.
