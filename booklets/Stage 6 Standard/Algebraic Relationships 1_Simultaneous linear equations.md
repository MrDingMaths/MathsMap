# Overview

## Syllabus Content

**MST-12-S2-01** represents the relationships between quantities
algebraically and graphically to solve problems and make predictions in
practical contexts

**Simultaneous linear equations**

- Graph two linear equations and identify the point of intersection,
  with and without using graphing applications

- Solve a pair of simultaneous linear equations using graphical and
  algebraic methods

- Develop a pair of simultaneous linear equations to model a practical
  situation

- Solve practical problems that involve simultaneous linear equations

- Use simultaneous equations to model and analyse the break-even point
  where cost and revenue are represented by linear equations

- Identify the break-even point and solve problems involving profit and
  loss using a spreadsheet

# Review of Linear Relationships

- A **linear** equation is in the form $y = mx + c$.

- We can show each $\mathbf{x}$ $\mathbf{y}$ **pair** from a table of
  values as **coordinates**.

1.  Construct a table of values by substituting $x$ values into the
    equation to find $y$.

2.  Plot the $(x,y)$ points on a Cartesian plane.

3.  Draw a straight line through the points and extend it.

## Sketching Graphs from the Equation

1.  Plot the $y$-intercept.

2.  Starting from the $y$-intercept, use the gradient to find another
    point.

    a.  The denominator is how much to move right (1 if the gradient is
        a whole number).

    b.  The numerator is how much to move up or down.

3.  Draw a straight line through both points and extend it.

## Finding the Equation from the Graph

1.  Identify the $y$-intercept, this is $c$.

2.  Calculate the gradient, this is $m$.

    a.  Choose two points on the line and create a right-angled
        triangle.

    b.  Going from left to right, determine the rise and the run.

    c.  Calculate gradient using the formula $m = \frac{Rise}{Run}$

3.  Write the equation $y = mx + c$.

# Linear Modelling

- Linear relationships model many real-life scenarios.

  - The $y$-axis is the dependent variable. It changes based on another
    variable.

  - The $x$-axis is the independent variable. It causes the change.

  - The **gradient is the rate;** it represents the **change in**
    $\mathbf{y}$ **per unit increase in** $\mathbf{x}$**.**

  - The $\mathbf{y}$**-intercept** is the **initial** or **fixed value
    at** $\mathbf{x = 0}$**.**

# The Substitution Method

- The goal is to construct an equation that has only one pronumeral,
  solve it, then use the value of the first pronumeral to find the
  other.

1.  Rearrange to make $x$ or $y$ the subject in one equation; avoid
    working with fractions.

2.  Substitute into the other equation.

3.  Solve.

4.  Substitute the solution into the other equation to solve for the
    other pronumeral.

| ![](media/Algebraic Relationships 1_Simultaneous linear equations/media/image80.png)Issy's |
| Muffin Shop has a fixed cost of \$320 per day and a variable production cost of \$2 per    |
| muffin. The selling price of each muffin is \$6.                                           |
|                                                                                            |
| Write an equation for the cost, $C$, of producing $n$ muffns.                              |
|                                                                                            |
| $C = 320 + 2n$                                                                             |
|                                                                                            |
| Write an equation for the revenue, $R$,\                                                   |
| for the number of muffins sold, $n$.                                                       |
|                                                                                            |
| $R = 6n$                                                                                   |
|                                                                                            |
| Determine the break-even point\                                                            |
| algebraically by letting $R = C$                                                           |
|                                                                                            |
| +-----------------+---------------------+                                                  |
| | $R = C$         | Substitute $n = 80$ |                                                  |
| |                 | into $R = 6n$       |                                                  |
| | $6n = 320 + 2n$ |                     |                                                  |
| |                 | $R = 6(80)$         |                                                  |
| | $4n = 320$      |                     |                                                  |
| |                 | $= 480$             |                                                  |
| | $n = 80$        |                     |                                                  |
| +=================+=====================+                                                  |
|                                                                                            |
| Break-even at 80 muffins, with revenue of \$480.                                           |
|                                                                                            |
| When 120 muffins are sold, is it profit or loss and by how much?                           |
|                                                                                            |
| When $n = 120:$ $R = 6(120) = 720$                                                         |
|                                                                                            |
| $C = 320 + 2(120) = 560$                                                                   |
|                                                                                            |
| $R - C = 720 - 560 = \$ 160$ profit.                                                       |
+============================================================================================+

Year 11 is making pizzas to raise money for a charity.\

represented by:\
$C = 2.5x + 6$ $R = 8x$

> Enough pizzas are sold so that a profit is made.
>
> By how much does the profit increase for each additional pizza sold?

Profit = R -- C

$$P = 8x - (2.5x + 6) = 5.5x + 6$$
