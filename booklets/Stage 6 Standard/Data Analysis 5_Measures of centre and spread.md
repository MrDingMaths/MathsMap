# Range

- The **maximum** is the largest data value.

- The **minimum** is the smallest data value

- The **range** is the difference between the maximum and minimum.

- The range shows how spread out the data values are.

- We can use **multiplication** to quickly add many identical values.

# Median from a Frequency Table

- **Cumulative frequency** is the total count of data values up to and
  including a value.

- To find the median from a frequency table:

1.  Create and fill in the
    $\mathbf{Cumulative\ Frequency\ (}\mathbf{cf)}$ column.

    a.  For the first data value, cumulative frequency equals its
        frequency.

    b.  For other values, add their frequency to the previous cumulative
        frequency.

2.  Find the median position using $Pos =$ $\frac{n + 1}{2}$

3.  The median is the value where cumulative frequency first equals or
    exceeds the median position.

### **Example** Calculate median from a frequency table.

+-------------------------------------------------+---------------------------------+
|   --------------------------------------------- | What is cumulative frequency?   |
|   **Data value**  **Frequency**   **Cumulative  |                                 |

|         2               3              3        | ............... of all the      |
|                                                 | scores up to and                |
|         3               5              8        | ..................... a         |
|                                                 | particular score.*              |
|         4               3              11       |                                 |
|                                                 | What are two ways to find the   |
|         5               4              15       | total number of data values?    |
|                                                 |                                 |
|         6               8              23       | *We can either ......... the    |
|                                                 | frequency column or look at the |
|         7               2              25       | final ..................        |

# Mean from Grouped Data

- When grouping data, individual data points are lost. Therefore, any
  calculation of mean and median will be an estimate.

- Estimation of mean is the same as calculation from ungrouped data in
  tables, except we use the class centre.

1.  Find the class centres.

2.  Multiply the class centres and the frequencies $cc \times f$

3.  Estimate the mean by dividing the sum of $ccf$ by the total
    frequency $\overline{x} = \frac{\Sigma ccf}{\Sigma f}$

    frequency.
