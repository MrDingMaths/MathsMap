# Clusters, Gaps, Outliers

- A **cluster** is data **grouped** around specific values.

- An **outlier** is a data value **much higher or lower** than other
  data.

## Impact of Outliers

- **Outliers** in a dataset **strongly affect the mean** but have **less
  impact on the median**.

  **Data**        **Mean**   **Median**
  -------------- ---------- ------------
  8 9 9 10 11       9.4          9

  8 9 9 10 11       24.5        9.5
  **100**                   
  --------------------------------------

# Formal Definition of Outlier

- Data values more than 1.5 times the IQR below Q~1~ and above Q~3~ is
  defined as an outlier.

![](media/Data Analysis 8_Clusters and outliers/media/image9.png){width="4.674305555555556in"
height="1.2680555555555555in"}

### **Example** Identify outliers using the formal definition.

+-------------------------------------------------------------------------------------------------+
| Identify any outliers in this sample of scores. Justify your answer.                            |
|                                                                                                 |
| ![](media/Data Analysis 8_Clusters and outliers/media/image10.png){width="3.0388888888888888in" |
| height="0.5118055555555555in"}30, 50, 50, 52, 55, 55, 57, 57, 58, 60, 60                        |
|                                                                                                 |
| $IQR = Q_{3} - Q_{1}$                                                                           |
|                                                                                                 |
| $= 58 - 50 = 8$                                                                                 |
|                                                                                                 |
| +--------------------------------+--------------------------------+                             |
| | Lower limit                    | Upper limit                    |                             |
| | $= Q_{1} - 1.5 \times IQR$     | $= Q_{3} + 1.5 \times IQR$     |                             |
| |                                |                                |                             |
| | $= 50 - 1.5 \times 8$          | $= 58 + 1.5 \times 8$          |                             |
| |                                |                                |                             |
| | $= 38$                         | $= 70$                         |                             |
| +================================+================================+                             |
|                                                                                                 |
| 30 is outside the range of 38 to 70, so it is an outlier.                                       |
+=================================================================================================+
