involving interest, depreciation and borrowing money

**Investment**

- Calculate simple interest for different rates and time periods using
  the formula $I = Prn\$where $I$ = simple interest, $P$ =
  principal, $r$ = interest rate per time period and $n$ = number of
  time periods

# Compound Interest Formula

- **Simple interest** is a percentage of the principal only.

- **Compound interest** is a percentage of the total, including previous
  interest.

### **Investigation** Establishing the compound interest formula.

+--------------------------------------------------------------------+
| \$1000 is invested at a **compound** interest rate of 10% p.a.     |
| Complete this table showing the interest paid and the balance      |
| including interest at the end of each year.                        |
|                                                                    |
|   ---------------------------------------------------------------- |
|    **Years**    **Starting    **Interest paid that     **Ending    |
|                 Balance**            year**           Balance**    |
|   ----------- -------------- ---------------------- -------------- |
|        1       $$\$ 1000$$    $$1000 \times 10\%$$   $$\$ 1100$$   |
|                                                                    |
|        2       $$\$ 1100$$    $$1100 \times 10\%$$                 |
|                                                                    |
|        3                                                           |
|                                                                    |
|        4                                                           |
|                                                                    |
|        5                                                           |
|   ---------------------------------------------------------------- |
|                                                                    |
| Every year, the amount is increased by 10%, which means            |
| multiplying by 1.1.                                                |
|                                                                    |
| After 1 year, $A = \$ 1000 \times 1.1$                             |
|                                                                    |
| After 2 years, $A = \$ 1000 \times 1.1 \times 1.1$                 |
|                                                                    |
| After 3 years, $A = \$ 1000 \times$ .... $\times$ .... $\times$    |
| ....                                                               |
|                                                                    |
| If you were to write the closing balance after 6 years, it would   |
| be:                                                                |
|                                                                    |
| $A = \$ 1000 \times$ ...... $\times$ ...... $\times$ ......        |
| $\times$ ...... $\times$ ...... $\times$ ......                    |
|                                                                    |
| $= \$ 1000 \times (\ \ \ \ \ \ \ \ )^{\ldots}$                     |
|                                                                    |
| If you were to write the closing balance after $n$ years, it would |
| be:                                                                |
|                                                                    |
| $A = \$ 1000 \times (\ \ \ \ \ \ \ \ )^{\ldots}$                   |
|                                                                    |
| For any initial amount $P$, and interest rate $r$, the total       |
| amount after $n$ years would be:                                   |
|                                                                    |
| $A =$ ......... $\times$ $(1 + \ \ \ \ \ )^{\ldots}$               |
+====================================================================+

- In compound interest:

  - The initial amount (**principal**) is called the **present value**
    (PV).

  - The **final** **amount** after growth is the **future value** (FV).

  ---------------------------------------------------------------------------------------------
  **Compound Interest Formula**

# Compounding Frequency

- **Compounding frequency** affects interest earned.

- **The compounding frequency, interest rate, and number of periods must
  match.**

  - Multiply the period.

  - Divide the rate.

### **Example** Calculate future value with non-annual compounding periods.

+--------------------------------------------------------------+--------------------------------------------------------------+
| $PV$ $n$ $r = 0.05$                                          | $PV$ $n = 24$ $r =$ $\frac{0.05}{12}$                        |
|                                                              |                                                              |
| \$3000 is invested for 2 years at 5% p.a. compounded         | \$3000 is invested for 2 years at 5% p.a. compounded         |
| **annually**.                                                | **monthly**.                                                 |
|                                                              |                                                              |
| Calculate the amount after 2 years.                          | Calculate the amount after 2 years.                          |
|                                                              |                                                              |
| $$FV = PV(1 + r)^{n}$$                                       | $FV = PV(1 + r)^{n}$                                         |
|                                                              |                                                              |
| $= \$ 3000(1 + 0.05)^{2}$                                    | $= \$ 3000$ $\left( 1 + \frac{0.05}{12} \right)^{24}$        |
|                                                              |                                                              |
| $= \$ 3307.50$                                               | $= \$ 3314.82$                                               |
+==============================================================+==============================================================+
| When compounding monthly, why did we multiply the periods by 12?                                                            |
|                                                                                                                             |
| *The period was given as 2 ..............., there are 12 months in a year, so we multiply by 12.*                           |
|                                                                                                                             |

# Buying Shares

- A **share** is part ownership of a company.

- The more shares you own, the more of the company you own.

- Companies sell shares to raise money for growth. The price that
  companies sell a share for is called **face value**.

- When shareholders trade shares between themselves, this is the
  **market value** (or price).

- You must pay **brokerage fees** when buying or selling shares through
  a stockbroker.

- **Brokerage fees** can be a percentage, or a flat dollar amount.
