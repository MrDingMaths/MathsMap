# Overview

## Syllabus Content

**MST-11-08** displays and analyses datasets using summary statistics
and graphical representations

**Five-number summary and box plots**

- Represent numerical datasets using a box plot to display a five-number
  summary, with and without using digital tools

- Compare and contrast the measures of centre, spread and shape using
  parallel box plots

- Determine quartiles from datasets displayed in histograms and dot
  plots, and represent these datasets as a box plot

- Interpret box plots to draw conclusions and make inferences about a
  dataset
