# Index Notation

+-------------------------------------------------------------------+
| - **Index Notation**                                              |
+===================================================================+
| - **Powers** show repeated multiplication.                        |
|                                                                   |
| - $8^{3}$ is read: '8 to the power of 3'                          |
|                                                                   |
| - We call **8 the** ***base*** and **3 the *power,* or *index.*** |
|                                                                   |
| - Any number without a power has a power of 1.                    |
|                                                                   |
| - ![Diagram illustrating exponentiation, showing 8 raised to the  |
|   power of 3 equals 8 multiplied by itself three times. Key       |
|   components include blue labels for base and red labels for      |
|   index and multiplication count, emphasizing the concept of      |
|   repeated                                                        |
|   multiplication.](media/image2.png){width="2.029166666666667in"  |
|   height="1.3284722222222223in"}"**index**", "**power**",         |
|   "**exponent**" all mean the same thing.                         |
+-------------------------------------------------------------------+

+----------------------------------------------------------------------------------------+--------------------------------------------------------------+

+----------------------------------------------------------------------------------------+--------------------------------------------------------------+

+-----------------------------------------------------------------------------------------------------------------+
| - **Rewrite** Index form in expanded form                                                                       |
+====================================+============================================================================+
| Write these numbers in expanded form.                                                                           |
+------------------------------------+----------------------------------------------------------------------------+

|                                    |                                                                            |

+------------------------------------+----------------------------------------------------------------------------+
| e.  $2^{3} \times 14^{5}$          | f.  $6^{5} \times \left( \frac{2}{3} \right)^{3}$                          |
+------------------------------------+----------------------------------------------------------------------------+

# Prime Factorisation

| - **Fundamental Theorem of Arithmetic**                           |
+===================================================================+
| We can represent **any integer greater than 1** as a **unique**   |
| **product** of **prime factors**.                                 |
|                                                                   |
| Prime factorisation breaks a number down into its prime factors.  |
|                                                                   |

| - **Key Ideas**                                                   |
+===================================================================+
| 1.  The fundamental theorem of arithmetic says that any integer   |
|     greater than 1 can be written as a product of                 |
|     .................. ..................                         |
|                                                                   |
| 2.  There is only ......... way to write each number as a product |
|     of prime factors, so we call this unique.                     |
+-------------------------------------------------------------------+

+-------------------------------------------------------------------+
| - **Factor Trees**                                                |
+===================================================================+
| - Factor Tree: <https://www.geogebra.org/m/qg9uepm9>              |
|                                                                   |
| <!-- -->                                                          |
|                                                                   |
| - A factor tree is useful for finding the prime factorisation of  |
|   large numbers.                                                  |
|                                                                   |
| 1.  Think of a factor pair and draw two branches with the two     |
|     factors at the ends.                                          |
|                                                                   |
| 2.  If the factor is a prime number, then end or **terminate**    |
|     the branch.\                                                  |
|     Move on to the other factor and repeat step 1.                |
|                                                                   |
| 3.  Once you have found all the prime factors, write them as a    |
|     product from smallest to largest.                             |
|                                                                   |
| 4.  Use index form to represent multiple prime factors.           |
+-------------------------------------------------------------------+

|                                                                   |
