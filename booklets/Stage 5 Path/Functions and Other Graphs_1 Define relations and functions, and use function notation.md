# Overview

## Syllabus Content

**MA5-FNC-P-01** uses function notation to describe and graph functions
of one variable and graphs inequalities in one and 2 variables

**Define relations and functions, and use function notation**

- Define a relation over 2 sets as an association between the elements
  of one set to the elements of another set

- Notate relations as a set of ordered pairs $(x,y)$ of real numbers

- Define a function as a relation over 2 sets where each element of the
  first set is associated with exactly one element in the second set

- Apply the vertical line test on a graph to decide whether it
  represents a function

- Use the notation $f(x)$ when expressing a function

- Use the notation $f(c)$ to determine the value of $f(x)$ when $x = c$

# Function Notation

- Another way to understand functions is to think of it as a "machine."

- A "function machine" takes an input number, applies a rule, then gives
  out an output number.

- $x$ is the **input**, or **independent variable,** $y$ is the
  **output,** or **dependent variable**.

- ![](media/Functions and Other Graphs_1 Define relations and functions, and use function notation/media/image14.png){width="2.6470964566929136in"
  height="0.4990879265091864in"}An example is the linear function
  $y = 2x + 3$.

- The vertical line test determines whether a graph represents a
  function.

<!-- -->

- If all possible vertical lines cut the graph at only one point, it
  passes the test and is a function.\
  This shows that for every value of $x,$ there is only 1 value of $y$.

- A closed circle ● means the point is on the graph.

- An open circle ○ means it is not.
