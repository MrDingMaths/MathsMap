# Overview

## Syllabus Content

**MA5-IND-P-02** describes and performs operations with surds and
fractional indices

**Describe and use fractional indices**

- Apply index laws to describe fractional indices as:
  $a^{\frac{1}{n}} = \sqrt[n]{a}\$
  and${\ a}^{\frac{m}{n}} = \sqrt[n]{a^{m}} = \left( \sqrt[n]{a} \right)^{m}$

- Translate expressions in surd form to expressions in index form and
  vice versa

- Evaluate numerical expressions involving fractional indices including
  using digital tools
