# Overview

## Syllabus Content

**MA5-FNC-P-01** uses function notation to describe and graph functions
of one variable and graphs inequalities in one and 2 variables

**Find the domain and range of a function and graph functions**

- Define the domain as the set of all allowable values of $x$

- Define the range as the set of possible $y$-values as $x$ varies over
  the domain of the function

- Determine and describe the domain and range for a variety of functions

- Use graphing applications to graph and compare functions of the form
  $y = f(x),\ \ y = f(x) + c$, $y = f(x - b)$, $y = kf(x)$ and
  $y = f(ax)$, and describe their transformations

# Natural Domain

- A function\'s **natural domain** includes all possible $x$-values that
  produce valid $y$-values.\
  When finding the natural domain, consider that

  - **Zero denominators** **are undefined**.

  - **Numbers in square roots must be non-negative**.

    $y = (x - 1)(x + 1)^{2}$ by $180{^\circ}$?\
    \
    $y = \left( x^{2} - 1 \right)(x + 1)$
    $y = \left( x^{2} - 1 \right)(x - 1)$
    $y = \left( x^{2} - 1 \right)(1 - x)$

Let $f(x) = (x - 1)(x + 1)^{2}$

$f( - x) = ( - x - 1)( - x + 1)^{2}$

$- f( - x) = - ( - x - 1)(1 - x)^{2}$

> $= (x + 1)(1 - x)^{2}$
>
> $= (x + 1)(1 - x)(1 - x)$
>
> $= \left( 1 - x^{2} \right)(1 - x)$
>
> $= - \left( x^{2} - 1 \right) \times - (x - 1)$
