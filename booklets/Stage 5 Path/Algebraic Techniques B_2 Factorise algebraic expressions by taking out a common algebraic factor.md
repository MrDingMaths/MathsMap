# Factorising a Negative HCF

- To **factorise** with a **negative HCF**:

1.  Identify the negative **highest common factor** (HCF) of the terms.
    Write it outside brackets.

2.  Divide each term by the HCF. Put the results inside the brackets.

### **Example** Factorise by finding a finding a negative HCF.

+-----------------------------------------+---------------------------------------------+
| $\mathbf{\ \ \ \ \ \ \ \ \ } - 8x - 16$ | Why is the inside of the bracket $(x + 2)$? |
|                                         |                                             |
| 1\. $- 8(\ \ \ \ \ \ \ \ \ \ \ )$       | *We divided* $- 8x$ *and* $- 16$ *by        |
|                                         | ............*                               |
| 2\. $- 8(x + 2)$                        |                                             |
+=========================================+=============================================+
| $\ \ \ \ \ \ \ \ \ 12x - 4xy$           | Why is the inside of the bracket            |
|                                         | $( - 3 + y)$?                               |
| 1\.                                     |                                             |
| $- 4x(\ \ \ \ \ \ \ \ \ \ \ \ \ \ )$    | *We divided* $12x$ *and* $- 4xy$ *by        |
|                                         | ............*                               |
| 2\. $- 4x( - 3 + y)$                    |                                             |
+-----------------------------------------+---------------------------------------------+

Factorise these expressions by finding a negative HCF.

+---------------------------------------+---------------------------------------+-------------------------+
| a.  $- 8x - 4$                        | b.  $- 2m + 4n$                       | c.  $- 3xy + 18x$       |
|                                       |                                       |                         |
| $= - 4(\ \ \ \ \ \ \ \ \ \ \ \ \ \ )$ | $= - 2(\ \ \ \ \ \ \ \ \ \ \ \ \ \ )$ | $$- 3x(y - 6)$$         |
|                                       |                                       |                         |
| $$- 4(2x + 1)$$                       | $$- 2(m - 2n)$$                       |                         |
+=======================================+=======================================+=========================+
| d.  $8x^{2} - 12x$                    | e.  $6x - 20x^{2}$                    | f.  $5a^{3}x + 10a^{2}$ |
|                                       |                                       |                         |
| $$- 4x( - 2x + 3)$$                   | $$- 2x( - 3 + 10x)$$                  | $$- 5a^{2}( - ax - 2)$$ |
+---------------------------------------+---------------------------------------+-------------------------+

**Note:** You should factor out a negative HCF if both terms are
negative, or if the question asks for it.

This skill is used in a later topic (factorising by grouping in pairs).
