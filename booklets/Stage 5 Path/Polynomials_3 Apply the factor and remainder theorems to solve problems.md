# Factor Theorem

+------------------------------------+
| **Factor Theorem**                 |
+:==================================:+
| $(x - a)$ is a factor of $P(x)$ if |
| $P(a) = 0$.                        |
|                                    |
| $a$ is called a **zero** of the    |
| polynomial.                        |
|                                    |
| In general, $(ax - b)$ is a factor |
| of $P(x)$ if                       |
| $P\left( \frac{b}{a} \right) = 0$  |
+------------------------------------+

  -------------------------------------
   **Fundamental Theorem of Algebra**
  -------------------------------------
    A polynomial of degree $n$ has at
            most $n$ factors.

  -------------------------------------

### **Investigation** Factor Theorem.

| *If* $P(a) = 0$*, that means there is ......... remainder, hence* |

+===================================================================+

## Solving Polynomials

- To solve a polynomial, we can factorise it and use the null factor
  law.

1.  Use integer root theorem to find an integer zero $x = a$ of $P(x).$

2.  Then use polynomial long division to factor $P(x)$ in the form
    $P(x) = (x - a)Q(x)$

3.  Factorise $Q(x)$.

4.  Solve for $x$ using null factor law.

No real solution for $x^{2} = - 1$, so only real solution is $x = 0$

3

Step 2: Compare constant terms to find $c$:

The constant term on the LHS is ......

10

The constant term on the RHS would be ............

$$c$$

Therefore, $c =$ ......

10

Step 3: Rewrite with values of $a$ and $c$:

$3x^{3} - 8x^{2} - x + 10 = (x + 1)\left( 3x^{2} + bx + 10 \right)$

Step 4: Equate other coefficients to find $b$:

The coefficient of $x$ on the LHS is ......

$$- 1$$

The coefficient of $x$ on the RHS would be .....................

$$b + 10$$

Therefore, $b =$ ......

$$- 11$$

23. A ladder of length 10 metres is leaning against a wall so that it is
    just touching a cube of edge length one metre.

    a.  ![](media/Polynomials_3 Apply the factor and remainder theorems to solve problems/media/image4.png){width="0.7658978565179353in"
        height="2.5092475940507435in"}Let $x + 1$ be the height up the
        wall that the ladder reaches and $y + 1$ be the distance along
        the ground. Show that this scenario can be modelled using

$$x^{4} + 2x^{3} - 98x^{2} + 2x + 1 = 0$$

> Hint: Use Pythagoras and similar triangles.

Using Pythagoras' theorem:

$$(x + 1)^{2} + (1 + y)^{2} = 10^{2}$$

Using similar triangles, $\frac{y}{1}$ $=$ $\frac{1}{x}$

$$\therefore y = \frac{1}{x}$$

$$(x + 1)^{2} + \left( 1 + \frac{1}{x} \right)^{2} = 10^{2}$$

Expand, multiply every term by $x^{2}$, then rearrange.

$$x^{4} + 2x^{3} - 98x^{2} + 2x + 1 = 0$$

b.  Use an DESMOS to solve $x^{4} + 2x^{3} - 98x^{2} + 2x + 1 = 0\$ and
    hence find the height up the wall the ladder reaches to 2 decimal
    places.

$$x \approx 0.11\ m\ or\ 8.94\ m$$

So, height up the wall is 1.11 m or 9.94 m
