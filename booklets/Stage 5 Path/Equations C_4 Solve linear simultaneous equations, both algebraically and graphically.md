# Overview

## Syllabus Content

**MA5-EQU-P-02** solves linear equations of more than 3 steps, monic and
non-monic quadratic equations, and linear simultaneous equations

**Solve linear simultaneous equations, both algebraically and
graphically**

- Solve linear simultaneous equations by finding the point of
  intersection of their graphs

- Solve linear simultaneous equations using algebraic techniques
  including substitution and elimination methods

- Model and solve word problems using simultaneous equations and
  interpret their solutions

- Describe an identity as an equation that is true for all values of the
  pronumeral and relate the identity to coincident lines

- Describe a contradiction as an equation that has no solutions and
  relate the contradiction to parallel lines

**Identify Simultaneous Solution**

- Solving equations simultaneously means finding the $x$ and $y$ values
  that make two equations true at the same time.

- The $x,\ y$ coordinates where two lines meet solves both equations.

- To find the **simultaneous solution**:

1.  Identify point of intersection.

2.  State the $x$ and $y$ values at the point of intersection.

# The Substitution Method

- The goal is to construct an equation that has only one pronumeral,
  solve it, then use the value of the first pronumeral to find the
  other.

1.  Rearrange to make $x$ or $y$ the subject in one equation; avoid
    working with fractions.

2.  Substitute into the other equation.

3.  Solve.

4.  Substitute the solution into the other equation to solve for the
    other pronumeral.

# The Elimination Method (With Adjustment)

1.  Multiply one or both equations by a factor so that the coefficients
    of a variable are the same.

2.  Add or subtract the equations to eliminate one of the variables.

3.  Solve for the remaining variable.

4.  Substitute the value of the solved variable into one of the original
    equations to find the value of the other variable.

# Identities and Contradictions

- When asked to solve an equation, we usually work with **conditional**
  equations.

- A conditional equation is true for specific values:\

# Equating Coefficients

- **Equating coefficients** is a problem-solving strategy.**\**
  Reminder: a coefficient is a number that multiplies a pronumeral.

1.  Expand out any brackets.

2.  Identify and set equal the coefficients of matching variables on
    both sides of an equation.

3.  Find the value of the unknowns; you may need to use simultaneous
    equations.

### **Example** Find an unknown by equating coefficients.

| **Worked Example**                                       | **Your Turn**                   |

|                                                          |                                 |
| Find $p$ and $q$.                                        | Find $c$.                       |

|                                                          |                                 |

|                                                          |                                 |

|                                                          |                                 |

|                                                          |                                 |

| $p = 3,y = 5$\                                           |                                 |

2.  Find the missing values by equating coefficients.

    a.  $(x + 4)(x - 6) + ax + b = x^{2} + 8x - 25$

$$a = 10,\ b = - 1$$

b.  $(2x - 5)(2x - 7) + ax^{2} + b = 3x^{2} - 24x + 12$

$$a = - 1,b = - 23$$

c.  $(x - 2)^{2} + x + a(x + b) = x^{2} + 1$

$$a = 3,b = - 1$$

d.  $(x + b)^{2} = x^{2} + cx + 49$

$$if\ b = 7,\ c = 14\ |\ if\ b = - 7,\ c = - 14$$

e.  $(x + 2c)(x + c) = x^{2} + dx + 50$

$$if\ c = 5,\ d = 15\ |\ if\ c = - 5,\ d = - 15$$

Development

3.  A parabola has the equation $y = x^{2} + 2x + 3$.\
    Find the values of $a$ and $c$ in the equation $y = (x - a)^{2} + c$
    and hence determine the coordinates of the vertex.

$$( - 1,\ 2)$$

4.  A rectangular prism has width $y$ cm, length $(x + a)$ cm and height
    $(x + b)$ cm,\
    where *a* and *b* are integers and $a > b$.

    a.  Show that the surface area of the rectangular prism is given by
        the expression\
        $2y(2x + a + b) + 2(x + a)(x + b)$ cm^2^

    b.  If the surface area of the rectangular prism is equivalent to
        the expression $2y(2x + 1) + 2x^{2} + 2x - 12$ cm^2^, find $a$
        and $b$.

$$a = 3,b = - 2$$

5.  Find the value of $m$.

$$\ \ \ \ \ \ \ \ \ \ \ (x + 3)(m - 2) = - \frac{x}{3} - 1$$

$$m = \frac{5}{3}$$

Mastery

6.  Write $\frac{x + 3}{x + 5}$ in the form $a +$ $\frac{b}{x + 5}$.

$$1 - \frac{2}{x + 5}$$

7.  Write $3x^{2} + 18x - 5$ in the form $\frac{(x - a)^{2} + c}{b}$.

$$\frac{(x + 3)^{2} - \frac{32}{3}}{\left( \frac{1}{3} \right)}$$

8.  Write $x^{2} + 4x + y^{2} - 2y$ in the form
    $(x + a)^{2} + (y + b)^{2} + c$.

$$(x - 2)^{2} + (y + 1)^{2} - 5$$

9.  $5x^{2} + px - 8 = q(x - 1)^{2} + r$. Find $p,q,r$.

$$p = - 10,\ q = 5,\ r = - 13$$

10. Find the missing values by equating coefficients.

    a.  $4x^{3} + (x + a)(x + b) + cx = ax^{3} - 3x^{3} + x^{2} - x - 42$

$$a = 7,\ b = - 6,\ c = - 2$$

b.  $(ax + 1)(x + 5)(x + b) = 2x^{3} + 23x^{2} + 71x + 30$

$$a = 2,b = 6$$

c.  $(x + a)^{2}(x - 2) = x^{3} + bx^{2} - 3x - 18$

$$a = 3,b = 4$$

d.  When expanded, the term for $x^{2}$ in $(x + 4)(x + c)(2x - 1)$ has
    a coefficient of $- 9$.\
    What is the value of $c$?

$$c = - 8$$
