# Factorising Monic Quadratic Trinomials

- A **trinomial** is an expression with **three terms**.

- A **quadratic** expression has highest power $x^{2}$.

- A **monic** quadratic has **1** as the coefficient of $x^{2}$.

- A **monic quadratic trinomial** is an expression of the form
  $x^{2} + bx + c$

+----------------------------------------------------+
| **Factorising Monic Quadratic Trinomials (PSF      |
| Method)**                                          |
+====================================================+
| To factorise a monic quadratic trinomial:          |
| $x^{2} + bx + c = 0$:                              |
|                                                    |
| 1.  Identify the product: $c$                      |
|                                                    |
| 2.  Identify the sum: $b$                          |
|                                                    |
| 3.  Think of a factor pair, $\alpha$ and $\beta$,  |
|     that has a product of $c$ and sum to $b$.      |
|                                                    |
| 4.  Factorise                                      |
|     $x^{2} + bx + c = (x + \alpha)(x + \beta)$     |
+----------------------------------------------------+
