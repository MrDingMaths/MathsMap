## Simplifying Fractions involving Indices

- To simplify fractions:

1.  Divide the numerator and denominator by a common factor.

- When indices are involved, use the index law:

$$\frac{a^{m}}{a^{n}} = a^{m - n}$$

### **Example** Simplify algebraic fractions involving indices.

+---------------------------------------------------------------------------------------------------+---------------------------------------------+
| $$\mathbf{\ \ \ \ }\frac{\mathbf{10}\mathbf{a}^{\mathbf{5}}}{\mathbf{5}\mathbf{a}^{\mathbf{2}}}$$ | How did we know to divide numerator and     |
|                                                                                                   | denominator by $a^{2}$?                     |
| $$= \frac{10a^{5}}{5a^{2}}\ \ \frac{\div a^{2}}{\div a^{2}}\ \ $$                                 |                                             |
|                                                                                                   | $a^{2}$ *is a common .................. of* |
| $$= \frac{10a^{3}}{5}\ \ \frac{\div 5}{\div 5}$$                                                  | $10a^{5}$ *and* $5a^{2}$                    |
|                                                                                                   |                                             |
| $$= 2a^{3}$$                                                                                      | Why does $a^{5} \div a^{2} = a^{3}$?        |
|                                                                                                   |                                             |
|                                                                                                   | *The index law of division says that if the |
|                                                                                                   | bases are the same, we                      |
|                                                                                                   | ..................... the powers when       |
|                                                                                                   | dividing.*                                  |
+===================================================================================================+=============================================+
| $$\mathbf{\ \ \ \ }\frac{\mathbf{8ab}}{\mathbf{2}\mathbf{a}^{\mathbf{2}}}$$                       | In this example, why was $a$ left in the    |
|                                                                                                   | denominator?                                |
| $$= \frac{8ab}{2a^{2}}\ \frac{\div a}{\div a}\ \ $$                                               |                                             |
|                                                                                                   | *We divided the numerator and denominator   |
| $$= \frac{8b}{2a}\ \ \ \frac{\div 2}{\div 2\ }$$                                                  | by the common factor, .........*            |
|                                                                                                   |                                             |
| $$= \frac{4b}{a}$$                                                                                | *In the numerator,* $8ab \div 2a =$         |
|                                                                                                   | *............*                              |
|                                                                                                   |                                             |
|                                                                                                   | *In the denominator,* $2a^{2} \div 2a =$    |
|                                                                                                   | *............*                              |
|                                                                                                   |                                             |
|                                                                                                   | *So,* $a$ *was left in the denominator.*    |
+---------------------------------------------------------------------------------------------------+---------------------------------------------+

# Multiplying and Dividing Algebraic Fractions

  --------------------------------------------------------------------
                   **Rule for Multiplying Fractions**
  --------------------------------------------------------------------
   $$\frac{a}{b} \times \frac{c}{d} = \frac{a \times c}{b \times d}$$

  --------------------------------------------------------------------

### **Example** Multiply algebraic fractions.
