# Solving Quadratic Equations using Null Factor Law

+------------------------------------------+
| **Null Factor Law**                      |
+===================+======================+
| If                | If                   |
| $a \times b = 0$  | $(x + a)(x + b) = 0$ |
|                   |                      |
| Then $a = 0$ or   | Then $x + a = 0$     |
| $b = 0$           |                      |
|                   | or $x - b = 0$       |
+-------------------+----------------------+
