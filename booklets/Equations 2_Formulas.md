+------------------------------------------------------------------------------------------+
| - **Formulas**                                                                           |
+===================+===================+==============================+===================+
| A **formula** is a type of equation that relates two or more variables.                  |
+-------------------+-------------------+------------------------------+-------------------+
| ✖                 | ✔                 | ✔                            | ✖                 |
|                   |                   |                              |                   |
| $$5 = 2x - 7$$    | $$y = 2x - 7$$    | $$A = 2\pi r^{2} + 2\pi rh$$ | $$x = 3$$         |
+-------------------+-------------------+------------------------------+-------------------+
| Two formulas you would be familiar with are:                                             |
|                                                                                          |
| Area of a rectangle: $A = lw$                                                            |
|                                                                                          |
| Circumference of a circle: $C = 2\pi r$                                                  |
|                                                                                          |
| The **subject** is the **isolated variable** on the **left-hand side**.                  |
+------------------------------------------------------------------------------------------+

+-------------------------------------------------------------------+
| - **Key Ideas**                                                   |
+===================================================================+
| 1.  An equation that shows a r.................. between two or   |
|     more variables is called a ........................           |
|                                                                   |
| 2.  The .................. variable on the left-hand side is      |
|     called the subject.                                           |
+-------------------------------------------------------------------+

+-------------------------------------------------------------------+
| - **Using Formulas to Evaluate the Subject**                      |
+===================================================================+
| We use formulas to find the value of the subject when we know the |
| other values.                                                     |
|                                                                   |
| 1.  Write the formula.                                            |
|                                                                   |
| 2.  Substitute given values.                                      |
|                                                                   |
| 3.  Solve to find the unknown.                                    |
+-------------------------------------------------------------------+
