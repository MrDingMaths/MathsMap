# Rounding using the Critical Digit Method

+-------------------------------------------------------------------+
| - **The Critical Digit Method for Rounding**                      |
+===================================================================+
| 1.  Draw a "cut" after the required place value.                  |
|                                                                   |
| 2.  Circle the **critical digit** right after the cut.            |
|                                                                   |
| 3.  Decide to round up or down.                                   |
|                                                                   |
|     i.  If critical digit is less than 5, round down.             |
|                                                                   |
|     ii. If critical digit is 5 or more, round up.                 |
|                                                                   |
| 4.  Round.                                                        |
|                                                                   |
|     i.  rounding down: Remove digits after the cut.               |
|                                                                   |
|     ii. rounding up: Add 1 to digit before the cut, then remove   |
|         digits after the cut.                                     |
+-------------------------------------------------------------------+

+-----------------------------------------------------------------------+
| - **Example** Round decimals to a place value                         |
+===================================+===================================+
| Round 22.53486 to the nearest     | We draw a cut ............... the |
| hundredth.                        | required place value.             |
|                                   |                                   |
| 1\. Draw cut 2 2 . 5 3 4 8 6      | The critical digit is immediately |
|                                   | to the ............ of the cut.   |
| 2\. Critical digit 2 2 . 5 3 4 8  |                                   |
| 6                                 | When the critical digit is        |
|                                   | ......... or less, we round down  |
| 3\. Decide *Round down*           |                                   |
|                                   |                                   |
| 4\. Round 2 2 . 5 3               |                                   |
|                                   |                                   |
| $$22.53486 \approx 22.53$$        |                                   |
+-----------------------------------+-----------------------------------+

+-----------------------------------------------------------------------+

|     more, round up. If it is ...... or less, round down.          |

12. Josh is rounding 9.999 to two decimal places. He writes $9.90$.
    Explain why he is incorrect.

place value. This then causes the next place value to change.\
The answer should be 10.00

# Terminating and Recurring Decimals

+-------------------------------------------------------------------+
| - **Terminating and Recurring Decimals**                          |
+===================================================================+
| **Terminating** decimals **end**: 0.25                            |
|                                                                   |
| **Recurring** decimals **repeat forever**:                        |
| 0.33333333333333333333333333333...                                |
|                                                                   |
| Recurring digits are shown as:                                    |
|                                                                   |
| - a dot for a single recurring digit.                             |
|   $0.666666\ldots\  = \ 0.\dot{6}$                                |
|                                                                   |
| - a bar for many recurring digits.                                |
|   $0.686868\ldots\  = \ 0.\overline{68}$                          |
+-------------------------------------------------------------------+
